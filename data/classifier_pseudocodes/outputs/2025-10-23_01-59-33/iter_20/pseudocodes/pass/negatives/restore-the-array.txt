CLASS Solution  
    FUNCTION numberOfArrays WITH PARAMETERS s AND k  
        SET MOD TO ten RAISED TO THE POWER OF nine PLUS seven  
        SET n TO the LENGTH OF s  
        SET dp TO a LIST OF LENGTH n PLUS one FILLED WITH zero  
        SET the ELEMENT at POSITION zero OF dp TO one  
        FOR i FROM one TO n  
            FOR j FROM zero TO i MINUS one  
                IF the ELEMENT at POSITION j of s NOT EQUALS zero AND the NUMBER REPRESENTED BY the SUBSTRING FROM POSITION j TO POSITION i MINUS one OF s GREATER THAN OR EQUAL TO one AND the NUMBER REPRESENTED BY the SUBSTRING FROM POSITION j TO POSITION i MINUS one OF s LESS THAN OR EQUAL TO k  
                    SET the ELEMENT at POSITION i of dp TO the SUM OF the ELEMENT at POSITION i of dp PLUS the ELEMENT at POSITION j of dp MODULO MOD  
                END IF  
            END FOR  
        END FOR  
        RETURN the ELEMENT at POSITION n of dp  
    END FUNCTION  
END CLASS
