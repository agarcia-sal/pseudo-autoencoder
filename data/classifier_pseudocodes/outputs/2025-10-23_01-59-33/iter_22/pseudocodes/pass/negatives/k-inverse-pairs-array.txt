```
set MOD = 10^9+7
init dp as 2D array (n+1) x (k+1) filled with 0
dp[0][0] = 1

for i in 1 to n:
    prefix_sum = 0
    for j in 0 to k:
        prefix_sum = (prefix_sum + dp[i-1][j]) mod MOD
        if j >= i:
            prefix_sum = (prefix_sum - dp[i-1][j - i] + MOD) mod MOD
        dp[i][j] = prefix_sum

return dp[n][k]
```
