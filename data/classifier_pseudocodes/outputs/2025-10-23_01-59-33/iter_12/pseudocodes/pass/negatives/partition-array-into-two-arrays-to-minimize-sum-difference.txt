```
n ← len(nums)/2
total ← sum(nums)
target ← total/2
minDiff ← ∞
L, R ← nums[0:n], nums[n:2n]
leftSums, rightSums ← array of dicts size n+1

for i in 0..n:
    for combo in combinations(L, i):
        leftSums[i][sum(combo)] += 1
    for combo in combinations(R, i):
        rightSums[i][sum(combo)] += 1

for i in 0..n:
    leftVals ← sorted keys of leftSums[i]
    rightVals ← sorted keys of rightSums[n - i]

    j, k ← 0, len(rightVals) - 1
    while j < len(leftVals) and k ≥ 0:
        curr ← leftVals[j] + rightVals[k]
        diff ← |total - 2*curr|
        minDiff ← min(minDiff, diff)
        if curr < target: j += 1
        else if curr > target: k -= 1
        else: return 0

return minDiff
```
