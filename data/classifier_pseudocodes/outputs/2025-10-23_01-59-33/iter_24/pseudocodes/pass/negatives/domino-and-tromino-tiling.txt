CLASS Solution  
    FUNCTION numTilings(n)  
        SET MOD TO ten raised to the power of nine PLUS one  
        CREATE a list named dp with the LENGTH OF n PLUS one with all elements SET TO zero  
        CREATE a list named dp1 with the LENGTH OF n PLUS one with all elements SET TO zero  
        SET element at position zero of dp TO one  
        SET element at position one of dp TO one  
        IF n GREATER THAN OR EQUAL TO two  
            SET element at position two of dp TO two  
            SET element at position two of dp1 TO one  
        END IF  
        FOR i FROM three TO n  
            SET element at position i of dp TO the SUM of element at position i MINUS one of dp PLUS element at position i MINUS two of dp PLUS two MULTIPLIED BY element at position i MINUS one of dp1 MODULO MOD  
            SET element at position i of dp1 TO the SUM of element at position i MINUS two of dp PLUS element at position i MINUS one of dp1 MODULO MOD  
        END FOR  
        RETURN element at position n of dp  
    END FUNCTION  
END CLASS
