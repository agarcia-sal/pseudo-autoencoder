```
heap ← max-heap of negative target values
total ← sum(target)

while top(heap) < -1:
    largest ← -pop(heap)
    rest ← total - largest
    if largest ≤ rest or rest = 0: return False
    val ← largest mod rest
    if val = 0 and rest ≠ 1: return False
    total ← rest + val
    push(heap, -val)

return True
```
