CLASS Solution  
    FUNCTION distinctSubseqII(s)  
        SET MOD TO ten POWER nine PLUS seven  
        SET n TO the LENGTH OF s  
        SET dp TO a list OF zeroes WITH LENGTH n PLUS one  
        SET the element at position zero of dp TO one  
        SET last TO an empty mapping  
        FOR i FROM one TO n  
            SET char TO the element at position i MINUS one of s  
            SET the element at position i of dp TO the element at position i MINUS one of dp MULTIPLIED BY two  
            SET the element at position i of dp TO the element at position i of dp MODULO MOD  
            IF char EXISTS IN the keys OF last  
                SET the element at position i of dp TO the element at position i of dp MINUS the element at position last at char MINUS one of dp PLUS MOD  
                SET the element at position i of dp TO the element at position i of dp MODULO MOD  
            END IF  
            ASSIGN last at char TO i  
        END FOR  
        RETURN the element at position n of dp MINUS one MODULO MOD  
    END FUNCTION  
END CLASS
