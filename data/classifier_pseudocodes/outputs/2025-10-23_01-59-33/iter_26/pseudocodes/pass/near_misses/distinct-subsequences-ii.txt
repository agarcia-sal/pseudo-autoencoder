CLASS Solution  
    FUNCTION distinctSubseqII(s)  
        SET MOD TO ten raised to the power of nine PLUS one  
        SET n TO the LENGTH OF s  
        SET dp TO a list of zeros with the LENGTH OF n PLUS one  
        SET element at position zero of dp TO one  
        SET last TO an empty mapping  
        FOR i FROM one TO n  
            SET char TO the element at position i MINUS one of s  
            SET element at position i of dp TO two MULTIPLIED BY the element at position i MINUS one of dp MODULO MOD  
            IF char EXISTS AS a key IN last  
                SET element at position i of dp TO the element at position i of dp MINUS the element at position last mapped by char MINUS one of dp PLUS MOD MODULO MOD  
            END IF  
            ASSIGN last mapped by char TO i  
        END FOR  
        RETURN the element at position n of dp MINUS one MODULO MOD  
    END FUNCTION  
END CLASS
