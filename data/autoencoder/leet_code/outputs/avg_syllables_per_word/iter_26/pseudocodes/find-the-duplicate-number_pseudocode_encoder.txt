CLASS Solution
    FUNCTION findDuplicate(parameter nums AS List OF integer) RETURNS integer
        FUNCTION f(parameter x AS integer) RETURNS boolean
            RETURN the SUM OF evaluations WHERE each value v IN nums SATISFIES the CONDITION value v IS LESS THAN OR EQUAL TO parameter x IS GREATER THAN parameter x
        END FUNCTION
        RETURN the POSITION OF the FIRST occurrence OF TRUE USING A BINARY SEARCH OVER the RANGE FROM zero TO the LENGTH OF nums MINUS one WITH THE KEY FUNCTION f
    END FUNCTION
END CLASS
