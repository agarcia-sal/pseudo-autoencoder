CLASS Solution
    FUNCTION minBuildTime(blocks, split)
        CALL heapify ON blocks TO TRANSFORM blocks INTO a MINIMUM HEAP

        WHILE the LENGTH OF blocks IS GREATER THAN one HOLDS
            SET first TO the REMOVAL OF the SMALLEST ELEMENT FROM blocks
            SET second TO the REMOVAL OF the SMALLEST ELEMENT FROM blocks

            SET combined_time TO split PLUS second

            ADD combined_time INTO blocks MAINTAINING MINIMUM HEAP ORDER
        END WHILE

        RETURN the ELEMENT AT POSITION zero OF blocks
    END FUNCTION
END CLASS
