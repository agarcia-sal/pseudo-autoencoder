CLASS Solution
    FUNCTION getPermutation(n, k)
        IMPORT math

        SET numbers TO the LIST OF ELEMENTS FROM number one TO number n INCLUSIVE
        DECREMENT k BY number one
        SET result TO an EMPTY LIST

        FOR i FROM number n TO number one INCLUSIVE COUNTING DOWNWARD
            SET fact TO the FACTORIAL OF i MINUS number one
            SET index TO k DIVIDED BY fact
            APPEND the ELEMENT AT POSITION index PLUS one OF numbers CONVERTED TO STRING TO result
            REMOVE the ELEMENT AT POSITION index PLUS one OF numbers
            SET k TO k MODULO fact
        END FOR

        RETURN the CONCATENATION OF all ELEMENTS OF result INTO A SINGLE STRING
    END FUNCTION
END CLASS
