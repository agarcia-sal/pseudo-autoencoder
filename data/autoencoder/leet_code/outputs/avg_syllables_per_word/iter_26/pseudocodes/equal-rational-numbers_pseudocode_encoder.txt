CLASS Solution
    FUNCTION isRationalEqual(s, t)
        SET ratios TO a list containing one followed by one DIVIDED BY nine followed by one DIVIDED BY ninety nine followed by one DIVIDED BY nine hundred ninety nine followed by one DIVIDED BY nine thousand nine hundred ninety nine

        FUNCTION valueOf(s)
            IF the POSITION OF character open parenthesis IN s EQUALS negative one THEN
                RETURN the CONVERSION OF s TO floating point number
            END IF

            SET leftParenIndex TO the POSITION OF character open parenthesis IN s
            SET rightParenIndex TO the POSITION OF character close parenthesis IN s
            SET dotIndex TO the POSITION OF character period IN s

            SET integerAndNonRepeating TO the CONVERSION TO floating point number OF the substring FROM position zero TO position leftParenIndex MINUS one OF s
            SET nonRepeatingLength TO leftParenIndex MINUS dotIndex MINUS one

            SET repeating TO the CONVERSION TO integer OF the substring FROM position leftParenIndex PLUS one TO position rightParenIndex MINUS one OF s
            SET repeatingLength TO rightParenIndex MINUS leftParenIndex MINUS one

            RETURN integerAndNonRepeating PLUS repeating MULTIPLIED BY zero point one RAISED TO THE POWER OF nonRepeatingLength MULTIPLIED BY the element at position repeatingLength OF ratios
        END FUNCTION

        RETURN the ABSOLUTE VALUE OF valueOf(s) MINUS valueOf(t) LESS THAN one times ten RAISED TO THE POWER OF negative nine
    END FUNCTION
END CLASS
