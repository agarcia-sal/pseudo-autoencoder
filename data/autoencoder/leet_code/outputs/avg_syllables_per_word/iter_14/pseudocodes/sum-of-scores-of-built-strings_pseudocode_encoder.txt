CLASS Solution  
    FUNCTION sumScores WITH PARAMETER s  
        SET n TO the LENGTH OF s  
        SET lcp TO a list OF size n FILLED WITH zeroes  

        SET l TO zero  
        SET r TO zero  
        FOR i FROM one TO n MINUS one  
            IF i IS LESS THAN OR EQUAL TO r  
                SET lcp AT POSITION i TO THE MINIMUM OF r MINUS i PLUS one AND lcp AT POSITION i MINUS l  
            END IF  

            WHILE i PLUS lcp AT POSITION i IS LESS THAN n AND character AT POSITION lcp AT POSITION i OF s IS EQUAL TO character AT POSITION i PLUS lcp AT POSITION i OF s  
                INCREMENT lcp AT POSITION i BY one  
            END WHILE  

            IF i PLUS lcp AT POSITION i MINUS one IS GREATER THAN r  
                SET l TO i  
                SET r TO i PLUS lcp AT POSITION i MINUS one  
            END IF  
        END FOR  

        SET total_score TO sum OF all elements IN lcp PLUS n  
        RETURN total_score  
    END FUNCTION  
END CLASS
