CLASS Solution  
    FUNCTION numberOfWays WITH PARAMETER numPeople  
        SET MODULO TO one raised to the power of nine PLUS seven  
        INITIALIZE list dp WITH zero REPEATED numPeople PLUS one TIMES  
        SET element at position zero OF dp TO one  
        SET element at position two OF dp TO one  
        FOR index i FROM four TO numPeople IN STEPS OF two  
            FOR index j FROM zero TO i MINUS one IN STEPS OF two  
                SET element at position i OF dp TO the REMAINDER AFTER DIVIDING the SUM OF element at position i OF dp PLUS the PRODUCT OF element at position j OF dp AND element at position i MINUS two MINUS j OF dp BY MODULO  
            END FOR  
        END FOR  
        RETURN element at position numPeople OF dp  
    END FUNCTION  
END CLASS
