    FUNCTION originalDigits WITH PARAMETER s  
        INITIALIZE count TO FREQUENCY MAP OF CHARACTERS IN s  
        INITIALIZE digits TO LIST OF ten ZEROES  
        
        SET element at position zero OF digits TO VALUE OF element ASSOCIATED WITH CHARACTER z IN count  
        SET element at position two OF digits TO VALUE OF element ASSOCIATED WITH CHARACTER w IN count  
        SET element at position four OF digits TO VALUE OF element ASSOCIATED WITH CHARACTER u IN count  
        SET element at position six OF digits TO VALUE OF element ASSOCIATED WITH CHARACTER x IN count  
        SET element at position eight OF digits TO VALUE OF element ASSOCIATED WITH CHARACTER g IN count  
        
        SET element at position three OF digits TO VALUE OF element ASSOCIATED WITH CHARACTER h IN count MINUS element at position eight OF digits  
        SET element at position five OF digits TO VALUE OF element ASSOCIATED WITH CHARACTER f IN count MINUS element at position four OF digits  
        SET element at position seven OF digits TO VALUE OF element ASSOCIATED WITH CHARACTER s IN count MINUS element at position six OF digits  
        SET element at position nine OF digits TO VALUE OF element ASSOCIATED WITH CHARACTER i IN count MINUS element at position five OF digits MINUS element at position six OF digits MINUS element at position eight OF digits  
        SET element at position one OF digits TO VALUE OF element ASSOCIATED WITH CHARACTER o IN count MINUS element at position zero OF digits MINUS element at position two OF digits MINUS element at position four OF digits  
        
        INITIALIZE result TO EMPTY LIST  
        FOR each index FROM zero TO nine  
            APPEND STRING REPRESENTATION OF index REPEATED element at position index OF digits TIMES TO result  
        END FOR  
        
        RETURN CONCATENATION OF ALL ELEMENTS IN result  
    END FUNCTION  
END CLASS
