CLASS Solution  
    FUNCTION frequencySort WITH PARAMETER s  
        SET frequency TO the result of counting the occurrences of each character in s USING a helper subroutine for character counting  
        SET sorted_characters TO the list of characters in frequency sorted in descending order ACCORDING TO the frequency of each character  
        SET result TO an empty string  
        FOR each character IN sorted_characters  
            APPEND the character REPEATED frequency AT character TIMES TO result  
        END FOR  
        RETURN result  
    END FUNCTION  
END CLASS
