CLASS Solution  
    FUNCTION minBuildTime(blocks AS list OF integers, split AS integer)  
        CALL heapify THE list blocks TO TRANSFORM blocks INTO A MINIMUM HEAP  
        WHILE THE LENGTH OF blocks IS GREATER THAN one  
            SET first TO THE SMALLEST ELEMENT REMOVED FROM blocks  
            SET second TO THE SMALLEST ELEMENT REMOVED FROM blocks AFTER REMOVING first  
            SET combined_time TO split PLUS second  
            INSERT combined_time INTO blocks TO MAINTAIN THE MINIMUM HEAP PROPERTY  
        END WHILE  
        RETURN THE ELEMENT AT POSITION zero OF blocks  
    END FUNCTION  
END CLASS
