CLASS Solution  
    FUNCTION numberOfBoomerangs WITH PARAMETER points  
        SET ans TO zero  
        FOR each p1 IN points  
            INITIALIZE cnt AS a new counter mapping with default zero values  
            FOR each p2 IN points  
                SET dx TO element at position zero OF p1 MINUS element at position zero OF p2  
                SET dy TO element at position one OF p1 MINUS element at position one OF p2  
                SET d TO dx MULTIPLIED BY dx PLUS dy MULTIPLIED BY dy  
                INCREMENT ans BY the value associated to key d IN cnt  
                INCREMENT the value associated to key d IN cnt BY one  
            END FOR  
        END FOR  
        RETURN ans MULTIPLIED BY two  
    END FUNCTION  
END CLASS
