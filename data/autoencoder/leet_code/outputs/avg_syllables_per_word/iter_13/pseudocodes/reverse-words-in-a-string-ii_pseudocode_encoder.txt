CLASS Solution  
    FUNCTION reverseWords(s)  
        CALL METHOD reverse OF s TO REVERSE THE ENTIRE COLLECTION s  
        SET start TO zero  
        FOR end FROM zero TO the LENGTH OF s MINUS one  
            IF element at position end of s EQUALS a space character  
                CALL FUNCTION reversed WITH ELEMENTS from position start TO position end MINUS one of s AND ASSIGN THE RESULT TO the ELEMENTS from position start TO position end MINUS one of s  
                SET start TO end PLUS one  
            END IF  
        END FOR  
        CALL FUNCTION reversed WITH ELEMENTS from position start TO the last position of s AND ASSIGN THE RESULT TO THE SAME ELEMENTS of s  
    END FUNCTION  
END CLASS
