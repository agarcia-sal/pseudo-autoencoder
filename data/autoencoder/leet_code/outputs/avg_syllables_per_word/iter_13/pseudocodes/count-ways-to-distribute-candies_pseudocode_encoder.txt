CLASS Solution  
    FUNCTION waysToDistribute(n, k)  
        SET MOD TO one billion PLUS seven  
        SET dp TO two dimensional list of zeroes with the first dimension of size n PLUS one and the second dimension of size k PLUS one  
        SET element at position zero of dp and position zero of that element TO one  
        FOR i FROM one TO n  
            FOR j FROM one TO k  
                SET element at position i of dp and position j of that element TO the remainder after division by MOD of the sum of the product of j and element at position i MINUS one of dp and position j of that element AND element at position i MINUS one of dp and position j MINUS one of that element  
            END FOR  
        END FOR  
        RETURN element at position n of dp and position k of that element  
    END FUNCTION  
END CLASS
