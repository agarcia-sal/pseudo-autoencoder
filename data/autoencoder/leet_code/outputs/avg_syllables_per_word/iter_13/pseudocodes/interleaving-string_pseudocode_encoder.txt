CLASS Solution  
    FUNCTION isInterleave WITH PARAMETERS s1 s2 s3  
        IF the LENGTH OF s1 PLUS the LENGTH OF s2 DOES NOT EQUAL the LENGTH OF s3  
            RETURN false  
        END IF  
        SET dp TO a list of boolean values all set to false with the LENGTH OF s2 PLUS one elements  
        SET the element at position zero of dp TO true  
        FOR index i FROM zero TO the LENGTH OF s1 INCLUSIVE  
            FOR index j FROM zero TO the LENGTH OF s2 INCLUSIVE  
                IF index i IS GREATER THAN zero  
                    SET the element at position j of dp TO the logical AND of the element at position j of dp AND the condition that the element at position i MINUS one of s1 IS EQUALS TO the element at position i PLUS j MINUS one of s3  
                END IF  
                IF index j IS GREATER THAN zero  
                    SET the element at position j of dp TO the logical OR of the element at position j of dp AND the logical AND of the element at position j MINUS one of dp AND the condition that the element at position j MINUS one of s2 IS EQUALS TO the element at position i PLUS j MINUS one of s3  
                END IF  
            END FOR  
        END FOR  
        RETURN the element at position the LENGTH OF s2 of dp  
    END FUNCTION  
END CLASS
