CLASS Solution  
    FUNCTION letterCasePermutation WITH PARAMETER s  
        FUNCTION dfs WITH PARAMETER i  
            IF i IS GREATER THAN OR EQUAL TO the LENGTH OF t  
                APPEND the JOINING OF ALL ELEMENTS OF t TO ans  
                RETURN  
            END IF  
            CALL dfs WITH ARGUMENT i INCREMENTED BY one  
            IF the ELEMENT at POSITION i OF t IS AN ALPHABETIC CHARACTER  
                SET the ELEMENT at POSITION i OF t TO the CHARACTER CORRESPONDING TO the ASCII VALUE OF the ELEMENT at POSITION i OF t EXCLUSIVE ORed WITH the NUMBER thirty two  
                CALL dfs WITH ARGUMENT i INCREMENTED BY one  
            END IF  
        END FUNCTION  
      
        SET t TO A LIST OF ALL CHARACTERS IN s  
        SET ans TO AN EMPTY LIST  
        CALL dfs WITH ARGUMENT zero  
        RETURN ans  
    END FUNCTION  
END CLASS
