CLASS Solution  
    FUNCTION productExceptSelf WITH PARAMETER nums  
        SET n TO the LENGTH OF nums  
        SET answer TO a list of one repeated n times  
        SET left_product TO one  
        FOR index FROM zero TO n MINUS one  
            SET element at position index of answer TO left_product  
            SET left_product TO left_product MULTIPLIED BY element at position index of nums  
        END FOR  
        SET right_product TO one  
        FOR index FROM n MINUS one DOWN TO zero  
            SET element at position index of answer TO element at position index of answer MULTIPLIED BY right_product  
            SET right_product TO right_product MULTIPLIED BY element at position index of nums  
        END FOR  
        RETURN answer  
    END FUNCTION  
END CLASS
