CLASS Solution  
    FUNCTION ambiguousCoordinates WITH PARAMETER s  
        FUNCTION make WITH PARAMETER frag  
            SET n TO the LENGTH OF frag  
            FOR d FROM one TO n  
                SET left TO the SUBSTRING from position one TO d of frag  
                SET right TO the SUBSTRING from position d PLUS one TO the LENGTH OF frag of frag  
                IF the first character of left IS NOT EQUAL TO zero OR left IS EQUAL TO zero  
                AND the last character of right IS NOT EQUAL TO zero  
                    YIELD the CONCATENATION of left PLUS the CHARACTER dot IF d IS NOT EQUAL TO n OTHERWISE an empty string PLUS right  
                END IF  
            END FOR  
        END FUNCTION  
        SET s TO the SUBSTRING from position two TO the LENGTH OF s MINUS one of s  
        RETURN a LIST COMPREHENSION of the STRING open parenthesis CONCATENATED WITH a CONCATENATED WITH comma CONCATENATED WITH space CONCATENATED WITH b CONCATENATED WITH close parenthesis FOR i FROM one TO the LENGTH OF s MINUS one FOR a AND b IN the CARTESIAN PRODUCT of make with the SUBSTRING from position one TO i of s AND make with the SUBSTRING from position i PLUS one TO the LENGTH OF s of s  
    END FUNCTION  
END CLASS
