CLASS Solution  
    FUNCTION poorPigs(buckets minutesToDie minutesToTest)  
        SET states TO the INTEGER RESULT OF minutesToTest DIVIDED BY minutesToDie PLUS one  
        SET pigs TO the SMALLEST INTEGER GREATER THAN OR EQUAL TO the RESULT OF the NATURAL LOGARITHM OF buckets DIVIDED BY the NATURAL LOGARITHM OF states  
        RETURN pigs  
    END FUNCTION  
END CLASS
