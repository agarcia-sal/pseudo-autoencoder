CLASS Solution  
    FUNCTION isNumber(s)  
        IMPORT re  
        SET number_regex TO a string that STARTS WITH an optional plus OR minus sign FOLLOWED BY EITHER one or more digits FOLLOWED BY an optional decimal point AND zero OR more digits OR a decimal point FOLLOWED BY one or more digits AND AN optional exponent part consisting of the letter e FOLLOWED BY an optional plus OR minus sign AND one or more digits  
        RETURN the result of the method fullmatch of the module re CALLED with number_regex AND s IS NOT EQUAL TO null  
    END FUNCTION  
END CLASS
