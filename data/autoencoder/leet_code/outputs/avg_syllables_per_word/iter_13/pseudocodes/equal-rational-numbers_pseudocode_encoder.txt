CLASS Solution  
    FUNCTION isRationalEqual WITH PARAMETERS s AS string AND t AS string RETURNS boolean  
        SET ratios TO list containing one and one DIVIDED BY nine and one DIVIDED BY ninety nine and one DIVIDED BY nine hundred ninety nine and one DIVIDED BY nine thousand nine hundred ninety nine  

        FUNCTION valueOf WITH PARAMETER s AS string RETURNS float  
            IF the POSITION OF opening parenthesis IN s EQUALS negative one  
                RETURN the numeric value of s CONVERTED TO float  
            END IF  

            SET leftParenIndex TO the POSITION OF opening parenthesis IN s  
            SET rightParenIndex TO the POSITION OF closing parenthesis IN s  
            SET dotIndex TO the POSITION OF period character IN s  

            SET integerAndNonRepeating TO the numeric value CONVERTED TO float OF the SUBSTRING from the START TO the POSITION before leftParenIndex of s  
            SET nonRepeatingLength TO leftParenIndex MINUS dotIndex MINUS one  

            SET repeating TO the numeric value CONVERTED TO integer OF the SUBSTRING from the POSITION after leftParenIndex TO the POSITION before rightParenIndex of s  
            SET repeatingLength TO rightParenIndex MINUS leftParenIndex MINUS one  

            RETURN integerAndNonRepeating PLUS repeating MULTIPLIED BY zero POINT one RAISED TO THE POWER OF nonRepeatingLength MULTIPLIED BY the ELEMENT at position repeatingLength OF ratios  
        END FUNCTION  

        RETURN the ABSOLUTE VALUE of valueOf applied to s MINUS valueOf applied to t IS LESS THAN one TIMES ten RAISED TO THE POWER OF negative nine  
    END FUNCTION  
END CLASS
