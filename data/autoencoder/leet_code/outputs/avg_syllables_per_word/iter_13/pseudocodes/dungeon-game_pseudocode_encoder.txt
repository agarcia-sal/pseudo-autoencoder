CLASS Solution  
    FUNCTION calculateMinimumHP WITH PARAMETERS dungeon  
        SET m TO the LENGTH OF dungeon  
        SET n TO the LENGTH OF element at position zero of dungeon  
        SET dp TO a two dimensional list with dimensions m PLUS one BY n PLUS one with all elements initialized to infinity  
        SET element at position m AND n MINUS one of dp TO one  
        SET element at position m MINUS one AND n of dp TO one  
        FOR i FROM m MINUS one DOWN TO zero  
            FOR j FROM n MINUS one DOWN TO zero  
                SET min_health TO the MINIMUM OF element at position i PLUS one AND j of dp AND element at position i AND j PLUS one of dp MINUS element at position i AND j of dungeon  
                SET element at position i AND j of dp TO the MAXIMUM OF min_health AND one  
            END FOR  
        END FOR  
        RETURN element at position zero AND zero of dp  
    END FUNCTION  
END CLASS
