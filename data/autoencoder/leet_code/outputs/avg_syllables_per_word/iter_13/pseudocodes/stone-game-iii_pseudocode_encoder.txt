CLASS Solution  
    FUNCTION stoneGameIII(stoneValue)  
        SET n TO the LENGTH OF stoneValue  
        SET dp TO a list of zeros with the LENGTH OF n PLUS one  
        FOR i FROM n MINUS one DOWN TO zero  
            SET dp element at position i TO element at position i of stoneValue MINUS dp element at position i PLUS one  
            IF i PLUS two IS LESS THAN OR EQUAL TO n THEN  
                SET dp element at position i TO the GREATER VALUE BETWEEN dp element at position i AND the SUM OF element at position i of stoneValue PLUS element at position i PLUS one of stoneValue MINUS dp element at position i PLUS two  
            END IF  
            IF i PLUS three IS LESS THAN OR EQUAL TO n THEN  
                SET dp element at position i TO the GREATER VALUE BETWEEN dp element at position i AND the SUM OF element at position i of stoneValue PLUS element at position i PLUS one of stoneValue PLUS element at position i PLUS two of stoneValue MINUS dp element at position i PLUS three  
            END IF  
        END FOR  
        IF dp element at position zero IS GREATER THAN zero THEN  
            RETURN Alice  
        ELSE IF dp element at position zero IS LESS THAN zero THEN  
            RETURN Bob  
        ELSE  
            RETURN Tie  
        END IF  
    END FUNCTION  
END CLASS
