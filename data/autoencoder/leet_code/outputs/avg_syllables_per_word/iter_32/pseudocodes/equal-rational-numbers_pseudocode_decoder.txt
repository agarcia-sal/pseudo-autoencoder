CLASS Solution  
    FUNCTION isRationalEqual(s, t)  
        SET ratios TO list containing one and one DIVIDED BY nine and one DIVIDED BY ninety nine and one DIVIDED BY nine hundred ninety nine and one DIVIDED BY nine thousand nine hundred ninety nine  

        FUNCTION valueOf(s)  
            IF the position of the character left parenthesis in s EQUALS minus one  
                RETURN convert s TO floating point number  
            END IF  
            SET leftParenIndex TO the position of the character left parenthesis in s  
            SET rightParenIndex TO the position of the character right parenthesis in s  
            SET dotIndex TO the position of the character period in s  

            SET integerAndNonRepeating TO convert to floating point number the substring from the start position TO the position leftParenIndex minus one of s  
            SET nonRepeatingLength TO leftParenIndex MINUS dotIndex MINUS one  

            SET repeating TO convert to integer the substring from the position leftParenIndex PLUS one TO the position rightParenIndex MINUS one of s  
            SET repeatingLength TO rightParenIndex MINUS leftParenIndex MINUS one  
            RETURN integerAndNonRepeating PLUS repeating MULTIPLIED BY zero point one RAISED TO THE POWER OF nonRepeatingLength MULTIPLIED BY the element at position repeatingLength of ratios  
        END FUNCTION  

        RETURN the absolute value of valueOf(s) MINUS valueOf(t) LESS THAN one times ten RAISED TO THE POWER OF minus nine  
    END FUNCTION  
END CLASS
