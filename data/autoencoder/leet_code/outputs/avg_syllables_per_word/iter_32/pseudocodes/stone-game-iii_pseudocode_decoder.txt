CLASS Solution  
    FUNCTION stoneGameIII(stoneValue)  
        SET n TO the LENGTH OF stoneValue  
        SET dp TO a list of zero values with the LENGTH OF n PLUS one  
        FOR i FROM the value of n MINUS one DOWN TO zero  
            SET dp element at position i TO element at position i of stoneValue MINUS dp element at position i PLUS one  
            IF the value of i PLUS two IS LESS THAN OR EQUAL TO n THEN  
                SET dp element at position i TO the GREATER OF dp element at position i AND element at position i of stoneValue PLUS element at position i PLUS one of stoneValue MINUS dp element at position i PLUS two  
            END IF  
            IF the value of i PLUS three IS LESS THAN OR EQUAL TO n THEN  
                SET dp element at position i TO the GREATER OF dp element at position i AND element at position i of stoneValue PLUS element at position i PLUS one of stoneValue PLUS element at position i PLUS two of stoneValue MINUS dp element at position i PLUS three  
            END IF  
        END FOR  
        IF dp element at position zero IS GREATER THAN zero THEN  
            RETURN the string Alice  
        ELSE IF dp element at position zero IS LESS THAN zero THEN  
            RETURN the string Bob  
        ELSE  
            RETURN the string Tie  
        END IF  
    END FUNCTION  
END CLASS
