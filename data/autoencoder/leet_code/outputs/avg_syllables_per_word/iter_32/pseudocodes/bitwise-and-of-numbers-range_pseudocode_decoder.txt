CLASS Solution  
    FUNCTION rangeBitwiseAnd(left parameter, right parameter)  
        SET shift TO zero  
        WHILE left LESS THAN right  
            SET left TO the integer value of left DIVIDED BY two  
            SET right TO the integer value of right DIVIDED BY two  
            INCREMENT shift BY one  
        END WHILE  
        RETURN left MULTIPLIED BY two RAISED TO THE POWER OF shift  
    END FUNCTION  
END CLASS
