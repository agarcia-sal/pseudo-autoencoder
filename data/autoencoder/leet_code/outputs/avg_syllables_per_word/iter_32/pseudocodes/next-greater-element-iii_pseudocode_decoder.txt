CLASS Solution  
    FUNCTION nextGreaterElement(n)  
        SET digits TO the list of characters from the string representation of n  
        SET length TO the LENGTH OF digits  
        SET i TO length MINUS two  
        WHILE i IS GREATER THAN OR EQUAL TO zero AND element at position i of digits IS GREATER THAN OR EQUAL TO element at position i PLUS one of digits  
            DECREMENT i BY one  
        END WHILE  
        IF i EQUALS negative one  
            RETURN negative one  
        END IF  
        SET j TO length MINUS one  
        WHILE element at position j of digits IS LESS THAN OR EQUAL TO element at position i of digits  
            DECREMENT j BY one  
        END WHILE  
        SET temporary TO element at position i of digits  
        SET element at position i of digits TO element at position j of digits  
        SET element at position j of digits TO temporary  
        SET digits TO concatenation of the elements of digits from position zero TO position i PLUS one FOLLOWED BY the elements of digits from position i PLUS one TO the end IN REVERSE ORDER  
        SET result TO the integer value of the concatenation of the characters in digits  
        IF result IS GREATER THAN the value of two raised to the power of thirty one MINUS one  
            RETURN negative one  
        END IF  
        RETURN result  
    END FUNCTION  
END CLASS
