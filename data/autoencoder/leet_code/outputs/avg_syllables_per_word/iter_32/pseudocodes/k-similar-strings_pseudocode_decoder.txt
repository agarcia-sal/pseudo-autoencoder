CLASS Solution  
    FUNCTION kSimilarity(s1, s2)  
        FUNCTION neighbors(s)  
            SET i TO zero  
            WHILE i LESS THAN the LENGTH OF s AND the element at position i of s EQUALS the element at position i of s2  
                INCREMENT i BY one  
            END WHILE  
            FOR j FROM i PLUS one TO the LENGTH OF s MINUS one  
                IF the element at position j of s EQUALS the element at position i of s2 AND the element at position j of s NOT EQUALS the element at position j of s2  
                    YIELD a new string constructed by concatenating the substring from position zero TO position i MINUS one of s PLUS the element at position j of s PLUS the substring from position i PLUS one TO position j MINUS one of s PLUS the element at position i of s PLUS the substring from position j PLUS one TO the end of s  
                END IF  
            END FOR  
        END FUNCTION  

        ASSIGN reference TO a new double ended queue initialized with a tuple consisting of s1 and zero  
        ASSIGN reference TO a new set containing s1  

        WHILE the queue is not empty  
            REMOVE and ASSIGN the leftmost tuple of the queue TO s and swaps  
            IF s EQUALS s2  
                RETURN swaps  
            END IF  
            FOR each neighbor generated by neighbors called with s  
                IF neighbor is not an element of visited  
                    ADD neighbor TO visited  
                    APPEND a tuple consisting of neighbor and swaps PLUS one TO the queue  
                END IF  
            END FOR  
        END WHILE  
    END FUNCTION  
END CLASS
