CLASS Solution  
    FUNCTION jobScheduling(startTime, endTime, profit)  
        SET jobs TO list of tuples each containing element at position n of startTime element at position n of endTime element at position n of profit for index n FROM zero TO length of startTime MINUS one  
        SORT jobs BY the second element of each tuple in ascending order  
        SET end_times TO list of second elements of each tuple in jobs  
        SET dp TO list of zeros with length EQUAL TO length of jobs PLUS one  
        FOR i FROM one TO length of jobs  
            SET current_start_time TO first element of tuple at position i MINUS one of jobs  
            SET j TO result of binary search for position to insert current_start_time in end_times such that order is maintained  
            SET dp AT position i TO the GREATER OF dp AT position i MINUS one AND dp AT position j PLUS the third element of tuple at position i MINUS one of jobs  
        END FOR  
        RETURN the element at position length of dp MINUS one of dp  
    END FUNCTION  
END CLASS
