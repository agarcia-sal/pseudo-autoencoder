CLASS Solution  
    FUNCTION findClosestElements(arr k x)  
        SET left TO the position where x would be inserted in arr to keep arr sorted MINUS one  
        SET right TO left PLUS one  
        WHILE right MINUS left MINUS one LESS THAN k  
            IF left EQUALS negative one OR right LESS THAN the LENGTH OF arr AND the ABSOLUTE VALUE OF element at position right of arr MINUS x LESS THAN the ABSOLUTE VALUE OF element at position left of arr MINUS x  
                INCREMENT right BY one  
            ELSE  
                DECREMENT left BY one  
            END IF  
        END WHILE  
        RETURN the sublist of arr from position left PLUS one TO position right MINUS one  
    END FUNCTION  
END CLASS
