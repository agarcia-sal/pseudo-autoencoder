CLASS Solution  
    FUNCTION originalDigits(s)  
        SET count TO a mapping of characters to their frequency in s  
        SET digits TO a list of ten zeros each representing digit counts from zero TO nine  
        SET element at position zero of digits TO the frequency of character corresponding TO the spoken letter z in count  
        SET element at position two of digits TO the frequency of character corresponding TO the spoken letter w in count  
        SET element at position four of digits TO the frequency of character corresponding TO the spoken letter u in count  
        SET element at position six of digits TO the frequency of character corresponding TO the spoken letter x in count  
        SET element at position eight of digits TO the frequency of character corresponding TO the spoken letter g in count  
        SET element at position three of digits TO the frequency of character corresponding TO the spoken letter h in count MINUS the element at position eight of digits  
        SET element at position five of digits TO the frequency of character corresponding TO the spoken letter f in count MINUS the element at position four of digits  
        SET element at position seven of digits TO the frequency of character corresponding TO the spoken letter s in count MINUS the element at position six of digits  
        SET element at position nine of digits TO the frequency of character corresponding TO the spoken letter i in count MINUS the element at position five of digits MINUS the element at position six of digits MINUS the element at position eight of digits  
        SET element at position one of digits TO the frequency of character corresponding TO the spoken letter o in count MINUS the element at position zero of digits MINUS the element at position two of digits MINUS the element at position four of digits  
        SET result TO an empty list  
        FOR each i FROM zero TO nine  
            FOR each occurrence FROM one TO the element at position i of digits  
                APPEND the string representation of i TO result  
            END FOR  
        END FOR  
        RETURN the concatenation of all elements in result  
    END FUNCTION  
END CLASS
