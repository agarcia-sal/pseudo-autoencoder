CLASS Solution  
    FUNCTION stoneGameIII(stoneValue)  
        SET n TO the LENGTH OF stoneValue  
        SET dp TO a list of zeros with the LENGTH OF n PLUS one  
        FOR i FROM n MINUS one DOWN TO zero  
            SET dp AT position i TO element at position i of stoneValue MINUS dp AT position i PLUS one  
            IF i PLUS two IS LESS THAN OR EQUAL TO n  
                SET dp AT position i TO the GREATER OF dp AT position i AND the SUM OF element at position i of stoneValue AND element at position i PLUS one of stoneValue MINUS dp AT position i PLUS two  
            END IF  
            IF i PLUS three IS LESS THAN OR EQUAL TO n  
                SET dp AT position i TO the GREATER OF dp AT position i AND the SUM OF element at position i of stoneValue AND element at position i PLUS one of stoneValue AND element at position i PLUS two of stoneValue MINUS dp AT position i PLUS three  
            END IF  
        END FOR  
        IF dp AT position zero IS GREATER THAN zero  
            RETURN Alice  
        ELSE IF dp AT position zero IS LESS THAN zero  
            RETURN Bob  
        ELSE  
            RETURN Tie  
        END IF  
    END FUNCTION  
END CLASS
