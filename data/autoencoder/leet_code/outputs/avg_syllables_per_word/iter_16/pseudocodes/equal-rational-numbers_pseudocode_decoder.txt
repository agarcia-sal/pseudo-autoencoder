        SET ratios TO list containing one followed by one DIVIDED BY nine followed by one DIVIDED BY ninety nine followed by one DIVIDED BY nine hundred ninety nine followed by one DIVIDED BY nine thousand nine hundred ninety nine  

        FUNCTION valueOf(s)  
            IF position of opening parenthesis IN s EQUALS negative one  
                RETURN conversion of s TO float number  
            END IF  
            
            SET leftParenIndex TO position of opening parenthesis IN s  
            SET rightParenIndex TO position of closing parenthesis IN s  
            SET dotIndex TO position of decimal point IN s  
            
            SET integerAndNonRepeating TO conversion TO float number OF substring from start TO position one BEFORE leftParenIndex of s  
            SET nonRepeatingLength TO leftParenIndex MINUS dotIndex MINUS one  
            
            SET repeating TO conversion TO integer number OF substring from position one AFTER leftParenIndex TO position one BEFORE rightParenIndex of s  
            SET repeatingLength TO rightParenIndex MINUS leftParenIndex MINUS one  
            
            RETURN integerAndNonRepeating PLUS repeating MULTIPLIED BY ten RAISED TO THE POWER OF negative nonRepeatingLength MULTIPLIED BY element at position repeatingLength of ratios  
        END FUNCTION  

        RETURN absolute value of result of valueOf called with s MINUS result of valueOf called with t LESS THAN one DIVIDED BY one billion  
    END FUNCTION  
END CLASS
