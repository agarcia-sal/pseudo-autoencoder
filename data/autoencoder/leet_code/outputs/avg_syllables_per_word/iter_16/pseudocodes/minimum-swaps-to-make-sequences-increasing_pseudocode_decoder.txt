CLASS Solution  
    FUNCTION minSwap(nums1, nums2)  
        SET n TO the LENGTH OF nums1  
        INITIALIZE dp AS a list of length n WITH each element BEING a list of two zeros  
        SET element at position zero position zero of dp TO zero  
        SET element at position zero position one of dp TO one  
        FOR i FROM one TO n MINUS one  
            IF element at position i of nums1 IS GREATER THAN element at position i MINUS one of nums1  
            AND element at position i of nums2 IS GREATER THAN element at position i MINUS one of nums2  
                IF element at position i of nums1 IS GREATER THAN element at position i MINUS one of nums2  
                AND element at position i of nums2 IS GREATER THAN element at position i MINUS one of nums1  
                    SET element at position i position zero of dp TO THE MINIMUM OF element at position i MINUS one position zero of dp AND element at position i MINUS one position one of dp  
                    SET element at position i position one of dp TO element at position i position zero of dp PLUS one  
                ELSE  
                    SET element at position i position zero of dp TO element at position i MINUS one position zero of dp  
                    SET element at position i position one of dp TO element at position i MINUS one position one of dp PLUS one  
                END IF  
            ELSE  
                IF element at position i of nums1 IS GREATER THAN element at position i MINUS one of nums2  
                AND element at position i of nums2 IS GREATER THAN element at position i MINUS one of nums1  
                    SET element at position i position zero of dp TO element at position i MINUS one position one of dp  
                    SET element at position i position one of dp TO element at position i MINUS one position zero of dp PLUS one  
                ELSE  
                    SET element at position i position zero of dp TO element at position i MINUS one position one of dp  
                    SET element at position i position one of dp TO element at position i MINUS one position zero of dp PLUS one  
                END IF  
            END IF  
        END FOR  
        RETURN THE MINIMUM OF element at position n MINUS one position zero of dp AND element at position n MINUS one position one of dp  
    END FUNCTION  
END CLASS
