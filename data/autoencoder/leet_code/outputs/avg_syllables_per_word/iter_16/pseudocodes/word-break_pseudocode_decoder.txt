CLASS Solution  
    FUNCTION wordBreak WITH PARAMETERS s string AND wordDict list of strings  
        SET wordSet TO the CONVERSION OF wordDict INTO A SET  
        SET dp TO A LIST OF BOOLEAN FALSE VALUES WITH THE LENGTH EQUAL TO THE LENGTH OF s PLUS one  
        SET element at position zero of dp TO TRUE  
        FOR position i FROM one TO THE LENGTH OF s INCLUSIVE  
            FOR position j FROM zero TO position i MINUS one  
                IF element at position j of dp EQUALS TRUE AND substring from position j TO position i of s IS AN ELEMENT OF wordSet  
                    SET element at position i of dp TO TRUE  
                    BREAK OUT OF THE INNER FOR LOOP  
                END IF  
            END FOR  
        END FOR  
        RETURN element at position THE LENGTH OF s of dp  
    END FUNCTION  
END CLASS
