CLASS Solution  
    FUNCTION numTilings WITH PARAMETER n  
        SET MOD TO one billion PLUS seven  
        INITIALIZE dp AS a list of zeros WITH length n PLUS one  
        INITIALIZE dp1 AS a list of zeros WITH length n PLUS one  
        SET element at position zero of dp TO one  
        SET element at position one of dp TO one  
        IF n IS GREATER THAN OR EQUAL TO two THEN  
            SET element at position two of dp TO two  
            SET element at position two of dp1 TO one  
        END IF  
        FOR i FROM three TO n  
            SET element at position i of dp TO the remainder after division by MOD of the sum of the element at position i minus one of dp PLUS the element at position i minus two of dp PLUS two TIMES the element at position i minus one of dp1  
            SET element at position i of dp1 TO the remainder after division by MOD of the sum of the element at position i minus two of dp PLUS the element at position i minus one of dp1  
        END FOR  
        RETURN element at position n of dp  
    END FUNCTION  
END CLASS
