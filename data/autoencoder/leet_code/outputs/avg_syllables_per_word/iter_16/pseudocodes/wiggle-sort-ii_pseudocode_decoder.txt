CLASS Solution  
    FUNCTION wiggleSort(nums)  
        CALL the SORT method of the list nums to arrange elements in ascending order  
        SET sorted_nums TO a new list that is a copy of the list nums  
        SET n TO the LENGTH OF the list nums  
        SET left TO the integer division of n MINUS one BY two  
        SET right TO n MINUS one  
        FOR each index i FROM zero TO n MINUS one  
            IF the remainder of i DIVIDED BY two EQUALS zero  
                SET the element at position i of list nums TO the element at position left of list sorted_nums  
                DECREMENT left BY one  
            ELSE  
                SET the element at position i of list nums TO the element at position right of list sorted_nums  
                DECREMENT right BY one  
            END IF  
        END FOR  
    END FUNCTION  
END CLASS
