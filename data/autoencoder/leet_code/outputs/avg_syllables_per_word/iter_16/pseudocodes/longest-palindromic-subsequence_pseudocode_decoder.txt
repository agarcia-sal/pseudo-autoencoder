CLASS Solution  
    FUNCTION longestPalindromeSubseq(s)  
        SET n TO the LENGTH OF s  
        SET dp TO a two dimensional array of size n by n initialized TO zero  
        FOR each index i FROM zero TO n MINUS one  
            SET the element at position i at position i of dp TO one  
        END FOR  
        FOR each length FROM two TO n  
            FOR each i FROM zero TO n MINUS length  
                SET j TO i PLUS length MINUS one  
                IF the character at position i of s EQUALS the character at position j of s  
                    SET the element at position i at position j of dp TO the element at position i PLUS one at position j MINUS one of dp PLUS two  
                ELSE  
                    SET the element at position i at position j of dp TO the GREATER VALUE BETWEEN the element at position i PLUS one at position j of dp AND the element at position i at position j MINUS one of dp  
                END IF  
            END FOR  
        END FOR  
        RETURN the element at position zero at position n MINUS one of dp  
    END FUNCTION  
END CLASS
