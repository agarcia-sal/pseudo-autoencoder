CLASS Solution  
    FUNCTION reverse(x)  
        SET sign TO negative one IF x IS LESS THAN zero OTHERWISE positive one  
        SET absolute_value_string TO string conversion of absolute value of x  
        SET reversed_string TO characters of absolute_value_string in reverse order  
        SET reversed_x TO integer conversion of reversed_string  
        SET result TO sign MULTIPLIED BY reversed_x  
        IF result IS LESS THAN negative two RAISED TO THE THIRTY FIRST POWER OR result IS GREATER THAN positive two RAISED TO THE THIRTY FIRST POWER MINUS one  
            RETURN zero  
        END IF  
        RETURN result  
    END FUNCTION  
END CLASS
