CLASS Solution  
    FUNCTION pathsWithMaxScore(board)  
        SET MOD TO the value of ten raised to the power of nine PLUS one  
        SET n TO the LENGTH OF board  
        SET dp TO a two dimensional array of size n by n where each element is a two element list containing NEGATIVE_INFINITY and zero  
        SET element at position n MINUS one of dp and position n MINUS one of that to a list containing zero and one  
        FOR i FROM n MINUS one DOWN TO zero  
            FOR j FROM n MINUS one DOWN TO zero  
                IF element at position i of board and position j of that IS EQUAL TO the character X OR element at position i of board and position j of that IS EQUAL TO the character S  
                    CONTINUE to next iteration  
                END IF  
                IF element at position i of board and position j of that IS EQUAL TO the character E  
                    SET value TO zero  
                ELSE  
                    SET value TO the INTEGER REPRESENTATION OF element at position i of board and position j of that  
                END IF  
                FOR each pair x AND y IN the list containing the pairs one zero, zero one, one one  
                    SET ni TO i PLUS x  
                    SET nj TO j PLUS y  
                    IF ni IS GREATER THAN OR EQUAL TO zero AND ni IS LESS THAN n AND nj IS GREATER THAN OR EQUAL TO zero AND nj IS LESS THAN n  
                        IF element at position ni of dp and position nj of that and position zero of that IS GREATER THAN element at position i of dp and position j of that and position zero of that  
                            SET element at position i of dp and position j of that TO a list containing element at position ni of dp and position nj of that and position zero of that PLUS value and element at position ni of dp and position nj of that and position one of that  
                        ELSE IF element at position ni of dp and position nj of that and position zero of that IS EQUAL TO element at position i of dp and position j of that and position zero of that  
                            SET element at position i of dp and position j of that and position one of that TO the sum of element at position i of dp and position j of that and position one of that PLUS element at position ni of dp and position nj of that and position one of that MODULO MOD  
                        END IF  
                    END IF  
                END FOR  
            END FOR  
        END FOR  
        IF element at position zero of dp and position zero of that and position zero of that IS EQUAL TO NEGATIVE_INFINITY  
            RETURN the list containing zero and zero  
        END IF  
        RETURN the list containing element at position zero of dp and position zero of that and position zero of that AND element at position zero of dp and position zero of that and position one of that  
    END FUNCTION  
END CLASS
