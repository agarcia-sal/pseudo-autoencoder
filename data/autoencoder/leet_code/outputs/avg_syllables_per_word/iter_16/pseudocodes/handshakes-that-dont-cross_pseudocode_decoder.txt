CLASS Solution  
    FUNCTION numberOfWays(numPeople)  
        SET MOD TO one billion PLUS seven  
        SET dp TO a list of zeros with length equal to numPeople PLUS one  
        SET element at position zero of dp TO one  
        SET element at position two of dp TO one  
        FOR i FROM four TO numPeople IN STEPS OF two  
            FOR j FROM zero TO i MINUS one IN STEPS OF two  
                SET element at position i of dp TO the remainder of the sum of element at position i of dp PLUS the product of element at position j of dp AND element at position i MINUS two MINUS j of dp DIVIDED BY MOD  
            END FOR  
        END FOR  
        RETURN element at position numPeople of dp  
    END FUNCTION  
END CLASS
