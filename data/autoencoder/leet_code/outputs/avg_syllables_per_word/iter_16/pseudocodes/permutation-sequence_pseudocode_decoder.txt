CLASS Solution  
    FUNCTION getPermutation WITH PARAMETERS n AND k  
        SET numbers TO list of integers FROM one TO n  
        DECREMENT k BY one to adjust zero indexing  
        SET result TO empty list  
        FOR i FROM n DOWN TO one  
            SET fact TO factorial of i MINUS one  
            SET index TO integer division of k BY fact  
            APPEND string conversion of element at position index of numbers TO result  
            REMOVE element at position index FROM numbers  
            SET k TO remainder of k DIVIDED BY fact  
        END FOR  
        RETURN concatenation of all elements in result as a single string  
    END FUNCTION  
END CLASS
