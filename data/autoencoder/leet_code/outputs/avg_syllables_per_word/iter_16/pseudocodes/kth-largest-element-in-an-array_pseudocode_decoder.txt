CLASS Solution  
    FUNCTION findKthLargest with parameters nums and k  
        SET min_heap TO empty list  
        FOR each num IN nums  
            CALL heappush FUNCTION with parameters min_heap and num to add num into min_heap as a heap structure  
            IF the LENGTH OF min_heap GREATER THAN k  
                CALL heappop FUNCTION with parameter min_heap to remove the smallest element from min_heap  
            END IF  
        END FOR  
        RETURN the element at position zero of min_heap  
    END FUNCTION  
END CLASS
