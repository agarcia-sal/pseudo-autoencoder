CLASS Solution  
    FUNCTION minInsertions WITH PARAMETER s  
        FUNCTION longest_palindromic_subsequence WITH PARAMETER s  
            SET n TO the LENGTH OF s  
            SET dp TO a two dimensional array with n rows and n columns each element initialized TO zero  
            FOR each i FROM zero TO n MINUS one  
                SET element at position i of dp at position i TO one  
            END FOR  
            FOR each cl FROM two TO n  
                FOR each i FROM zero TO n MINUS cl  
                    SET j TO i PLUS cl MINUS one  
                    IF character at position i of s EQUALS character at position j of s AND cl EQUALS two  
                        SET element at position i of dp at position j TO two  
                    ELSE IF character at position i of s EQUALS character at position j of s  
                        SET element at position i of dp at position j TO element at position i PLUS one of dp at position j MINUS one PLUS two  
                    ELSE  
                        SET element at position i of dp at position j TO the GREATER OF element at position i of dp at position j MINUS one AND element at position i PLUS one of dp at position j  
                    END IF  
                END FOR  
            END FOR  
            RETURN element at position zero of dp at position n MINUS one  
        END FUNCTION  
        RETURN the LENGTH OF s MINUS longest_palindromic_subsequence WITH ARGUMENT s  
    END FUNCTION  
END CLASS
