CLASS Solution  
    FUNCTION minimumObstacles WITH PARAMETERS grid  
        SET m TO the LENGTH OF grid  
        SET n TO the LENGTH OF element at position zero of grid  
        SET directions TO the LIST of tuples with values zero AND one AND one AND zero AND zero AND negative one AND negative one AND zero  
        SET min_heap TO the LIST containing a tuple of zero zero zero representing cost x and y  
        SET visited TO a SET containing a tuple of zero zero representing x and y  
        
        WHILE min_heap is NOT empty  
            EXTRACT the smallest element from min_heap representing cost x and y  
            
            IF x EQUALS m MINUS one AND y EQUALS n MINUS one  
                RETURN cost  
            END IF  
            
            FOR each pair dx AND dy IN directions  
                SET nx TO x PLUS dx  
                SET ny TO y PLUS dy  
                IF nx IS GREATER THAN OR EQUAL TO zero AND nx IS LESS THAN m AND ny IS GREATER THAN OR EQUAL TO zero AND ny IS LESS THAN n AND the tuple nx ny IS NOT IN visited  
                    ADD the tuple nx ny TO visited  
                    SET new_cost TO cost PLUS element at position nx of grid THEN element at position ny of that row  
                    ADD the tuple new_cost nx ny TO min_heap  
                END IF  
            END FOR  
        END WHILE  
        
        RETURN negative one  
    END FUNCTION  
END CLASS
