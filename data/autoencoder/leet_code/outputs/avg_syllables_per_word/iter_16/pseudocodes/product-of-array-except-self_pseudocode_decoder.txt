CLASS Solution  
    FUNCTION productExceptSelf(nums)  
        SET n TO the LENGTH OF nums  
        SET answer TO a list of ones with the LENGTH n  
        SET left_product TO one  
        FOR each index i FROM zero TO n MINUS one  
            SET element at position i of answer TO left_product  
            SET left_product TO left_product MULTIPLIED BY element at position i of nums  
        END FOR  
        SET right_product TO one  
        FOR each index i FROM n MINUS one DOWN TO zero  
            SET element at position i of answer TO element at position i of answer MULTIPLIED BY right_product  
            SET right_product TO right_product MULTIPLIED BY element at position i of nums  
        END FOR  
        RETURN answer  
    END FUNCTION  
END CLASS
