CLASS Solution  
    FUNCTION monotoneIncreasingDigits(n)  
        SET digits TO list OF the string representation of n  
        SET length TO the LENGTH OF digits  
        SET marker TO length  
        FOR i FROM length MINUS one TO one WITH STEP minus one  
            IF element at position i of digits IS LESS THAN element at position i MINUS one of digits  
                SET element at position i MINUS one of digits TO the string representation of the integer conversion of element at position i MINUS one of digits MINUS one  
                SET marker TO i  
            END IF  
        END FOR  
        FOR i FROM marker TO length MINUS one  
            SET element at position i of digits TO the string NINE  
        END FOR  
        RETURN the integer conversion of the concatenation of all elements of digits  
    END FUNCTION  
END CLASS
