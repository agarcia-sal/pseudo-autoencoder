CLASS Solution  
    FUNCTION minBuildTime(blocks AS List of integers, split AS integer)  
        CALL heapify ON blocks TO CONVERT blocks INTO A MINIMUM HEAP  
        WHILE the LENGTH OF blocks IS GREATER THAN one  
            SET first TO the SMALLEST element REMOVED FROM blocks USING HEAP POP operation  
            SET second TO the NEXT SMALLEST element REMOVED FROM blocks USING HEAP POP operation  
            SET combined_time TO split PLUS second  
            ADD combined_time BACK INTO blocks USING HEAP PUSH operation  
        END WHILE  
        RETURN the ELEMENT AT POSITION zero OF blocks  
    END FUNCTION  
END CLASS
