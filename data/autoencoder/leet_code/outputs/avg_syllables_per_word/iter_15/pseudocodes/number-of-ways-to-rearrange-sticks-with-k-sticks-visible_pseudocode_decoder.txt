CLASS Solution  
    FUNCTION rearrangeSticks WITH PARAMETERS n AND k  
        SET MODULO VALUE TO one billion PLUS seven  
        INITIALIZE two dimensional list dp WITH zero VALUES FOR n PLUS one ROWS AND k PLUS one COLUMNS  
        SET element at position zero zero of dp TO one  
        FOR i FROM one TO n  
            FOR j FROM one TO k  
                SET element at position i j of dp TO the REMAINDER WHEN the SUM OF element at position i MINUS one j MINUS one of dp AND the PRODUCT OF i MINUS one AND element at position i MINUS one j of dp IS DIVIDED BY MODULO VALUE  
            END FOR  
        END FOR  
        RETURN element at position n k of dp  
    END FUNCTION  
END CLASS
