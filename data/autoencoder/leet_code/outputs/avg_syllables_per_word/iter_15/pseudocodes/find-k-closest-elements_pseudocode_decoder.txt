CLASS Solution  
    FUNCTION findClosestElements(arr WITH TYPE List OF integers, k WITH TYPE integer, x WITH TYPE integer) RETURNS List OF integers  
        SET left TO result of applying bisect left function to arr and x MINUS one  
        SET right TO left PLUS one  
        WHILE the value of right MINUS left MINUS one IS LESS THAN k  
            IF left EQUALS negative one OR the value of right IS LESS THAN the LENGTH OF arr AND the absolute value of element at position right of arr MINUS x IS LESS THAN the absolute value of element at position left of arr MINUS x  
                INCREMENT right BY one  
            ELSE  
                DECREMENT left BY one  
            END IF  
        END WHILE  
        RETURN slice of arr from position left PLUS one UP TO position right  
    END FUNCTION  
END CLASS
