CLASS Solution  
    FUNCTION checkRecord WITH PARAMETER n  
        SET MOD TO one billion PLUS seven  
        
        CALL initialize three dimensional list dp WITH DIMENSIONS n PLUS one BY two BY three FILLED WITH zero  
        
        SET element at position zero zero zero of dp TO one  
        
        FOR i FROM one TO n  
            FOR j FROM zero TO one  
                FOR k FROM zero TO two  
                    SET element at position i j zero of dp TO element at position i j zero of dp PLUS element at position i MINUS one j k of dp MODULO MOD  
                    
                    IF j EQUALS zero  
                        SET element at position i one zero of dp TO element at position i one zero of dp PLUS element at position i MINUS one zero k of dp MODULO MOD  
                    END IF  
                    
                    IF k LESS THAN two  
                        SET element at position i j k PLUS one of dp TO element at position i j k PLUS one of dp PLUS element at position i MINUS one j k of dp MODULO MOD  
                    END IF  
                END FOR  
            END FOR  
        END FOR  
        
        SET result TO zero  
        FOR j FROM zero TO one  
            FOR k FROM zero TO two  
                SET result TO result PLUS element at position n j k of dp MODULO MOD  
            END FOR  
        END FOR  
        
        RETURN result  
    END FUNCTION  
END CLASS
