CLASS Solution  
    FUNCTION distinctSubseqII WITH PARAMETER s  
        SET MOD TO one billion PLUS seven  
        SET n TO LENGTH OF s  
        INITIALIZE dp AS a list of zeros with LENGTH n PLUS one  
        SET element at position zero of dp TO one  
        INITIALIZE last AS an empty mapping  
        
        FOR i FROM one TO n  
            SET char TO element at position i MINUS one of s  
            SET element at position i of dp TO element at position i MINUS one of dp MULTIPLIED BY two MODULO MOD  
            
            IF char EXISTS AS A KEY IN last  
                SET element at position i of dp TO element at position i of dp MINUS element at position value at key char in last MINUS one of dp PLUS MODULO MOD  
            END IF  
            
            SET the value at key char in last TO i  
        END FOR  
        
        RETURN element at position n of dp MINUS one MODULO MOD  
    END FUNCTION  
END CLASS
