CLASS Solution  
    FUNCTION originalDigits(s)  
        SET count TO A FREQUENCY MAP OF CHARACTERS IN s  
        SET digits TO A LIST OF TEN ZERO VALUES EACH REPRESENTING COUNT OF DIGITS FROM ZERO TO NINE  
        SET the element at position zero of digits TO the COUNT OF character Z in count  
        SET the element at position two of digits TO the COUNT OF character W in count  
        SET the element at position four of digits TO the COUNT OF character U in count  
        SET the element at position six of digits TO the COUNT OF character X in count  
        SET the element at position eight of digits TO the COUNT OF character G in count  
        SET the element at position three of digits TO THE COUNT OF character H in count MINUS the element at position eight of digits  
        SET the element at position five of digits TO THE COUNT OF character F in count MINUS the element at position four of digits  
        SET the element at position seven of digits TO THE COUNT OF character S in count MINUS the element at position six of digits  
        SET the element at position nine of digits TO THE COUNT OF character I in count MINUS the element at position five of digits MINUS the element at position six of digits MINUS the element at position eight of digits  
        SET the element at position one of digits TO THE COUNT OF character O in count MINUS the element at position zero of digits MINUS the element at position two of digits MINUS the element at position four of digits  
        SET result TO AN EMPTY LIST  
        FOR each i FROM zero TO nine  
            APPEND THE STRING REPRESENTATION OF i REPEATED the element at position i of digits TIMES TO result  
        END FOR  
        RETURN THE CONCATENATION OF ALL ELEMENTS IN result  
    END FUNCTION  
END CLASS
