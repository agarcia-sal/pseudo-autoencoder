CLASS Solution  
    FUNCTION numTrees WITH PARAMETER n  
        SET dp TO a new list of zero repeated the sum of n PLUS one times  
        SET element at position zero of dp TO one  
        SET element at position one of dp TO one  
        FOR i FROM two TO n  
            FOR j FROM one TO i  
                SET element at position i of dp TO element at position i of dp PLUS the product of element at position j MINUS one of dp AND element at position i MINUS j of dp  
            END FOR  
        END FOR  
        RETURN element at position n of dp  
    END FUNCTION  
END CLASS
