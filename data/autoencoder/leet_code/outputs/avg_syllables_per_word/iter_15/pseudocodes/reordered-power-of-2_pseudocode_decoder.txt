CLASS Solution  
    FUNCTION reorderedPowerOf2 WITH PARAMETERS n  
        SET power_of_2_sorted_digits TO an empty set of tuples of digits  
        SET power_of_2 TO one  
        WHILE power_of_2 IS LESS THAN OR EQUAL TO ten RAISED TO THE POWER OF nine  
            ADD the tuple of digits in the sorted sequence of the string representation of power_of_2 TO power_of_2_sorted_digits  
            SET power_of_2 TO power_of_2 MULTIPLIED BY two  
        END WHILE  
        SET sorted_n_digits TO the tuple of digits in the sorted sequence of the string representation of n  
        RETURN the value that sorted_n_digits IS A MEMBER OF power_of_2_sorted_digits  
    END FUNCTION  
END CLASS
