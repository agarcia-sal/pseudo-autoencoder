CLASS Solution  
    FUNCTION stoneGameIII(stoneValue)  
        SET n TO the LENGTH OF stoneValue  
        SET dp TO a list of ZERO of the LENGTH OF n PLUS one  
        
        FOR i FROM n MINUS one DOWN TO zero  
            SET dp at position i TO element at position i of stoneValue MINUS dp at position i PLUS one  
            
            IF i PLUS two IS LESS THAN OR EQUAL TO n  
                SET temp_score TO element at position i of stoneValue PLUS element at position i PLUS one of stoneValue MINUS dp at position i PLUS two  
                SET dp at position i TO the GREATER OF dp at position i AND temp_score  
            END IF  
            
            IF i PLUS three IS LESS THAN OR EQUAL TO n  
                SET temp_score TO element at position i of stoneValue PLUS element at position i PLUS one of stoneValue PLUS element at position i PLUS two of stoneValue MINUS dp at position i PLUS three  
                SET dp at position i TO the GREATER OF dp at position i AND temp_score  
            END IF  
        END FOR  
        
        IF dp at position zero IS GREATER THAN zero  
            RETURN the string Alice  
        ELSE IF dp at position zero IS LESS THAN zero  
            RETURN the string Bob  
        ELSE  
            RETURN the string Tie  
        END IF  
    END FUNCTION  
END CLASS
