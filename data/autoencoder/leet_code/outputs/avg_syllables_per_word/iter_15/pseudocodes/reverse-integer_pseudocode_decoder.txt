CLASS Solution  
    FUNCTION reverse WITH parameter x  
        SET sign TO negative one IF x LESS THAN zero ELSE positive one  
        SET reversed_x TO integer CONVERSION OF string CONVERSION OF absolute value of x REVERSED  
        SET result TO sign MULTIPLIED BY reversed_x  
        IF result LESS THAN negative two RAISED TO THE power OF thirty ONE OR result GREATER THAN two RAISED TO THE power OF thirty ONE MINUS one  
            RETURN zero  
        END IF  
        RETURN result  
    END FUNCTION  
END CLASS
