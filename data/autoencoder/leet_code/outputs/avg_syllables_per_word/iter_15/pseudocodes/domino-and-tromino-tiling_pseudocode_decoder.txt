CLASS Solution  
    FUNCTION numTilings(n)  
        SET MODULO TO one billion PLUS seven  
        SET dp TO a list filled with zero of LENGTH n PLUS one  
        SET dp1 TO a list filled with zero of LENGTH n PLUS one  
        SET element at position zero of dp TO one  
        SET element at position one of dp TO one  
        IF n IS GREATER THAN OR EQUAL TO two  
            SET element at position two of dp TO two  
            SET element at position two of dp1 TO one  
        END IF  
        FOR index FROM three TO n  
            SET element at position index of dp TO the SUM of element at position index MINUS one of dp PLUS element at position index MINUS two of dp PLUS two TIMES element at position index MINUS one of dp1 THEN TAKE MODULO MODULO  
            SET element at position index of dp1 TO the SUM of element at position index MINUS two of dp PLUS element at position index MINUS one of dp1 THEN TAKE MODULO MODULO  
        END FOR  
        RETURN element at position n of dp  
    END FUNCTION  
END CLASS
