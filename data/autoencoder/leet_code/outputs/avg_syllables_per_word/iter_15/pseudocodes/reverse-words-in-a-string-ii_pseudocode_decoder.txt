CLASS Solution  
    FUNCTION reverseWords WITH PARAMETER s AS list of strings  
        CALL reverse on s to invert the order of all elements in s  
        SET start TO zero  
        FOR end FROM zero TO LENGTH OF s MINUS one  
            IF element at position end of s EQUALS a single space character  
                CALL reverse on the subsection of s from position start TO position end MINUS one to invert that subsection  
                SET start TO end PLUS one  
            END IF  
        END FOR  
        CALL reverse on the subsection of s from position start TO the last position in s to invert the last word  
    END FUNCTION  
END CLASS
