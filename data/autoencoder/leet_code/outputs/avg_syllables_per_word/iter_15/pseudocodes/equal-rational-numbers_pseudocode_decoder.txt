CLASS Solution  
    FUNCTION isRationalEqual WITH PARAMETERS s AS string AND t AS string RETURNS boolean  
        SET ratios TO list containing one THEN one DIVIDED BY nine THEN one DIVIDED BY ninety nine THEN one DIVIDED BY nine hundred ninety nine THEN one DIVIDED BY nine thousand nine hundred ninety nine

        FUNCTION valueOf WITH PARAMETER s AS string RETURNS floating point number  
            IF position of the first occurrence of opening parenthesis in s EQUALS minus one THEN  
                RETURN numerical value converted from s AS floating point number  
            END IF  

            SET leftParenIndex TO position of first occurrence of opening parenthesis in s  
            SET rightParenIndex TO position of first occurrence of closing parenthesis in s  
            SET dotIndex TO position of first occurrence of decimal point in s  

            SET integerAndNonRepeating TO numerical value converted from substring of s from start TO position leftParenIndex MINUS one  
            SET nonRepeatingLength TO leftParenIndex MINUS dotIndex MINUS one  

            SET repeating TO integer converted from substring of s from position leftParenIndex PLUS one TO position rightParenIndex MINUS one  
            SET repeatingLength TO rightParenIndex MINUS leftParenIndex MINUS one  

            RETURN integerAndNonRepeating PLUS repeating MULTIPLIED BY zero point one raised to the power of nonRepeatingLength MULTIPLIED BY element at position repeatingLength of ratios  
        END FUNCTION  

        RETURN absolute value of valueOf applied to s MINUS valueOf applied to t LESS THAN one times ten raised to the power of minus nine  
    END FUNCTION  
END CLASS
