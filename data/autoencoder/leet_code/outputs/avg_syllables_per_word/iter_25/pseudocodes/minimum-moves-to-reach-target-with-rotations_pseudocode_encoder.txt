CLASS Solution  
    FUNCTION minimumMoves WITH PARAMETER grid  
        SET n TO the LENGTH OF grid  
        SET queue TO a new double-ended queue containing a single element consisting of the number zero the number zero the number zero the number one and the number zero  
        SET visited TO a new set containing a single tuple consisting of the number zero the number zero the number zero and the number one  

        WHILE the LENGTH OF queue is greater than zero  
            REMOVE and RETURN the element at the front of queue SETTING r1 TO the first value c1 TO the second value r2 TO the third value c2 TO the fourth value and steps TO the fifth value  

            IF the tuple consisting of r1 c1 r2 and c2 EQUALS the tuple consisting of n MINUS one n MINUS two n MINUS one and n MINUS one  
                RETURN steps  
            END IF  

            IF c2 PLUS one LESS THAN n AND the element at position r1 of grid at position c1 PLUS one EQUALS zero AND the element at position r2 of grid at position c2 PLUS one EQUALS zero AND the tuple consisting of r1 c1 PLUS one r2 and c2 PLUS one is NOT IN visited  
                ADD the tuple consisting of r1 c1 PLUS one r2 and c2 PLUS one TO visited  
                APPEND to queue the tuple consisting of r1 c1 PLUS one r2 c2 PLUS one and steps PLUS one  
            END IF  

            IF r2 PLUS one LESS THAN n AND the element at position r1 PLUS one of grid at position c1 EQUALS zero AND the element at position r2 PLUS one of grid at position c2 EQUALS zero AND the tuple consisting of r1 PLUS one c1 r2 PLUS one and c2 is NOT IN visited  
                ADD the tuple consisting of r1 PLUS one c1 r2 PLUS one and c2 TO visited  
                APPEND to queue the tuple consisting of r1 PLUS one c1 r2 PLUS one c2 and steps PLUS one  
            END IF  

            IF r1 EQUALS r2 AND r1 PLUS one LESS THAN n AND the element at position r1 PLUS one of grid at position c1 EQUALS zero AND the element at position r1 PLUS one of grid at position c2 EQUALS zero AND the tuple consisting of r1 c1 r1 PLUS one and c1 is NOT IN visited  
                ADD the tuple consisting of r1 c1 r1 PLUS one and c1 TO visited  
                APPEND to queue the tuple consisting of r1 c1 r1 PLUS one c1 and steps PLUS one  
            END IF  

            IF c1 EQUALS c2 AND c1 PLUS one LESS THAN n AND the element at position r1 of grid at position c1 PLUS one EQUALS zero AND the element at position r2 of grid at position c2 PLUS one EQUALS zero AND the tuple consisting of r1 c1 r1 and c1 PLUS one is NOT IN visited  
                ADD the tuple consisting of r1 c1 r1 and c1 PLUS one TO visited  
                APPEND to queue the tuple consisting of r1 c1 r1 c1 PLUS one and steps PLUS one  
            END IF  
        END WHILE  

        RETURN negative one  
    END FUNCTION  
END CLASS
