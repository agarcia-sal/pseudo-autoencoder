CLASS Solution  
    FUNCTION numberOfUniqueGoodSubsequences WITH PARAMETER binary  
        SET MOD TO ten TO THE POWER OF nine PLUS one  
        SET end_with_0 TO zero  
        SET end_with_1 TO zero  
        SET has_zero TO WHETHER the character zero IS PRESENT IN binary  
        FOR each bit IN binary  
            IF bit EQUALS one  
                SET end_with_1 TO the REMAINDER WHEN the SUM of end_with_0 PLUS end_with_1 PLUS one IS DIVIDED BY MOD  
            ELSE IF bit NOT EQUALS one  
                SET end_with_0 TO the REMAINDER WHEN the SUM of end_with_0 PLUS end_with_1 IS DIVIDED BY MOD  
            END IF  
        END FOR  
        RETURN the REMAINDER WHEN the SUM of end_with_0 PLUS end_with_1 PLUS has_zero IS DIVIDED BY MOD  
    END FUNCTION  
END CLASS
