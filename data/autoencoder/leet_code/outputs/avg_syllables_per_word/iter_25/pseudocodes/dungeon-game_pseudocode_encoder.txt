CLASS Solution  
    FUNCTION calculateMinimumHP WITH PARAMETER dungeon  
        SET m TO the LENGTH OF dungeon  
        SET n TO the LENGTH OF element at position zero of dungeon  
        SET dp TO a list of size m PLUS one where each element is a list of size n PLUS one filled each with positive infinity  
        SET element at position m of dp and position n MINUS one TO one  
        SET element at position m MINUS one of dp and position n TO one  
        FOR i FROM m MINUS one DOWNTO zero  
            FOR j FROM n MINUS one DOWNTO zero  
                SET min_health TO the MINIMUM of element at position i PLUS one of dp and position j AND element at position i of dp and position j PLUS one MINUS the element at position i of dungeon and position j  
                SET element at position i of dp and position j TO the MAXIMUM of min_health AND one  
            END FOR  
        END FOR  
        RETURN element at position zero of dp and position zero  
    END FUNCTION  
END CLASS
