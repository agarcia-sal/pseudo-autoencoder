CLASS Solution  
    FUNCTION wiggleSort WITH PARAMETER nums  
        CALL sorting of nums into ascending order  
        SET sorted_nums TO a copy of nums from the beginning to the end  
        SET n TO the LENGTH OF nums  
        SET left TO the RESULT OF the subtraction of one FROM n THEN INTEGER DIVIDED BY two  
        SET right TO the RESULT OF n MINUS one  
        FOR variable i FROM zero TO n MINUS one  
            IF the RESULT OF i MODULO two EQUALS zero  
                SET element at position i of nums TO element at position left of sorted_nums  
                DECREMENT left BY one  
            ELSE  
                SET element at position i of nums TO element at position right of sorted_nums  
                DECREMENT right BY one  
            END IF  
        END FOR  
    END FUNCTION  
END CLASS
