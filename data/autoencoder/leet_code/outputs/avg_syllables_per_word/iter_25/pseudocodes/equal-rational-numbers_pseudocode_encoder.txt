    FUNCTION isRationalEqual WITH PARAMETERS s AND t  
        SET ratios TO a list containing one followed by one DIVIDED BY nine followed by one DIVIDED BY ninety nine followed by one DIVIDED BY nine hundred ninety nine followed by one DIVIDED BY nine thousand nine hundred ninety nine  
        FUNCTION valueOf WITH PARAMETER s  
            IF the position of the opening parenthesis character in s EQUALS negative one  
                RETURN the floating point number represented by s  
            END IF  
            SET leftParenIndex TO the position of the opening parenthesis character in s  
            SET rightParenIndex TO the position of the closing parenthesis character in s  
            SET dotIndex TO the position of the decimal point character in s  
            SET integerAndNonRepeating TO the floating point number represented by the substring of s from the start up to the position leftParenIndex NOT INCLUDING that position  
            SET nonRepeatingLength TO leftParenIndex MINUS dotIndex MINUS one  
            SET repeating TO the integer number represented by the substring of s from the position leftParenIndex PLUS one UP TO the position rightParenIndex NOT INCLUDING that position  
            SET repeatingLength TO rightParenIndex MINUS leftParenIndex MINUS one  
            RETURN integerAndNonRepeating PLUS repeating MULTIPLIED BY zero point one RAISED TO THE POWER OF nonRepeatingLength MULTIPLIED BY the element at position repeatingLength of ratios  
        END FUNCTION  
        RETURN the absolute value of valueOf applied to s MINUS valueOf applied to t LESS THAN one times ten RAISED TO THE POWER OF negative nine  
    END FUNCTION  
END CLASS
