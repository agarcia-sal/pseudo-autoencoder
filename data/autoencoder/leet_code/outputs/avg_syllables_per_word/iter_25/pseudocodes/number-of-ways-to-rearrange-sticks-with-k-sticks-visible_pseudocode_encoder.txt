CLASS Solution  
    FUNCTION rearrangeSticks WITH PARAMETERS n k  
        SET MOD TO one billion PLUS seven  
        CREATE dp AS a two dimensional collection with n PLUS one rows and k PLUS one columns FILLED WITH zero  
        SET element at position zero of element at position zero of dp TO one  
        FOR i FROM one TO n  
            FOR j FROM one TO k  
                SET element at position j of element at position i of dp TO the remainder when the sum of element at position j MINUS one of element at position i MINUS one of dp AND the product of i MINUS one AND element at position j of element at position i MINUS one of dp is divided by MOD  
            END FOR  
        END FOR  
        RETURN element at position k of element at position n of dp  
    END FUNCTION  
END CLASS
