CLASS Solution  
    FUNCTION coinChange WITH PARAMETERS coins AND amount  
        SET dp TO a list where each element is the value of amount PLUS one repeated amount PLUS one times  
        SET the element at position zero of dp TO zero  
        FOR each variable a FROM one TO amount inclusive  
            FOR each element coin IN coins  
                IF variable a MINUS element coin IS GREATER THAN OR EQUAL TO zero  
                    SET the element at position a of dp TO the minimum value BETWEEN the element at position a of dp AND the element at position a MINUS coin of dp PLUS one  
                END IF  
            END FOR  
        END FOR  
        IF the element at position amount of dp IS NOT EQUAL TO amount PLUS one  
            RETURN the element at position amount of dp  
        ELSE  
            RETURN negative one  
        END IF  
    END FUNCTION  
END CLASS
