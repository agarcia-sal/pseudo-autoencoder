CLASS Solution  
    FUNCTION minBuildTime(blocks, split)  
        PERFORM the action of organizing blocks into a structure that always provides the smallest element first  
        WHILE the count of elements in blocks is GREATER THAN one  
            SET first TO the smallest element removed from blocks  
            SET second TO the next smallest element removed from blocks  
            SET combined_time TO split PLUS second  
            INSERT combined_time into blocks maintaining the smallest element first order  
        END WHILE  
        RETURN the element at position zero of blocks  
    END FUNCTION  
END CLASS
