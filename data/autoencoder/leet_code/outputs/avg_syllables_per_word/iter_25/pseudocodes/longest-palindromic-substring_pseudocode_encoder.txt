CLASS Solution  
    FUNCTION longestPalindrome with parameter s  
        SET n TO the LENGTH OF s  
        SET f TO a two dimensional list where for each integer from zero TO one LESS THAN n there is a list of boolean TRUE of LENGTH n  
        SET k TO zero  
        SET mx TO one  
        FOR i FROM n MINUS two DOWN TO zero  
            FOR j FROM i PLUS one TO one LESS THAN n  
                SET the element at row i and column j of f TO FALSE  
                IF the element at position i of s EQUALS the element at position j of s  
                    SET the element at row i and column j of f TO the element at row i PLUS one and column j MINUS one of f  
                    IF the element at row i and column j of f IS TRUE AND mx LESS THAN j MINUS i PLUS one  
                        SET k TO i  
                        SET mx TO j MINUS i PLUS one  
                    END IF  
                END IF  
            END FOR  
        END FOR  
        RETURN the substring of s from position k TO position k PLUS mx MINUS one  
    END FUNCTION  
END CLASS
