CLASS Solution  
    FUNCTION originalDigits WITH PARAMETER s  
        IMPORT collections Counter FROM collections  
        SET count TO Counter OF s  
        SET digits TO a list containing zero repeated ten times  
        SET element at position zero of digits TO count element at position corresponding to letter z  
        SET element at position two of digits TO count element at position corresponding to letter w  
        SET element at position four of digits TO count element at position corresponding to letter u  
        SET element at position six of digits TO count element at position corresponding to letter x  
        SET element at position eight of digits TO count element at position corresponding to letter g  
        SET element at position three of digits TO count element at position corresponding to letter h MINUS element at position eight of digits  
        SET element at position five of digits TO count element at position corresponding to letter f MINUS element at position four of digits  
        SET element at position seven of digits TO count element at position corresponding to letter s MINUS element at position six of digits  
        SET element at position nine of digits TO count element at position corresponding to letter i MINUS element at position five of digits MINUS element at position six of digits MINUS element at position eight of digits  
        SET element at position one of digits TO count element at position corresponding to letter o MINUS element at position zero of digits MINUS element at position two of digits MINUS element at position four of digits  
        SET result TO an empty list  
        FOR i FROM zero TO nine  
            APPEND the string representing i repeated element at position i of digits times TO result  
        END FOR  
        RETURN the concatenation of all elements of result  
    END FUNCTION  
END CLASS
