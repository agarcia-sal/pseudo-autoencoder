CLASS Solution  
    FUNCTION reverse WITH PARAMETER x  
        SET sign TO negative one IF x IS LESS THAN zero OTHERWISE SET sign TO one  
        SET reversed_x TO integer CONVERTION OF the DIGITS OF the ABSOLUTE VALUE OF x IN REVERSE ORDER  
        SET result TO sign MULTIPLIED BY reversed_x  
        IF result IS LESS THAN negative two RAISED TO THE POWER OF thirty ONE OR result IS GREATER THAN two RAISED TO THE POWER OF thirty ONE MINUS one  
            RETURN zero  
        END IF  
        RETURN result  
    END FUNCTION  
END CLASS
