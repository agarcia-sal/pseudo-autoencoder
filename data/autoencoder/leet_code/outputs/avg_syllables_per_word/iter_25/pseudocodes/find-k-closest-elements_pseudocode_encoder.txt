CLASS Solution  
    FUNCTION findClosestElements WITH PARAMETERS arr k x  
        SET left TO the position found by locating where x would be inserted into arr keeping arr sorted MINUS one  
        SET right TO left INCREMENTED BY one  
        WHILE the difference of right MINUS left MINUS one is less than k  
            IF left EQUALS minus one OR the value right is less than the LENGTH OF arr AND the absolute value of the element at position right of arr MINUS x is LESS THAN the absolute value of the element at position left of arr MINUS x  
                INCREMENT right BY one  
            ELSE  
                DECREMENT left BY one  
            END IF  
        END WHILE  
        RETURN the segment of arr from position left INCREMENTED BY one TO position right EXCLUSIVE  
    END FUNCTION  
END CLASS
