CLASS Solution  
    FUNCTION minSwap(nums1, nums2)  
        SET n TO the LENGTH OF nums1  
        SET dp TO a new list of length n where each element is a list containing zero and zero  
        SET the first element at position zero of dp at position zero TO zero  
        SET the first element at position zero of dp at position one TO one  
        FOR i FROM one TO n MINUS one  
            IF the element at position i of nums1 is GREATER THAN the element at position i MINUS one of nums1 AND the element at position i of nums2 is GREATER THAN the element at position i MINUS one of nums2 THEN  
                IF the element at position i of nums1 is GREATER THAN the element at position i MINUS one of nums2 AND the element at position i of nums2 is GREATER THAN the element at position i MINUS one of nums1 THEN  
                    SET the element at position i of dp at position zero TO the MINIMUM OF the element at position i MINUS one of dp at position zero AND the element at position i MINUS one of dp at position one  
                    SET the element at position i of dp at position one TO the element at position i of dp at position zero PLUS one  
                ELSE  
                    SET the element at position i of dp at position zero TO the element at position i MINUS one of dp at position zero  
                    SET the element at position i of dp at position one TO the element at position i MINUS one of dp at position one PLUS one  
                END IF  
            ELSE  
                IF the element at position i of nums1 is GREATER THAN the element at position i MINUS one of nums2 AND the element at position i of nums2 is GREATER THAN the element at position i MINUS one of nums1 THEN  
                    SET the element at position i of dp at position zero TO the element at position i MINUS one of dp at position one  
                    SET the element at position i of dp at position one TO the element at position i MINUS one of dp at position zero PLUS one  
                ELSE  
                    SET the element at position i of dp at position zero TO the element at position i MINUS one of dp at position one  
                    SET the element at position i of dp at position one TO the element at position i MINUS one of dp at position zero PLUS one  
                END IF  
            END IF  
        END FOR  
        RETURN the MINIMUM OF the element at position n MINUS one of dp at position zero AND the element at position n MINUS one of dp at position one  
    END FUNCTION  
END CLASS
