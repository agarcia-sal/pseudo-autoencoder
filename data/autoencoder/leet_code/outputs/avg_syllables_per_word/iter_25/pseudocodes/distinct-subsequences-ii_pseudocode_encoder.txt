CLASS Solution  
    FUNCTION distinctSubseqII WITH PARAMETER s  
        SET MOD TO ten raised to the power of nine PLUS one  
        SET n TO the LENGTH OF s  
        SET dp TO a list of zeros with LENGTH n PLUS one  
        SET the element at position zero of dp TO one  
        SET last TO an empty dictionary  
        FOR i FROM one TO n  
            SET char TO the element at position i MINUS one of s  
            SET the element at position i of dp TO the element at position i MINUS one of dp MULTIPLIED BY two THEN MODULO MOD  
            IF char EXISTS IN last  
                SET the element at position i of dp TO the element at position i of dp MINUS the element at position last of char MINUS one of dp PLUS MOD THEN MODULO MOD  
            END IF  
            SET the value of last at key char TO i  
        END FOR  
        RETURN the element at position n of dp MINUS one THEN MODULO MOD  
    END FUNCTION  
END CLASS
