CLASS Solution  
    FUNCTION rangeBitwiseAnd WITH PARAMETERS left right  
        SET shift TO zero  
        WHILE left LESS THAN right  
            SET left TO the result of left DIVIDED BY two discarding remainder  
            SET right TO the result of right DIVIDED BY two discarding remainder  
            INCREMENT shift BY one  
        END WHILE  
        RETURN the result of left MULTIPLIED BY two RAISED TO THE POWER OF shift  
    END FUNCTION  
END CLASS
