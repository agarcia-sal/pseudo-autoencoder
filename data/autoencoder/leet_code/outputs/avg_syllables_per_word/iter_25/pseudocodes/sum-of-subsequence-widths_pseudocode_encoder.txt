CLASS Solution  
    FUNCTION sumSubseqWidths WITH PARAMETER nums  
        SET MOD TO ten TO THE POWER OF nine PLUS seven  
        ARRANGE nums IN ascending order  
        SET n TO the LENGTH OF nums  
        SET total_sum TO zero  
        CREATE list power WITH n ELEMENTS EACH SET TO one  
        FOR i FROM one TO n MINUS one  
            SET element at position i of power TO the REMAINDER WHEN element at position i MINUS one of power MULTIPLIED BY two IS DIVIDED BY MOD  
        END FOR  
        FOR i FROM zero TO n MINUS one  
            SET total_sum TO the REMAINDER WHEN total_sum PLUS element at position i of nums MULTIPLIED BY the DIFFERENCE BETWEEN element at position i of power AND element at position n MINUS i MINUS one of power IS DIVIDED BY MOD  
        END FOR  
        RETURN total_sum  
    END FUNCTION  
END CLASS
