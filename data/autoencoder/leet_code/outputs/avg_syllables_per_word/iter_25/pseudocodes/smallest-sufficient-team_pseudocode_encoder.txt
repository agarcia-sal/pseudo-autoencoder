CLASS Solution  
    FUNCTION smallestSufficientTeam WITH PARAMETERS self req_skills people  
        CREATE a dictionary d mapping each skill s to its index i where i and s are the index and element from enumerating req_skills  
        SET m TO the length of req_skills  
        SET n TO the length of people  
        CREATE a list p of length n filled with zero  
        FOR each index i and skill list ss from enumerating people  
            FOR each skill s IN ss  
                SET element at index i of p TO element at index i of p BITWISE OR one shifted left by the value at key s in d  
            END FOR  
        END FOR  
        CREATE a list f of length two raised to the power of m filled with infinity  
        CREATE a list g of length two raised to the power of m filled with zero  
        CREATE a list h of length two raised to the power of m filled with zero  
        SET element at index zero of f TO zero  
        FOR each integer i FROM zero TO two raised to the power of m MINUS one  
            IF element at index i of f EQUALS infinity  
                CONTINUE to next iteration  
            END IF  
            FOR each integer j FROM zero TO n MINUS one  
                IF element at index i of f PLUS one LESS THAN element at index i BITWISE OR element at index j of p of f  
                    SET element at index i BITWISE OR element at index j of p of f TO element at index i of f PLUS one  
                    SET element at index i BITWISE OR element at index j of p of g TO j  
                    SET element at index i BITWISE OR element at index j of p of h TO i  
                END IF  
            END FOR  
        END FOR  
        SET i TO two raised to the power of m MINUS one  
        CREATE an empty list ans  
        WHILE i is not zero  
            APPEND element at index i of g TO ans  
            SET i TO element at index i of h  
        END WHILE  
        RETURN ans  
    END FUNCTION  
END CLASS
