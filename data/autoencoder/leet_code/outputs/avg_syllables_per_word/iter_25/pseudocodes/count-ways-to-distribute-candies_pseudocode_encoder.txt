CLASS Solution  
    FUNCTION waysToDistribute WITH PARAMETERS n k  
        SET MOD TO one billion PLUS seven  
        SET dp TO a list of lists each containing zero repeated k PLUS one times repeated n PLUS one times  
        SET element at position zero of list at position zero of dp TO one  
        FOR i FROM one TO n  
            FOR j FROM one TO k  
                SET element at position j of list at position i of dp TO the remainder when the sum of j MULTIPLIED BY element at position j of list at position i MINUS one of dp and element at position j MINUS one of list at position i MINUS one of dp is divided by MOD  
            END FOR  
        END FOR  
        RETURN element at position k of list at position n of dp  
    END FUNCTION  
END CLASS
