CLASS Solution  
    FUNCTION reverseWords WITH PARAMETER s  
        CALL the ACTION to reverse the order of all elements in s  
        SET start TO zero  
        FOR end FROM zero TO the LENGTH OF s MINUS one  
            IF the element at position end of s EQUALS a space character  
                CALL the ACTION to reverse the order of elements starting at position start UP TO position end MINUS one of s  
                SET start TO end PLUS one  
            END IF  
        END FOR  
        CALL the ACTION to reverse the order of elements starting at position start TO the last element of s  
    END FUNCTION  
END CLASS
