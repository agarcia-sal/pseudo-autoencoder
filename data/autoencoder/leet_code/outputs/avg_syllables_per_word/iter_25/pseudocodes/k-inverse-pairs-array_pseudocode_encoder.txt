CLASS Solution  
    FUNCTION kInversePairs WITH PARAMETERS n AND k  
        SET MOD TO ten MULTIPLIED BY nine MULTIPLIED BY nine MULTIPLIED BY nine PLUS seven  
        SET dp TO a two dimensional list with each element zero with the number of rows equal to n PLUS one and the number of columns equal to k PLUS one  
        SET the element at row zero and column zero of dp TO one  
        FOR i FROM one TO n  
            SET prefix_sum TO zero  
            FOR j FROM zero TO k  
                SET prefix_sum TO prefix_sum PLUS the element at row i MINUS one and column j of dp  
                SET prefix_sum TO prefix_sum MODULO MOD  
                IF j GREATER THAN OR EQUAL TO i THEN  
                    SET prefix_sum TO prefix_sum MINUS the element at row i MINUS one and column j MINUS i of dp PLUS MOD  
                    SET prefix_sum TO prefix_sum MODULO MOD  
                END IF  
                SET the element at row i and column j of dp TO prefix_sum  
            END FOR  
        END FOR  
        RETURN the element at row n and column k of dp  
    END FUNCTION  
END CLASS
