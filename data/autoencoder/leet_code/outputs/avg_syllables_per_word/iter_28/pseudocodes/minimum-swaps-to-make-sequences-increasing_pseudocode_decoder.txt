CLASS Solution  
    FUNCTION minSwap(nums1 nums1 OF TYPE List OF int, nums2 nums2 OF TYPE List OF int) RETURNS int  
        SET n TO the LENGTH OF nums1  
        CREATE dp AS a LIST OF LISTS CONTAINING two zeroes EACH WITH LENGTH n  

        SET element at position zero of dp WITH index zero TO zero  
        SET element at position zero of dp WITH index one TO one  

        FOR i FROM one TO n MINUS one  
            IF element at position i of nums1 GREATER THAN element at position i MINUS one of nums1 AND element at position i of nums2 GREATER THAN element at position i MINUS one of nums2  
                IF element at position i of nums1 GREATER THAN element at position i MINUS one of nums2 AND element at position i of nums2 GREATER THAN element at position i MINUS one of nums1  
                    SET element at position i of dp WITH index zero TO THE MINIMUM VALUE BETWEEN element at position i MINUS one of dp WITH index zero AND element at position i MINUS one of dp WITH index one  
                    SET element at position i of dp WITH index one TO element at position i of dp WITH index zero PLUS one  
                ELSE  
                    SET element at position i of dp WITH index zero TO element at position i MINUS one of dp WITH index zero  
                    SET element at position i of dp WITH index one TO element at position i MINUS one of dp WITH index one PLUS one  
                END IF  
            ELSE  
                IF element at position i of nums1 GREATER THAN element at position i MINUS one of nums2 AND element at position i of nums2 GREATER THAN element at position i MINUS one of nums1  
                    SET element at position i of dp WITH index zero TO element at position i MINUS one of dp WITH index one  
                    SET element at position i of dp WITH index one TO element at position i MINUS one of dp WITH index zero PLUS one  
                ELSE  
                    SET element at position i of dp WITH index zero TO element at position i MINUS one of dp WITH index one  
                    SET element at position i of dp WITH index one TO element at position i MINUS one of dp WITH index zero PLUS one  
                END IF  
            END IF  
        END FOR  

        RETURN THE MINIMUM VALUE BETWEEN element at position n MINUS one of dp WITH index zero AND element at position n MINUS one of dp WITH index one  
    END FUNCTION  
END CLASS
