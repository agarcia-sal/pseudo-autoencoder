CLASS Solution  
    FUNCTION numMusicPlaylists WITH PARAMETERS n goal k  
        SET MOD TO ten TO THE POWER OF nine PLUS seven  
        INITIALIZE dp AS list containing goal PLUS one elements EACH BEING a list containing n PLUS one zeros  
        SET element at position zero of list at position zero of dp TO one  
        FOR i FROM one TO goal  
            FOR j FROM one TO n  
                INCREMENT element at position j of list at position i of dp BY element at position j MINUS one of list at position i MINUS one of dp MULTIPLIED BY n MINUS j MINUS one PLUS one  
                SET element at position j of list at position i of dp TO element at position j of list at position i of dp MODULO MOD  
                IF j GREATER THAN k THEN  
                    INCREMENT element at position j of list at position i of dp BY element at position j of list at position i MINUS one of dp MULTIPLIED BY j MINUS k  
                    SET element at position j of list at position i of dp TO element at position j of list at position i of dp MODULO MOD  
                END IF  
            END FOR  
        END FOR  
        RETURN element at position n of list at position goal of dp  
    END FUNCTION  
END CLASS
