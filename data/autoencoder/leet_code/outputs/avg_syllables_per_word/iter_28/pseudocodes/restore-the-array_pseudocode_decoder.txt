CLASS Solution  
    FUNCTION numberOfArrays(s, k)  
        SET MOD TO ten raised to the power of nine PLUS seven  
        SET n TO the LENGTH OF s  
        SET dp TO a list of zeros with the size of n PLUS one  
        SET the element at position zero of dp TO one  
        FOR i FROM one TO n  
            FOR j FROM zero TO i MINUS one  
                IF the character at position j of s NOT EQUALS the character zero AND the integer value of the substring from position j TO position i of s GREATER THAN OR EQUAL TO one AND the integer value of the substring from position j TO position i of s LESS THAN OR EQUAL TO k  
                    SET the element at position i of dp TO the remainder of the sum of the element at position i of dp PLUS the element at position j of dp divided by MOD  
                END IF  
            END FOR  
        END FOR  
        RETURN the element at position n of dp  
    END FUNCTION  
END CLASS
