CLASS Solution  
    FUNCTION rotate(nums parameter, k parameter)  
        SET n TO the LENGTH OF nums parameter  
        SET k TO the REMAINDER of k parameter DIVIDED BY n  
        
        FUNCTION reverse(start parameter, end parameter)  
            WHILE start parameter LESS THAN end parameter  
                SET temporary variable TO element at position start parameter of nums parameter  
                SET element at position start parameter of nums parameter TO element at position end parameter of nums parameter  
                SET element at position end parameter of nums parameter TO temporary variable  
                INCREMENT start parameter BY one  
                DECREMENT end parameter BY one  
            END WHILE  
        END FUNCTION  
        
        CALL reverse WITH start parameter EQUAL TO zero AND end parameter EQUAL TO n MINUS one  
        CALL reverse WITH start parameter EQUAL TO zero AND end parameter EQUAL TO k MINUS one  
        CALL reverse WITH start parameter EQUAL TO k AND end parameter EQUAL TO n MINUS one  
    END FUNCTION  
END CLASS
