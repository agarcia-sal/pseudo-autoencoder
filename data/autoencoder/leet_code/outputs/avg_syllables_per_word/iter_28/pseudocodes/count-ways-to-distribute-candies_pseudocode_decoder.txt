CLASS Solution  
    FUNCTION waysToDistribute(n, k)  
        SET MOD TO one billion PLUS seven  
        SET dp TO a list containing n PLUS one elements each of which is a list containing k PLUS one zeros  
        SET the element at position zero of dp to a list containing k PLUS one elements each of which is zero  
        SET the element at position zero of the element at position zero of dp TO one  
        FOR i FROM one TO n  
            FOR j FROM one TO k  
                SET the element at position j of the element at position i of dp TO the remainder after division by MOD of the sum of the product of j AND the element at position j of the element at position i MINUS one of dp AND the element at position j MINUS one of the element at position i MINUS one of dp  
            END FOR  
        END FOR  
        RETURN the element at position k of the element at position n of dp  
    END FUNCTION  
END CLASS
