CLASS Solution  
    FUNCTION reverse WITH PARAMETER x  
        SET sign TO negative one IF x LESS THAN zero ELSE positive one  
        SET reversed_x TO convert TO integer the string of the absolute value of x reversed in order  
        SET result TO sign MULTIPLIED BY reversed_x  
        IF result LESS THAN negative two POWER thirty one OR result GREATER THAN positive two POWER thirty one MINUS one  
            RETURN zero  
        END IF  
        RETURN result  
    END FUNCTION  
END CLASS
