CLASS Solution  
    FUNCTION smallestSufficientTeam(req_skills, people)  
        SET d TO mapping of each skill in req_skills to its index  
        SET m TO length of req_skills  
        SET n TO length of people  
        SET p TO list of zeroes of length n  

        FOR each index i and skill_set ss IN people  
            FOR each skill s IN ss  
                SET p at index i TO p at index i combined with bit shifted representation of skill s according to d  

        SET f TO list filled with infinity of length two raised to power m  
        SET g TO list of zeroes of length two raised to power m  
        SET h TO list of zeroes of length two raised to power m  
        SET f at index 0 TO 0  

        FOR each integer i FROM 0 TO two raised to power m minus one  
            IF f at i equals infinity  
                CONTINUE to next iteration  
            FOR each integer j FROM 0 TO n minus one  
                IF f at i plus one less than f at i combined with p at j  
                    SET f at i combined with p at j TO f at i plus one  
                    SET g at i combined with p at j TO j  
                    SET h at i combined with p at j TO i  

        SET i TO two raised to power m minus one  
        SET ans TO empty list  

        WHILE i is not zero  
            APPEND g at i TO ans  
            SET i TO h at i  

        RETURN ans  
    END FUNCTION  
END CLASS
