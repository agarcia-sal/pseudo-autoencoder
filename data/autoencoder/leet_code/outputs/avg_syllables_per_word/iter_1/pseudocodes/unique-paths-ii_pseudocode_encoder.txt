```
if empty grid or start blocked: return 0
m, n = grid dims
dp = zero m√ón matrix
dp[0][0] = 1
for j in 1..n-1: if no obstacle at (0,j): dp[0][j] = dp[0][j-1]
for i in 1..m-1: if no obstacle at (i,0): dp[i][0] = dp[i-1][0]
for i in 1..m-1:
  for j in 1..n-1:
    if no obstacle at (i,j): dp[i][j] = dp[i-1][j] + dp[i][j-1]
return dp[m-1][n-1]
```
