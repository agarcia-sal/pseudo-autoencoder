```
if picture empty return 0
m, n = dims of picture
row_count = [0]*m
col_count = [0]*n

for i in 0..m-1
  for j in 0..n-1
    if picture[i][j] == 'B'
      row_count[i] += 1
      col_count[j] += 1

lonely_pixel_count = 0
for i in 0..m-1
  for j in 0..n-1
    if picture[i][j] == 'B' and row_count[i] == 1 and col_count[j] == 1
      lonely_pixel_count += 1

return lonely_pixel_count
```
