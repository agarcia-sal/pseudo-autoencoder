```
convert stickers to char frequency counters
sort stickers by max freq descending
init queue with (target, 0)
init memo {target:0}

while queue not empty:
    cur, used = dequeue
    for sticker in stickers:
        if cur[0] not in sticker: continue
        rem = counter(cur) - sticker
        new_t = sorted string from rem
        if new_t empty: return used + 1
        if new_t not in memo:
            memo[new_t] = used + 1
            enqueue (new_t, used + 1)

return -1
```
