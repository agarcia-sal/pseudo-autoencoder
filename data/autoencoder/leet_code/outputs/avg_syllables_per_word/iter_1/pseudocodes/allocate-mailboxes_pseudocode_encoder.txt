```
sort houses; n = len(houses)
cost = 2D array n×n zeroed
for i from n-1 down to 0:
  for j from i+1 to n-1:
    cost[i][j] = cost[i+1][j-1] + houses[j] - houses[i]

dp = 2D array (n+1)×(k+1) filled with ∞
dp[0][0] = 0

for i in 1 to n:
  for j in 1 to k:
    for p in 0 to i-1:
      dp[i][j] = min(dp[i][j], dp[p][j-1] + cost[p][i-1])

return dp[n][k]
```
