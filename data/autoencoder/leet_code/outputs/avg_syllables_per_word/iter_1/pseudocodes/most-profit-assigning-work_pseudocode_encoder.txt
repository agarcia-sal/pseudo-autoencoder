```
jobs ← sorted(zip(difficulty, profit)) by difficulty
worker ← sorted(worker)
max_profit ← 0
best_profit ← 0
i ← 0
for ability in worker:
    while i < len(jobs) and jobs[i].difficulty ≤ ability:
        best_profit ← max(best_profit, jobs[i].profit)
        i ← i + 1
    max_profit ← max_profit + best_profit
return max_profit
```
