```
m, n ← dims(grid)
dirs ← [(0,1),(1,0),(0,-1),(-1,0)]
heap ← [(0,0,0)]   // cost, x, y
visited ← {(0,0)}

while heap not empty:
    cost, x, y ← pop heap
    if (x,y) = (m-1,n-1): return cost
    for dx, dy in dirs:
        nx, ny ← x+dx, y+dy
        if inside grid and (nx,ny) not in visited:
            visited.add((nx,ny))
            push (cost+grid[nx][ny], nx, ny) into heap

return -1
```
