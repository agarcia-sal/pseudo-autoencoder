CLASS Solution  
    FUNCTION topKFrequent(words k)  
        SET count TO the frequency count of each element IN words  
        SET heap TO a new list created by FOR each element word AND frequency freq IN count items APPENDING a tuple consisting of the negative of freq AND word  
        APPLY heapify operation TO the heap  
        SET result TO a new list created by FOR a number iterating zero TO the value of k MINUS one APPENDING the second element of the tuple returned by popping the smallest element FROM the heap  
        RETURN result  
    END FUNCTION  
END CLASS
