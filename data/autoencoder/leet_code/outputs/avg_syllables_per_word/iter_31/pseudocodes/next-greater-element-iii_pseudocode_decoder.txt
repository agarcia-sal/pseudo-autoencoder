CLASS Solution  
    FUNCTION nextGreaterElement(n)  
        SET digits TO the list of characters representing the string form of n  
        SET length TO the LENGTH OF digits  
        SET i TO length MINUS two  
        WHILE i GREATER THAN OR EQUAL TO zero AND element at position i of digits GREATER THAN OR EQUAL TO element at position i PLUS one of digits  
            DECREMENT i BY one  
        END WHILE  
        IF i EQUALS negative one  
            RETURN negative one  
        END IF  
        SET j TO length MINUS one  
        WHILE element at position j of digits LESS THAN OR EQUAL TO element at position i of digits  
            DECREMENT j BY one  
        END WHILE  
        SET temporary variable TO element at position i of digits  
        SET element at position i of digits TO element at position j of digits  
        SET element at position j of digits TO temporary variable  
        SET digits TO the concatenation of the elements from position zero TO position i of digits WITH the elements from position i PLUS one TO the end of digits IN reverse order  
        SET result TO the integer value formed by concatenating all elements in digits in order  
        IF result GREATER THAN two RAISED TO THE POWER OF thirty ONE MINUS one  
            RETURN negative one  
        END IF  
        RETURN result  
    END FUNCTION  
END CLASS
