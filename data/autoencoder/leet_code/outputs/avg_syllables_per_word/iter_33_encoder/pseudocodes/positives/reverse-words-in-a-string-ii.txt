CLASS Solution  
    FUNCTION reverseWords(s)  
        CALL reverse the elements of s in place  
        SET start TO zero  
        FOR end FROM zero TO the LENGTH OF s MINUS one  
            IF the element at position end of s EQUALS a space character  
                CALL reverse the elements from position start TO position end MINUS one of s in place  
                SET start TO end PLUS one  
            END IF  
        END FOR  
        CALL reverse the elements from position start TO the last position of s in place  
    END FUNCTION  
END CLASS
