CLASS Solution  
    FUNCTION isRationalEqual(s, t)  
        SET ratios TO list of one followed by one DIVIDED BY nine followed by one DIVIDED BY ninety nine followed by one DIVIDED BY nine hundred ninety nine followed by one DIVIDED BY nine thousand nine hundred ninety nine  

        FUNCTION valueOf(s)  
            IF the POSITION OF character opening parenthesis IN s EQUALS negative one THEN  
                RETURN the conversion of s TO floating point number  
            END IF  

            SET leftParenIndex TO the POSITION OF character opening parenthesis IN s  
            SET rightParenIndex TO the POSITION OF character closing parenthesis IN s  
            SET dotIndex TO the POSITION OF character dot IN s  

            SET integerAndNonRepeating TO the conversion TO floating point number OF the substring OF s FROM the START TO the POSITION leftParenIndex  
            SET nonRepeatingLength TO leftParenIndex MINUS dotIndex MINUS one  

            SET repeating TO the conversion TO integer OF the substring OF s FROM the POSITION leftParenIndex PLUS one TO the POSITION rightParenIndex  
            SET repeatingLength TO rightParenIndex MINUS leftParenIndex MINUS one  

            RETURN integerAndNonRepeating PLUS repeating MULTIPLIED BY zero point one RAISED TO THE POWER OF minus nonRepeatingLength MULTIPLIED BY ratios AT POSITION repeatingLength  
        END FUNCTION  

        RETURN the absolute value OF valueOf CALLED WITH s MINUS valueOf CALLED WITH t LESS THAN one times ten RAISED TO THE POWER OF minus nine  
    END FUNCTION  
END CLASS
