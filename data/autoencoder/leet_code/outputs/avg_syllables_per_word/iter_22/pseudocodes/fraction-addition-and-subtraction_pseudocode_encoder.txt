CLASS Solution  
    FUNCTION fractionAddition(expression)  
        SET list_of_fractions TO the result of replacing every occurrence of the word minus in expression with the word plus followed by minus and splitting the resulting string by the word plus  
        SET result_fraction TO the summation of converting each element IN list_of_fractions that is not empty TO a fraction type  
        RETURN the string composed of the numerator value of result_fraction FOLLOWED BY the forward slash character FOLLOWED BY the denominator value of result_fraction  
    END FUNCTION  
END CLASS
