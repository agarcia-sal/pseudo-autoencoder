CLASS Solution  
    FUNCTION jobScheduling(startTime list OF integers endTime list OF integers profit list OF integers) RETURNS integer  
        SET jobs TO the list obtained by sorting the list of tuples composed of element at position index of startTime element at position index of endTime element at position index of profit for each index from one TO the LENGTH OF startTime according to the second element of each tuple in ascending order  
        SET end_times TO the list composed of the second element of each tuple in jobs for each tuple in jobs  
        SET dp TO a list of zeros with a length EQUAL TO the LENGTH OF jobs PLUS one  
        FOR i FROM one TO the LENGTH OF jobs  
            SET j TO the position where the element at position i MINUS one of jobs at its first element would be inserted in end_times such that the order is maintained and the insertion position is to the right of any existing entries of the same value  
            SET dp at position i TO the greater value BETWEEN dp at position i MINUS one AND dp at position j PLUS the third element of the element at position i MINUS one of jobs  
        END FOR  
        RETURN the last element of dp  
    END FUNCTION  
END CLASS
