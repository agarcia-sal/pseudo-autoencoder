CLASS Solution  
    FUNCTION waysToDistribute(n, k)  
        SET MOD TO the number one followed by nine zeros PLUS the number seven  
        CREATE a two dimensional list named dp with the number of rows equal TO n PLUS one and number of columns equal TO k PLUS one with all elements SET TO zero  
        SET element at position zero of dp at position zero TO one  
        FOR i FROM the number one TO n INCLUSIVE  
            FOR j FROM the number one TO k INCLUSIVE  
                SET element at position i of dp at position j TO the remainder after dividing the sum of the product of j AND element at position i MINUS one of dp at position j AND element at position i MINUS one of dp at position j MINUS one BY MOD  
            END FOR  
        END FOR  
        RETURN element at position n of dp at position k  
    END FUNCTION  
END CLASS
