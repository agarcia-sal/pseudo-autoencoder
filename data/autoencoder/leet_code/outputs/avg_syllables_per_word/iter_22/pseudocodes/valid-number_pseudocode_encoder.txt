CLASS Solution  
    FUNCTION isNumber(s)  
        RETURN the entire string s MATCHES the pattern representing an optional plus or minus sign FOLLOWED BY either one or more digits FOLLOWED BY an optional decimal point FOLLOWED BY zero or more digits OR a decimal point FOLLOWED BY one or more digits FOLLOWED BY an optional exponent part starting with e FOLLOWED BY an optional plus or minus sign FOLLOWED BY one or more digits IS TRUE  
    END FUNCTION  
END CLASS
