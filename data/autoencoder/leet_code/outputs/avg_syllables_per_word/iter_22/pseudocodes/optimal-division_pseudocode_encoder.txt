CLASS Solution  
    FUNCTION optimalDivision(nums)  
        IF the LENGTH OF nums EQUALS one  
            RETURN the ELEMENT at position one of nums CONVERTED TO string  
        END IF  
        IF the LENGTH OF nums EQUALS two  
            RETURN the ELEMENT at position one of nums CONVERTED TO string CONCATENATED WITH the WORD division BY CONCATENATED WITH the ELEMENT at position two of nums CONVERTED TO string  
        END IF  
        RETURN the ELEMENT at position one of nums CONVERTED TO string CONCATENATED WITH the WORD division BY CONCATENATED WITH the OPENING PARENTHESIS CONCATENATED WITH the STRING CREATED BY JOINING each ELEMENT FROM position two TO the END of nums CONVERTED TO string WITH the WORD division BY IN BETWEEN EACH ELEMENT CONCATENATED WITH the CLOSING PARENTHESIS  
    END FUNCTION  
END CLASS
