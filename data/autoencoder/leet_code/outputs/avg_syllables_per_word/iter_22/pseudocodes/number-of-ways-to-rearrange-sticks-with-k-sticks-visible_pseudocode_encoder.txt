CLASS Solution  
    FUNCTION rearrangeSticks(n parameter, k parameter) RETURNS integer  
        SET MOD TO one billion PLUS seven  
        SET dp TO a two dimensional list of zeros with the first dimension size of n PLUS one and the second dimension size of k PLUS one  
        SET element at position zero of dp at position zero TO one  
        FOR i FROM one TO n  
            FOR j FROM one TO k  
                SET value one TO element at position i MINUS one of dp at position j MINUS one  
                SET value two TO i MINUS one MULTIPLIED BY element at position i MINUS one of dp at position j  
                SET element at position i of dp at position j TO the remainder of value one PLUS value two DIVIDED BY MOD  
            END FOR  
        END FOR  
        RETURN element at position n of dp at position k  
    END FUNCTION  
END CLASS
