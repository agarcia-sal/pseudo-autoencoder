CLASS Solution  
    FUNCTION distinctSubseqII(s)  
        SET MOD TO ten raised to the power of nine PLUS seven  
        SET n TO the LENGTH OF s  
        CREATE dp AS a list of zero repeated n PLUS one times  
        SET element at position zero of dp TO one  
        CREATE last AS an empty dictionary  
        FOR i FROM one TO n  
            SET char TO element at position i MINUS one of s  
            SET element at position i of dp TO the element at position i MINUS one of dp MULTIPLIED BY two  
            SET element at position i of dp TO element at position i of dp MODULO MOD  
            IF char EXISTS in last  
                SET element at position i of dp TO element at position i of dp MINUS element at position equal to element at key char in last MINUS one of dp PLUS MOD  
                SET element at position i of dp TO element at position i of dp MODULO MOD  
            END IF  
            ASSIGN key char in last TO i  
        END FOR  
        RETURN element at position n of dp MINUS one MODULO MOD  
    END FUNCTION  
END CLASS
