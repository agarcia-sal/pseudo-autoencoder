CLASS Solution  
    FUNCTION numberOfArrays WITH PARAMETERS s AS string AND k AS integer RETURNS integer  
        SET MOD TO ten POWERED BY nine PLUS seven  
        SET n TO the LENGTH OF s  
        SET dp TO a list containing zero repeated n PLUS one times  
        SET the element at position zero of dp TO one  
        FOR i FROM one TO n  
            FOR j FROM zero TO i MINUS one  
                IF the character at position j of s NOT EQUALS zero AS string AND the integer value of the substring from position j TO i MINUS one of s GREATER THAN OR EQUAL TO one AND the integer value of the substring from position j TO i MINUS one of s LESS THAN OR EQUAL TO k  
                    SET the element at position i of dp TO the sum of the element at position i of dp AND the element at position j of dp MODULO MOD  
                END IF  
            END FOR  
        END FOR  
        RETURN the element at position n of dp  
    END FUNCTION  
END CLASS
