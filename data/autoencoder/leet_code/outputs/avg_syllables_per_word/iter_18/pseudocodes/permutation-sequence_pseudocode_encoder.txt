CLASS Solution  
    FUNCTION getPermutation WITH PARAMETERS n and k  
        IMPORT mathematical functions  
        SET numbers TO list of numbers FROM one TO n  
        DECREMENT k BY one  
        SET result TO empty list  
        FOR i FROM n DOWN TO one BY step minus one  
            SET fact TO factorial of i MINUS one  
            SET index TO integer division of k BY fact  
            APPEND string representation of element at position index of numbers TO result  
            REMOVE element at position index FROM numbers  
            SET k TO remainder of k DIVIDED BY fact  
        END FOR  
        RETURN concatenation of all elements in result as a single string  
    END FUNCTION  
END CLASS
