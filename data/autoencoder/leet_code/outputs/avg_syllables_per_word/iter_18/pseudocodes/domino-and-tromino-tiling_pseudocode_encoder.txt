CLASS Solution  
    FUNCTION numTilings WITH parameter n  
        SET MODULO TO ten raised to the power of nine PLUS one  
        
        SET dp TO list of zeroes of the LENGTH OF n PLUS one  
        SET dp1 TO list of zeroes of the LENGTH OF n PLUS one  
        
        SET element at position zero of dp TO one  
        SET element at position one of dp TO one  
        IF n GREATER THAN OR EQUAL TO two THEN  
            SET element at position two of dp TO two  
            SET element at position two of dp1 TO one  
        END IF  
        
        FOR index i FROM three TO n  
            SET element at position i of dp TO the REMAINDER AFTER DIVISION BY MODULO of element at position i MINUS one of dp PLUS element at position i MINUS two of dp PLUS two TIMES element at position i MINUS one of dp1  
            SET element at position i of dp1 TO the REMAINDER AFTER DIVISION BY MODULO of element at position i MINUS two of dp PLUS element at position i MINUS one of dp1  
        END FOR  
        
        RETURN element at position n of dp  
    END FUNCTION  
END CLASS
