CLASS Solution  
    FUNCTION numTrees WITH PARAMETER n  
        SET dp TO a list of zero repeated for the count of the value of n plus one  
        SET the element at position zero of dp TO one  
        SET the element at position one of dp TO one  
        FOR i FROM two TO the value of n inclusive  
            FOR j FROM one TO the value of i inclusive  
                SET the element at position i of dp TO the element at position i of dp PLUS the element at position j minus one of dp MULTIPLIED BY the element at position i minus j of dp  
            END FOR  
        END FOR  
        RETURN the element at position n of dp  
    END FUNCTION  
END CLASS
