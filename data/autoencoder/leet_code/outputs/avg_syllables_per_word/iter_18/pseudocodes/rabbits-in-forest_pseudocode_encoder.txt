CLASS Solution  
    FUNCTION numRabbits WITH parameter answers OF TYPE List OF integers  
        SET answer_counts TO Counter OF answers  
        SET total_rabbits TO zero  
        FOR each answer AND count IN answer_counts  
            SET groups TO the integer division RESULT of the SUM of count AND answer BY the SUM of answer AND one  
            INCREMENT total_rabbits BY the PRODUCT of groups AND the SUM of answer AND one  
        END FOR  
        RETURN total_rabbits  
    END FUNCTION  
END CLASS
