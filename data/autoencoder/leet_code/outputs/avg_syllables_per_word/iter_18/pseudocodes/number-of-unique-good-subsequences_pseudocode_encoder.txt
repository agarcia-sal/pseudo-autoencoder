CLASS Solution  
    FUNCTION numberOfUniqueGoodSubsequences WITH PARAMETER binary  
        SET MOD TO ten RAISED TO THE power OF nine PLUS one  
        SET end_with_0 TO zero  
        SET end_with_1 TO zero  
        SET has_zero TO whether the character zero IS PRESENT IN the string binary  
        FOR each bit IN binary  
            IF the bit IS EQUAL TO the character one THEN  
                SET end_with_1 TO the sum of end_with_0 PLUS end_with_1 PLUS one MODULO MOD  
            ELSE  
                SET end_with_0 TO the sum of end_with_0 PLUS end_with_1 MODULO MOD  
            END IF  
        END FOR  
        RETURN the sum of end_with_0 PLUS end_with_1 PLUS has_zero MODULO MOD  
    END FUNCTION  
END CLASS
