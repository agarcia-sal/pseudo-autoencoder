CLASS Solution  
    FUNCTION isNumber WITH PARAMETER s OF TYPE string  
        SET pattern TO a string representing a regular expression that optionally begins with a plus sign OR a minus sign followed BY either a sequence of one or more digits optionally followed BY a decimal point FOLLOWED BY zero or more digits OR a decimal point FOLLOWED BY one or more digits optionally followed BY an exponent part which consists OF the letter e optionally followed BY a plus sign OR a minus sign FOLLOWED BY one or more digits  
        SET match_result TO the result OF applying a regular expression full match OF pattern AGAINST s  
        IF match_result is not equal TO nothing  
            RETURN true  
        ELSE  
            RETURN false  
        END IF  
    END FUNCTION  
END CLASS
