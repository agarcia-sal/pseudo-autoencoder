CLASS Solution  
    FUNCTION minBuildTime WITH parameters blocks AND split  
        APPLY heapify OPERATION TO transform blocks INTO a minimum heap structure  
        WHILE the LENGTH OF blocks IS GREATER THAN one  
            SET first TO the result of removing the smallest element FROM blocks  
            SET second TO the result of removing the smallest element FROM blocks  
            SET combined_time TO split PLUS second  
            INSERT combined_time INTO blocks maintaining the heap property  
        END WHILE  
        RETURN the element at position zero OF blocks  
    END FUNCTION  
END CLASS
