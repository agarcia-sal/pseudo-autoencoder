CLASS Solution  
    FUNCTION coinChange WITH PARAMETERS coins AND amount  
        SET dp TO a list containing the value of amount PLUS one repeated for the number of times equal to amount PLUS one  
        SET the element at position zero of dp TO zero  
        FOR each integer a FROM one TO amount inclusive  
            FOR each element coin IN coins  
                IF the result of a MINUS coin IS GREATER THAN OR EQUAL TO zero  
                    SET the element at position a of dp TO the smaller value BETWEEN the element at position a of dp AND the sum of the element at position the result of a MINUS coin of dp PLUS one  
                END IF  
            END FOR  
        END FOR  
        IF the element at position amount of dp IS NOT EQUAL TO amount PLUS one  
            RETURN the element at position amount of dp  
        ELSE  
            RETURN negative one  
        END IF  
    END FUNCTION  
END CLASS
