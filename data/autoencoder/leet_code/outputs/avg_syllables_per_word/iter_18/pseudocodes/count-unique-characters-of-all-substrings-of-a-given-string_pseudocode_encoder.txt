CLASS Solution  
    FUNCTION uniqueLetterString WITH PARAMETER s  
        SET MOD TO ten raised to the power of nine PLUS one  
        SET n TO the LENGTH OF s  
        INITIALIZE last TO a list of negative one repeated twenty six times  
        INITIALIZE last2 TO a list of negative one repeated twenty six times  
        SET result TO zero  
        FOR each index i and character char IN s with enumeration  
            SET idx TO the position of char in the alphabet calculated as the numerical value of char MINUS the numerical value of uppercase letter A  
            SET contribution TO the product of the difference between i and element at position idx of last AND the difference between element at position idx of last AND element at position idx of last2  
            SET result TO the sum of result AND contribution MODULO MOD  
            SET element at position idx of last2 TO element at position idx of last  
            SET element at position idx of last TO i  
        END FOR  
        FOR each index i FROM zero TO twenty five  
            SET contribution TO the product of the difference between n AND element at position i of last AND the difference between element at position i of last AND element at position i of last2  
            SET result TO the sum of result AND contribution MODULO MOD  
        END FOR  
        RETURN result  
    END FUNCTION  
END CLASS
