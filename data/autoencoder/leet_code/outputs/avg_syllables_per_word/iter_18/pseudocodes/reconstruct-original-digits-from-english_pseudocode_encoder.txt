CLASS Solution  
    FUNCTION originalDigits WITH PARAMETER s AS string  
        FROM the collection library IMPORT Counter  
        SET count TO the result of invoking Counter ON s  
        SET digits TO a list containing ten items all set TO zero  
        SET element at position zero of digits TO the element at key z in count  
        SET element at position two of digits TO the element at key w in count  
        SET element at position four of digits TO the element at key u in count  
        SET element at position six of digits TO the element at key x in count  
        SET element at position eight of digits TO the element at key g in count  
        SET element at position three of digits TO the element at key h in count MINUS the element at position eight of digits  
        SET element at position five of digits TO the element at key f in count MINUS the element at position four of digits  
        SET element at position seven of digits TO the element at key s in count MINUS the element at position six of digits  
        SET element at position nine of digits TO the element at key i in count MINUS the element at position five of digits MINUS the element at position six of digits MINUS the element at position eight of digits  
        SET element at position one of digits TO the element at key o in count MINUS the element at position zero of digits MINUS the element at position two of digits MINUS the element at position four of digits  
        SET result TO an empty list  
        FOR each index i FROM zero TO nine  
            APPEND a string consisting of the string representation of i repeated element at position i of digits times TO result  
        END FOR  
        RETURN the concatenation of all elements in result  
    END FUNCTION  
END CLASS
