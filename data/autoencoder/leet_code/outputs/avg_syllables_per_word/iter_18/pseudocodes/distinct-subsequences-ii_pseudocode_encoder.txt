CLASS Solution  
    FUNCTION distinctSubseqII WITH parameter s  
        SET MOD TO ten raised to the power of nine PLUS one  
        SET n TO the LENGTH OF s  
        SET dp TO a list of zeroes with the length of n PLUS one  
        SET element at position zero of dp TO one  
        SET last TO an empty mapping  
        
        FOR index i FROM one TO n inclusive  
            SET char TO character at position i MINUS one of s  
            SET element at position i of dp TO the element at position i MINUS one of dp MULTIPLIED BY two MODULO MOD  
            
            IF char EXISTS IN last THEN  
                SET element at position i of dp TO the element at position i of dp MINUS the element at position the value of char in last MINUS one of dp PLUS MOD MODULO MOD  
            END IF  
            
            SET the value of char in last TO i  
        END FOR  
        
        RETURN the element at position n of dp MINUS one MODULO MOD  
    END FUNCTION  
END CLASS
