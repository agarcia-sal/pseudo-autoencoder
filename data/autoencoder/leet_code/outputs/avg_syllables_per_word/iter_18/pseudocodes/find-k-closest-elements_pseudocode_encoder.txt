CLASS Solution  
    FUNCTION findClosestElements WITH PARAMETERS arr AS List OF integer AND k AS integer AND x AS integer RETURNS List OF integer  
        SET left TO the result of bisect_left function applied to arr and x MINUS one  
        SET right TO left PLUS one  
        WHILE the amount of elements between right and left MINUS one IS LESS THAN k  
            IF left EQUALS minus one OR the value of right IS LESS THAN the LENGTH OF arr AND the absolute value of the element at position right of arr MINUS x IS LESS THAN the absolute value of the element at position left of arr MINUS x  
                INCREMENT right BY one  
            ELSE  
                DECREMENT left BY one  
            END IF  
        END WHILE  
        RETURN the sublist of arr from position left PLUS one UP TO right  
    END FUNCTION  
END CLASS
