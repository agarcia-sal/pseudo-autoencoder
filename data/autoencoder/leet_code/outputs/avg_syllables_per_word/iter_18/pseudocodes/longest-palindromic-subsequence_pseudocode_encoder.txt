CLASS Solution  
    FUNCTION longestPalindromeSubseq WITH PARAMETER s  
        SET n TO the LENGTH OF s  
        SET dp TO a two-dimensional array OF size n BY n FILLED WITH zero  
        FOR each index i FROM zero TO n MINUS one  
            SET element at position i AND position i OF dp TO one  
        END FOR  
        FOR each length FROM two TO n  
            FOR each index i FROM zero TO n MINUS length  
                SET j TO i PLUS length MINUS one  
                IF character at position i OF s EQUALS character at position j OF s THEN  
                    SET element at position i PLUS one AND j MINUS one OF dp TO value AT element at position i PLUS one AND j MINUS one OF dp  
                    SET element at position i AND j OF dp TO value AT element at position i PLUS one AND j MINUS one OF dp PLUS two  
                ELSE  
                    IF value AT element at position i PLUS one AND j OF dp GREATER THAN value AT element at position i AND j MINUS one OF dp THEN  
                        SET element at position i AND j OF dp TO value AT element at position i PLUS one AND j OF dp  
                    ELSE  
                        SET element at position i AND j OF dp TO value AT element at position i AND j MINUS one OF dp  
                    END IF  
                END IF  
            END FOR  
        END FOR  
        RETURN value AT element at position zero AND n MINUS one OF dp  
    END FUNCTION  
END CLASS
