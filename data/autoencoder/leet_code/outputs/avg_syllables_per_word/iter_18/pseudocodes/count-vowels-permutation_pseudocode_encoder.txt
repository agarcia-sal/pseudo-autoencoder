CLASS Solution  
    FUNCTION countVowelPermutation WITH PARAMETER n  
        SET MOD TO ten raised to the power of nine PLUS one  
        SET variable a TO one  
        SET variable e TO one  
        SET variable i TO one  
        SET variable o TO one  
        SET variable u TO one  
        FOR each number starting from one UNTIL less than n  
            SET a_next TO the sum of variable e PLUS variable i PLUS variable u  
            SET e_next TO the sum of variable a PLUS variable i  
            SET i_next TO the sum of variable e PLUS variable o  
            SET o_next TO variable i  
            SET u_next TO the sum of variable i PLUS variable o  
            SET variable a TO the remainder after dividing a_next BY MOD  
            SET variable e TO the remainder after dividing e_next BY MOD  
            SET variable i TO the remainder after dividing i_next BY MOD  
            SET variable o TO the remainder after dividing o_next BY MOD  
            SET variable u TO the remainder after dividing u_next BY MOD  
        END FOR  
        SET total_sum TO the sum of variable a PLUS variable e PLUS variable i PLUS variable o PLUS variable u  
        SET result TO the remainder after dividing total_sum BY MOD  
        RETURN result  
    END FUNCTION  
END CLASS
