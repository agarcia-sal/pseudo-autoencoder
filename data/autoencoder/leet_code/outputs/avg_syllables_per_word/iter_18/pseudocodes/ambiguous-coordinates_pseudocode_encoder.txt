CLASS Solution  
    FUNCTION ambiguousCoordinates WITH parameter s  
        FUNCTION make WITH parameter frag  
            SET n TO the LENGTH OF frag  
            FOR d FROM one TO n  
                SET left TO the SUBSTRING of frag FROM the FIRST character TO the character at position d  
                SET right TO the SUBSTRING of frag FROM the character at position d PLUS one TO the LAST character  
                IF left DOES NOT START WITH character zero OR left EQUALS character zero AND right DOES NOT END WITH character zero  
                    YIELD left CONCATENATED WITH IF d IS NOT EQUAL TO n THEN character point ELSE empty string CONCATENATED WITH right  
                END IF  
            END FOR  
        END FUNCTION  
        SET s TO the SUBSTRING of s FROM the SECOND character TO the CHARACTER BEFORE the LAST character  
        RETURN a LIST constructed by FOR i FROM one TO the LENGTH OF s MINUS one FOR each a and b IN the CARTESIAN PRODUCT OF make APPLIED TO the SUBSTRING of s FROM the FIRST character TO the CHARACTER BEFORE position i and make APPLIED TO the SUBSTRING of s FROM the character at position i PLUS one TO the LAST character THE STRING OF an OPENING parenthesis CONCATENATED WITH a CONCATENATED WITH a COMMA AND a SPACE CONCATENATED WITH b CONCATENATED WITH a CLOSING parenthesis  
    END FUNCTION  
END CLASS
