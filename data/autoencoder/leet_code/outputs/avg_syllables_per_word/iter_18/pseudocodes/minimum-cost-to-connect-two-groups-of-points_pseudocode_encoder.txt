CLASS Solution  
    FUNCTION connectTwoGroups WITH PARAMETER cost AS list of list of integer RETURNS integer  
        SET size1 TO the LENGTH OF cost  
        SET size2 TO the LENGTH OF element at position zero of cost  
        SET dp TO a two dimensional list of dimensions size1 plus one BY two RAISED TO the power of size2 EACH element initialized TO positive infinity  
        SET element at position zero zero of dp TO zero  
        FOR index i FROM zero TO size1 MINUS one  
            FOR index mask FROM zero TO two RAISED TO the power of size2 MINUS one  
                FOR index j FROM zero TO size2 MINUS one  
                    SET new_mask TO mask OR the value two RAISED TO the power of j  
                    SET element at position i plus one new_mask of dp TO the MINIMUM OF element at position i plus one new_mask of dp AND element at position i mask of dp PLUS element at position i j of cost  
                END FOR  
                FOR index j FROM zero TO size2 MINUS one  
                    IF bitwise AND of mask AND two RAISED TO the power of j holds  
                        SET element at position i plus one mask of dp TO the MINIMUM OF element at position i plus one mask of dp AND element at position i plus one bitwise XOR of mask AND two RAISED TO the power of j of dp PLUS element at position i j of cost  
                    END IF  
                END FOR  
            END FOR  
        END FOR  
        RETURN element at position size1 two RAISED TO the power of size2 MINUS one of dp  
    END FUNCTION  
END CLASS
