CLASS Solution  
    FUNCTION checkRecord(n)  
        SET MOD TO ten raised to the power of nine PLUS one hundred and seven  
        INITIALIZE dp AS a three dimensional collection of zeroes with dimensions of n incremented by one for the first dimension two for the second dimension and three for the third dimension  
        SET element at position zero zero zero of dp TO one  
        FOR i FROM one TO n  
            FOR j FROM zero TO one  
                FOR k FROM zero TO two  
                    SET element at position i j zero of dp TO element at position i j zero of dp PLUS element at position i decremented by one j k of dp MODULO MOD  
                    IF j EQUALS zero THEN  
                        SET element at position i one zero of dp TO element at position i one zero of dp PLUS element at position i decremented by one zero k of dp MODULO MOD  
                    END IF  
                    IF k LESS THAN two THEN  
                        SET element at position i j k incremented by one of dp TO element at position i j k incremented by one of dp PLUS element at position i decremented by one j k of dp MODULO MOD  
                    END IF  
                END FOR  
            END FOR  
        END FOR  
        SET result TO zero  
        FOR j FROM zero TO one  
            FOR k FROM zero TO two  
                SET result TO result PLUS element at position n j k of dp MODULO MOD  
            END FOR  
        END FOR  
        RETURN result  
    END FUNCTION  
END CLASS
