CLASS Solution  
    FUNCTION getModifiedArray(length PARAMETER, updates PARAMETER)  
        SET diff TO a list of zero repeated the LENGTH OF length PLUS one times  
        FOR each startIdx and endIdx and inc IN updates  
            INCREMENT element at position startIdx of diff BY inc  
            IF the value one MORE THAN endIdx is LESS THAN length THEN  
                DECREMENT element at position one MORE THAN endIdx of diff BY inc  
            END IF  
        END FOR  
        SET arr TO a list of zero repeated the LENGTH OF length times  
        SET element at position zero of arr TO element at position zero of diff  
        FOR i FROM one TO the LENGTH OF length MINUS one  
            SET element at position i of arr TO element at position i MINUS one of arr PLUS element at position i of diff  
        END FOR  
        RETURN arr  
    END FUNCTION  
END CLASS
