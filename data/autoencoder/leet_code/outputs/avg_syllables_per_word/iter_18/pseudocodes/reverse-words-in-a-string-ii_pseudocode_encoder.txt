CLASS Solution  
    FUNCTION reverseWords WITH PARAMETER s AS list of strings  
        CALL the reverse METHOD ON s to REVERSE the entire list s  
        SET start TO zero  
        FOR end FROM zero TO the LENGTH OF s MINUS one  
            IF the element at position end of s EQUALS a space character  
                CALL the reverse OPERATION ON the SUBLIST of s FROM position start TO position end MINUS one to REVERSE the word in place  
                SET start TO end PLUS one  
            END IF  
        END FOR  
        CALL the reverse OPERATION ON the SUBLIST of s FROM position start TO the END OF s to REVERSE the last word in place  
    END FUNCTION  
END CLASS
