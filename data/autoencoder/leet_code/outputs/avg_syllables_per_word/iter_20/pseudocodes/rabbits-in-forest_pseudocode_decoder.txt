CLASS Solution  
    FUNCTION numRabbits WITH parameter answers  
        SET answer_counts TO CALL Counter WITH answers AS parameter  
        SET total_rabbits TO zero  
        FOR each pair of answer AND count IN the ITEMS OF answer_counts  
            SET groups TO the result of INTEGER DIVISION of the SUM of count AND answer BY the SUM of answer AND one  
            INCREMENT total_rabbits BY the MULTIPLICATION of groups AND the SUM of answer AND one  
        END FOR  
        RETURN total_rabbits  
    END FUNCTION  
END CLASS
