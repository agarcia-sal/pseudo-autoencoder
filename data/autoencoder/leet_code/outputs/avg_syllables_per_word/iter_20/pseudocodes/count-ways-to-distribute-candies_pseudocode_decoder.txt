CLASS Solution  
    FUNCTION waysToDistribute(n, k)  
        SET MOD TO ten raised to the power of nine PLUS one  
        INITIALIZE dp AS a two dimensional list with n PLUS one rows each containing k PLUS one zeros  
        SET element at position zero of dp AND position zero to one  
        FOR i FROM one TO n  
            FOR j FROM one TO k  
                SET product TO j MULTIPLIED BY element at position i MINUS one of dp AND position j  
                SET sum TO product PLUS element at position i MINUS one of dp AND position j MINUS one  
                SET element at position i of dp AND position j TO sum MODULO MOD  
            END FOR  
        END FOR  
        RETURN element at position n of dp AND position k  
    END FUNCTION  
END CLASS
