CLASS Solution  
    FUNCTION multiply WITH PARAMETERS mat1 AND mat2  
        SET variable m TO the LENGTH OF mat1  
        SET variable k TO the LENGTH OF element at position zero of mat1  
        SET variable k TO the LENGTH OF mat2  
        SET variable n TO the LENGTH OF element at position zero of mat2  
        SET variable result TO a new list containing m elements where each element is a list of n elements each SET TO zero  
        FOR index i FROM zero TO m MINUS one  
            FOR index j FROM zero TO n MINUS one  
                FOR index p FROM zero TO k MINUS one  
                    SET element at position j of element at position i of result TO element at position j of element at position i of result PLUS element at position p of element at position i of mat1 MULTIPLIED BY element at position j of element at position p of mat2  
                END FOR  
            END FOR  
        END FOR  
        RETURN result  
    END FUNCTION  
END CLASS
