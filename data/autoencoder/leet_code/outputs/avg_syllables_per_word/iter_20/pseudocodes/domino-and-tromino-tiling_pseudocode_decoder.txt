CLASS Solution  
    FUNCTION numTilings(n)  
        SET MOD TO one billion PLUS seven  
        INITIALIZE dp AS a list of size n PLUS one filled with zero  
        INITIALIZE dp1 AS a list of size n PLUS one filled with zero  
        SET element at position zero of dp TO one  
        SET element at position one of dp TO one  
        IF n IS GREATER THAN OR EQUAL TO two  
            SET element at position two of dp TO two  
            SET element at position two of dp1 TO one  
        END IF  
        FOR integer i FROM three TO n  
            SET element at position i of dp TO the remainder when the sum of element at position i MINUS one of dp PLUS element at position i MINUS two of dp PLUS twice the element at position i MINUS one of dp1 is divided by MOD  
            SET element at position i of dp1 TO the remainder when the sum of element at position i MINUS two of dp PLUS element at position i MINUS one of dp1 is divided by MOD  
        END FOR  
        RETURN element at position n of dp  
    END FUNCTION  
END CLASS
