    FUNCTION isRationalEqual(s)  
        SET ratios TO a list containing the number one and the fractions one DIVIDED BY nine one DIVIDED BY ninety nine and one DIVIDED BY nine thousand nine hundred ninety nine and one DIVIDED BY ninety nine thousand nine hundred ninety nine

        FUNCTION valueOf(s)  
            IF the position of the character left parenthesis in s EQUALS negative one  
                RETURN the conversion of s TO float  
            END IF

            SET leftParenIndex TO the position of the character left parenthesis in s  
            SET rightParenIndex TO the position of the character right parenthesis in s  
            SET dotIndex TO the position of the character dot in s  

            SET integerAndNonRepeating TO the conversion TO float of the substring of s from the start TO the position before leftParenIndex  
            SET nonRepeatingLength TO leftParenIndex MINUS dotIndex MINUS one  

            SET repeating TO the integer conversion of the substring of s from one position after leftParenIndex TO the position before rightParenIndex  
            SET repeatingLength TO rightParenIndex MINUS leftParenIndex MINUS one  

            RETURN integerAndNonRepeating PLUS repeating MULTIPLIED BY the number zero point one raised TO THE POWER OF nonRepeatingLength MULTIPLIED BY the element at position repeatingLength of ratios  
        END FUNCTION

        RETURN the absolute value of the subtraction of valueOf applied to s MINUS valueOf applied to t LESS THAN one multiplied BY ten RAISED TO THE POWER OF negative nine  
    END FUNCTION  
END CLASS
