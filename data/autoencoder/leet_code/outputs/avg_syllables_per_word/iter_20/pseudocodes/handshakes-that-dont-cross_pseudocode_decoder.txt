CLASS Solution  
    FUNCTION numberOfWays WITH PARAMETER numPeople  
        SET MOD TO ten RAISED TO THE POWER OF nine PLUS seven  
        INITIALIZE dp AS A LIST OF ZEROES WITH THE LENGTH EQUAL TO numPeople PLUS one  
        SET element at position zero of dp TO one  
        SET element at position two of dp TO one  
        FOR each i FROM four TO numPeople IN STEPS OF two  
            FOR each j FROM zero TO i MINUS one IN STEPS OF two  
                SET element at position i of dp TO element at position i of dp PLUS element at position j of dp MULTIPLIED BY element at position i MINUS two MINUS j of dp  
                APPLY MODULO MOD TO element at position i of dp  
            END FOR  
        END FOR  
        RETURN element at position numPeople of dp  
    END FUNCTION  
END CLASS
