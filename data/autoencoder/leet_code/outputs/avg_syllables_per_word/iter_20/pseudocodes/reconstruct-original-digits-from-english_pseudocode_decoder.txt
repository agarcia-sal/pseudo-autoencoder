CLASS Solution  
    FUNCTION originalDigits(s)  
        SET count TO the frequency count of each character in s using a character counting method  
        SET digits TO an array of ten zeros representing counts for digits zero through nine  
        SET the count for digit zero TO the number of times character z appears in count  
        SET the count for digit two TO the number of times character w appears in count  
        SET the count for digit four TO the number of times character u appears in count  
        SET the count for digit six TO the number of times character x appears in count  
        SET the count for digit eight TO the number of times character g appears in count  
        SET the count for digit three TO the number of times character h appears in count MINUS the count for digit eight  
        SET the count for digit five TO the number of times character f appears in count MINUS the count for digit four  
        SET the count for digit seven TO the number of times character s appears in count MINUS the count for digit six  
        SET the count for digit nine TO the number of times character i appears in count MINUS the counts for digits five, six, and eight  
        SET the count for digit one TO the number of times character o appears in count MINUS the counts for digits zero, two, and four  
        SET result TO an empty list  
        FOR each index i FROM zero TO nine  
            APPEND the string representation of i repeated the count at position i in digits TO result  
        END FOR  
        RETURN the concatenation of all elements in result  
    END FUNCTION  
END CLASS
