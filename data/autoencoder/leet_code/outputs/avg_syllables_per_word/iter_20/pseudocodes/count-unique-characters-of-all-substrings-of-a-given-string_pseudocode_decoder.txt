CLASS Solution  
    FUNCTION uniqueLetterString WITH PARAMETER s  
        SET MOD TO one billion PLUS seven  
        SET n TO the LENGTH OF s  
        SET last TO a list composed of negative one repeated twenty six times  
        SET last2 TO a list composed of negative one repeated twenty six times  
        SET result TO zero  
        FOR each position i AND character char IN enumeration OF s  
            SET idx TO the numerical Unicode code of char MINUS the numerical Unicode code of uppercase A  
            SET result TO the REMAINDER AFTER DIVISION BY MOD OF result PLUS the PRODUCT OF the difference BETWEEN i AND element at position idx of last AND the difference BETWEEN element at position idx of last AND element at position idx of last2  
            SET element at position idx of last2 TO element at position idx of last  
            SET element at position idx of last TO i  
        END FOR  
        FOR each integer i FROM zero TO twenty five  
            SET result TO the REMAINDER AFTER DIVISION BY MOD OF result PLUS the PRODUCT OF the difference BETWEEN n AND element at position i of last AND the difference BETWEEN element at position i of last AND element at position i of last2  
        END FOR  
        RETURN result  
    END FUNCTION  
END CLASS
