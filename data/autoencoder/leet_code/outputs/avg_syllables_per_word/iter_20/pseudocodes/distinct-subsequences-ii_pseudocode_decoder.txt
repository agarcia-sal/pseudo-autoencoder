CLASS Solution  
    FUNCTION distinctSubseqII WITH PARAMETER s  
        SET MOD TO one billion PLUS seven  
        SET n TO the LENGTH OF s  
        SET dp TO a list containing zero repeated n PLUS one times  
        SET the element at position zero of dp TO one  
        SET last TO an empty mapping  
        FOR each integer i FROM one TO n  
            SET char TO the element at position i MINUS one of s  
            SET the element at position i of dp TO the element at position i MINUS one of dp MULTIPLIED BY two  
            SET the element at position i of dp TO the remainder AFTER DIVIDING the element at position i of dp BY MOD  
            IF char IS A KEY IN last  
                SET the element at position i of dp TO the element at position i of dp MINUS the element at position last OF char MINUS one of dp PLUS MOD  
                SET the element at position i of dp TO the remainder AFTER DIVIDING the element at position i of dp BY MOD  
            END IF  
            SET the value of last AT key char TO i  
        END FOR  
        SET result TO the element at position n of dp MINUS one  
        SET result TO the remainder AFTER DIVIDING result BY MOD  
        RETURN result  
    END FUNCTION  
END CLASS
