CLASS Solution  
    FUNCTION isInterleave WITH PARAMETERS s1 s2 s3  
        IF the LENGTH OF s1 PLUS the LENGTH OF s2 IS NOT EQUAL TO the LENGTH OF s3  
            RETURN false  
        END IF  
        SET dp TO a list of boolean false values with LENGTH equal to the LENGTH OF s2 PLUS one  
        SET element at position zero of dp TO true  
        FOR each index i FROM zero TO the LENGTH OF s1 INCLUSIVE  
            FOR each index j FROM zero TO the LENGTH OF s2 INCLUSIVE  
                IF i IS GREATER THAN zero  
                    SET element at position j of dp TO the logical AND of element at position j of dp AND the CONDITION THAT the element at position i MINUS one of s1 IS EQUAL TO the element at position i PLUS j MINUS one of s3  
                END IF  
                IF j IS GREATER THAN zero  
                    SET element at position j of dp TO the logical OR of element at position j of dp AND the logical AND of element at position j MINUS one of dp AND the CONDITION THAT the element at position j MINUS one of s2 IS EQUAL TO the element at position i PLUS j MINUS one of s3  
                END IF  
            END FOR  
        END FOR  
        RETURN element at position the LENGTH OF s2 of dp  
    END FUNCTION  
END CLASS
