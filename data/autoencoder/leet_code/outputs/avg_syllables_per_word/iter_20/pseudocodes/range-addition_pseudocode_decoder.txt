CLASS Solution  
    FUNCTION getModifiedArray(length, updates)  
        INITIALIZE diff TO a list of zeroes with the count of length PLUS one  
        FOR each update consisting of start index end index and increment IN updates  
            INCREMENT element at position start index of diff BY increment  
            IF element at position end index PLUS one IS LESS THAN length  
                DECREMENT element at position end index PLUS one of diff BY increment  
            END IF  
        END FOR  
        INITIALIZE arr TO a list of zeroes with the count of length  
        SET element at position zero of arr TO element at position zero of diff  
        FOR index FROM one TO length MINUS one  
            SET element at position index of arr TO element at position index MINUS one of arr PLUS element at position index of diff  
        END FOR  
        RETURN arr  
    END FUNCTION  
END CLASS
