CLASS Solution  
    FUNCTION findDuplicate(nums)  
        FUNCTION f(x)  
            RETURN the SUM of the count of elements v in nums WHERE v IS LESS THAN OR EQUAL TO x IS GREATER THAN x  
        END FUNCTION  
        RETURN the POSITION OF THE FIRST TRUE value IN the range FROM zero TO the LENGTH OF nums USING f AS THE COMPARISON FUNCTION  
    END FUNCTION  
END CLASS
