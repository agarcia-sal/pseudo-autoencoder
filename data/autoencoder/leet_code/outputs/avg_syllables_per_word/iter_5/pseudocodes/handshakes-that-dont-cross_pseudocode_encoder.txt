CLASS Solution  
    FUNCTION numberOfWays(numPeople)  
        SET MOD TO ten raised to the power of nine plus seven  
        SET dp TO a list of zeros with length numPeople plus one  
        SET dp at index zero TO one  
        SET dp at index two TO one  
        FOR i FROM four TO numPeople IN STEPS OF two  
            FOR j FROM zero TO i minus one IN STEPS OF two  
                SET dp at index i TO dp at index i plus dp at index j multiplied BY dp at index i minus two minus j  
                SET dp at index i TO dp at index i MODULO MOD  
        RETURN dp at index numPeople  
    END FUNCTION  
END CLASS
