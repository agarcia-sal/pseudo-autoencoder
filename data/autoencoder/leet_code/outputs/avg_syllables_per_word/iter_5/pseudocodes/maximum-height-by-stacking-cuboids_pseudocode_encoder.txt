CLASS Solution  
    FUNCTION maxHeight(cuboids)  
        FOR each cuboid IN cuboids  
            SORT dimensions of cuboid  
        SORT cuboids lexicographically  
        SET n TO LENGTH OF cuboids  
        INITIALIZE dp AS list of zeros with length n  
        FOR i FROM 0 TO n MINUS 1  
            SET dp at i TO height dimension of cuboids at i  
            FOR j FROM 0 TO i MINUS 1  
                IF length dimension of cuboids at j LESS THAN OR EQUAL TO length dimension of cuboids at i  
                AND width dimension of cuboids at j LESS THAN OR EQUAL TO width dimension of cuboids at i  
                AND height dimension of cuboids at j LESS THAN OR EQUAL TO height dimension of cuboids at i  
                    SET dp at i TO MAXIMUM OF dp at i AND dp at j PLUS height dimension of cuboids at i  
        RETURN MAXIMUM VALUE IN dp  
    END FUNCTION  
END CLASS
