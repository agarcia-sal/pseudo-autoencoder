CLASS Solution  
    FUNCTION reverse(x)  
        SET sign TO negative one IF x LESS THAN zero ELSE positive one  
        SET string_representation TO string form of absolute value of x  
        SET reversed_string TO string_representation in reverse order of characters  
        SET reversed_x TO integer conversion of reversed_string  
        SET result TO sign MULTIPLIED BY reversed_x  
        IF result LESS THAN negative one MULTIPLIED BY two RAISED TO the power of thirty one OR result GREATER THAN two RAISED TO the power of thirty one MINUS one  
            RETURN zero  
        END IF  
        RETURN result  
    END FUNCTION  
END CLASS
