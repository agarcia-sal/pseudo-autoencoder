CLASS Solution  
    FUNCTION numberOfWays(numPeople)  
        SET MOD TO ten raised to the power one billion PLUS seven  
        SET dp TO a list of zeros with the LENGTH OF numPeople PLUS one  
        SET the element at position zero of dp TO one  
        SET the element at position two of dp TO one  
        FOR i FROM four TO numPeople IN STEPS OF two  
            FOR j FROM zero TO i MINUS one IN STEPS OF two  
                SET the element at position i of dp TO the element at position i of dp PLUS the element at position j of dp MULTIPLIED BY the element at position i MINUS two MINUS j of dp  
                SET the element at position i of dp TO the element at position i of dp MODULO MOD  
            END FOR  
        END FOR  
        RETURN the element at position numPeople of dp  
    END FUNCTION  
END CLASS
