CLASS Solution  
    FUNCTION distinctSubseqII(s)  
        SET MOD TO one billion PLUS seven  
        SET n TO the LENGTH OF s  
        SET dp TO a list of zeros of size n PLUS one  
        SET the element at position zero of dp TO one  
        SET last TO an empty dictionary  
        
        FOR i FROM one TO n  
            SET char TO the element at position i MINUS one of s  
            SET the element at position i of dp TO two MULTIPLIED BY the element at position i MINUS one of dp  
            SET the element at position i of dp TO the remainder of the element at position i of dp DIVIDED BY MOD  
            
            IF char IS IN last  
                SET the element at position i of dp TO the element at position i of dp MINUS the element at position the value of last at char MINUS one of dp PLUS MOD  
                SET the element at position i of dp TO the remainder of the element at position i of dp DIVIDED BY MOD  
            END IF  
            
            SET the value of last at char TO i  
        END FOR  
        
        RETURN the remainder of the element at position n of dp MINUS one DIVIDED BY MOD  
    END FUNCTION  
END CLASS
