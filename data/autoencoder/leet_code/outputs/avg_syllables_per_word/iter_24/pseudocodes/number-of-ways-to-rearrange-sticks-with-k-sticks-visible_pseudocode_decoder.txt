CLASS Solution  
    FUNCTION rearrangeSticks(n k)  
        SET MOD TO one billion PLUS seven  
        INITIALIZE dp AS a two dimensional list of zeros with n PLUS one rows and k PLUS one columns  
        SET element at position zero of dp at position zero TO one  
        FOR i FROM one TO n  
            FOR j FROM one TO k  
                SET element at position i of dp at position j TO element at position i MINUS one of dp at position j MINUS one PLUS i MINUS one MULTIPLIED BY element at position i MINUS one of dp at position j  
                SET element at position i of dp at position j TO element at position i of dp at position j MODULO MOD  
            END FOR  
        END FOR  
        RETURN element at position n of dp at position k  
    END FUNCTION  
END CLASS
