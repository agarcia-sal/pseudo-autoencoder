CLASS Solution  
    FUNCTION originalDigits(s)  
        SET count TO a mapping of character frequencies in s  
        SET digits TO a list of ten zeros  

        SET element at position zero of digits TO the frequency of character z in count  
        SET element at position two of digits TO the frequency of character w in count  
        SET element at position four of digits TO the frequency of character u in count  
        SET element at position six of digits TO the frequency of character x in count  
        SET element at position eight of digits TO the frequency of character g in count  

        SET element at position three of digits TO the frequency of character h in count MINUS element at position eight of digits  
        SET element at position five of digits TO the frequency of character f in count MINUS element at position four of digits  
        SET element at position seven of digits TO the frequency of character s in count MINUS element at position six of digits  
        SET element at position nine of digits TO the frequency of character i in count MINUS element at position five of digits MINUS element at position six of digits MINUS element at position eight of digits  
        SET element at position one of digits TO the frequency of character o in count MINUS element at position zero of digits MINUS element at position two of digits MINUS element at position four of digits  

        SET result TO an empty list  
        FOR i FROM zero TO nine  
            FOR each repetition FROM zero TO element at position i of digits MINUS one  
                APPEND string representation of i TO result  
            END FOR  
        END FOR  

        RETURN concatenation of all elements in result  
    END FUNCTION  
END CLASS
