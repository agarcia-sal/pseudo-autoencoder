CLASS Solution  
    FUNCTION minSwap(nums1 nums2)  
        SET n TO the LENGTH OF nums1  
        INITIALIZE dp AS a list of n elements each containing a list of two zeros  
        
        SET element at position zero of element at position zero of dp TO zero  
        SET element at position one of element at position zero of dp TO one  
        
        FOR i FROM one TO n MINUS one  
            IF element at position i of nums1 GREATER THAN element at position i MINUS one of nums1 AND element at position i of nums2 GREATER THAN element at position i MINUS one of nums2  
                IF element at position i of nums1 GREATER THAN element at position i MINUS one of nums2 AND element at position i of nums2 GREATER THAN element at position i MINUS one of nums1  
                    SET element at position zero of element at position i of dp TO the MINIMUM OF element at position zero of element at position i MINUS one of dp AND element at position one of element at position i MINUS one of dp  
                    SET element at position one of element at position i of dp TO element at position zero of element at position i of dp PLUS one  
                ELSE  
                    SET element at position zero of element at position i of dp TO element at position zero of element at position i MINUS one of dp  
                    SET element at position one of element at position i of dp TO element at position one of element at position i MINUS one of dp PLUS one  
                END IF  
            ELSE  
                IF element at position i of nums1 GREATER THAN element at position i MINUS one of nums2 AND element at position i of nums2 GREATER THAN element at position i MINUS one of nums1  
                    SET element at position zero of element at position i of dp TO element at position one of element at position i MINUS one of dp  
                    SET element at position one of element at position i of dp TO element at position zero of element at position i MINUS one of dp PLUS one  
                ELSE  
                    SET element at position zero of element at position i of dp TO element at position one of element at position i MINUS one of dp  
                    SET element at position one of element at position i of dp TO element at position zero of element at position i MINUS one of dp PLUS one  
                END IF  
            END IF  
        END FOR  
        
        RETURN the MINIMUM OF element at position zero of element at position n MINUS one of dp AND element at position one of element at position n MINUS one of dp  
    END FUNCTION  
END CLASS
