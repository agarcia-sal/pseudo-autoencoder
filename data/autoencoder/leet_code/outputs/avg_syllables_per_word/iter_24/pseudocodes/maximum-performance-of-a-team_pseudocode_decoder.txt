CLASS Solution  
    FUNCTION maxPerformance WITH PARAMETERS n speed efficiency k  
        SET engineers TO a list of pairs consisting of each element of efficiency paired with the corresponding element of speed  
        SORT engineers in descending order by the first element of each pair representing efficiency  
        SET speed_heap TO an empty list  
        SET speed_sum TO zero  
        SET max_performance TO zero  
        FOR each pair consisting of eff and spd IN engineers  
            ADD spd TO speed_heap maintaining a heap structure  
            INCREMENT speed_sum BY spd  
            IF the LENGTH OF speed_heap GREATER THAN k  
                REMOVE the smallest element FROM speed_heap maintaining a heap structure  
                DECREMENT speed_sum BY the removed element  
            END IF  
            SET current_performance TO speed_sum MULTIPLIED BY eff  
            IF current_performance GREATER THAN max_performance  
                SET max_performance TO current_performance  
            END IF  
        END FOR  
        RETURN the remainder of dividing max_performance BY the sum of ten raised to the power of nine PLUS one  
    END FUNCTION  
END CLASS
