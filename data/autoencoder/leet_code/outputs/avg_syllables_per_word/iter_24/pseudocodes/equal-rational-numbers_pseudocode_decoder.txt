CLASS Solution  
    FUNCTION isRationalEqual(s, t)  
        SET ratios TO list of values one and one DIVIDED BY nine and one DIVIDED BY ninety nine and one DIVIDED BY nine hundred ninety nine and one DIVIDED BY nine thousand nine hundred ninety nine  

        FUNCTION valueOf(s)  
            IF the position of character left parenthesis IN s EQUALS negative one THEN  
                RETURN the converted float value of s  
            END IF  

            SET leftParenIndex TO the position of character left parenthesis IN s  
            SET rightParenIndex TO the position of character right parenthesis IN s  
            SET dotIndex TO the position of character dot IN s  

            SET integerAndNonRepeating TO the float converted value of substring from position zero TO position leftParenIndex MINUS one OF s  
            SET nonRepeatingLength TO leftParenIndex MINUS dotIndex MINUS one  

            SET repeating TO the integer converted value of substring from position leftParenIndex PLUS one TO position rightParenIndex MINUS one OF s  
            SET repeatingLength TO rightParenIndex MINUS leftParenIndex MINUS one  

            RETURN integerAndNonRepeating PLUS repeating MULTIPLIED BY ten RAISED TO the power of negative nonRepeatingLength MULTIPLIED BY the element at position repeatingLength OF ratios  
        END FUNCTION  

        RETURN the absolute value of the difference BETWEEN valueOf applied to s AND valueOf applied to t LESS THAN one TIMES ten RAISED TO the power of negative nine  
    END FUNCTION  
END CLASS
