CLASS Solution  
    FUNCTION constrainedSubsetSum(nums k)  
        SET n TO the LENGTH OF nums  
        INITIALIZE dp AS a list of zeros with LENGTH n  
        SET element at position zero of dp TO element at position zero of nums  
        SET max_sum TO element at position zero of nums  
        INITIALIZE q AS a deque containing the single element zero  
        FOR i FROM one TO n MINUS one  
            IF element at position zero of q is a valid index AND element at position element at position zero of q of dp GREATER THAN zero  
                SET value FROM dp AT element at position zero of q  
            ELSE  
                SET value TO zero  
            END IF  
            SET element at position i of dp TO element at position i of nums PLUS value  
            IF max_sum LESS THAN element at position i of dp  
                SET max_sum TO element at position i of dp  
            END IF  
            WHILE q IS NOT empty AND element at position i of dp GREATER THAN OR EQUAL TO element at position last element of q of dp  
                REMOVE last element FROM q  
            END WHILE  
            APPEND i TO q  
            IF element at position zero of q EQUALS i MINUS k  
                REMOVE first element FROM q  
            END IF  
        END FOR  
        RETURN max_sum  
    END FUNCTION  
END CLASS
