CLASS Solution  
    FUNCTION waysToDistribute WITH PARAMETERS n AND k  
        SET MOD TO one billion PLUS seven  
        INITIALIZE dp AS a list of lists where each list has k PLUS one zeros and the total number of such lists is n PLUS one  
        SET element at position zero of list at position zero of dp TO one  
        FOR i FROM one TO n  
            FOR j FROM one TO k  
                SET element at position j of list at position i of dp TO the remainder after dividing the sum of j MULTIPLIED BY element at position j of list at position i MINUS one of dp AND element at position j MINUS one of list at position i MINUS one of dp BY MOD  
            END FOR  
        END FOR  
        RETURN element at position k of list at position n of dp  
    END FUNCTION  
END CLASS
