CLASS Solution  
    FUNCTION numTilings WITH PARAMETER n  
        SET MOD TO one billion PLUS seven  
        CREATE list dp WITH n PLUS one ELEMENTS ALL SET TO zero  
        CREATE list dp1 WITH n PLUS one ELEMENTS ALL SET TO zero  
        SET element at position zero of dp TO one  
        SET element at position one of dp TO one  
        IF n GREATER THAN OR EQUAL TO two THEN  
            SET element at position two of dp TO two  
            SET element at position two of dp1 TO one  
        END IF  
        FOR i FROM three TO n DO  
            SET element at position i of dp TO the SUM of element at position i MINUS one of dp PLUS element at position i MINUS two of dp PLUS two MULTIPLIED BY element at position i MINUS one of dp1 FOLLOWED BY MODULO MOD  
            SET element at position i of dp1 TO the SUM of element at position i MINUS two of dp PLUS element at position i MINUS one of dp1 FOLLOWED BY MODULO MOD  
        END FOR  
        RETURN element at position n of dp  
    END FUNCTION  
END CLASS
