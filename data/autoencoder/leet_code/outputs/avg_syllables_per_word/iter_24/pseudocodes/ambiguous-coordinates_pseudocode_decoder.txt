CLASS Solution  
    FUNCTION ambiguousCoordinates WITH PARAMETER s  
        FUNCTION make WITH PARAMETER frag  
            SET n TO the LENGTH OF frag  
            FOR d FROM one TO n  
                SET left TO the SUBSTRING OF frag FROM POSITION one TO POSITION d  
                SET right TO the SUBSTRING OF frag FROM POSITION d PLUS one TO THE END OF frag  
                IF left DOES NOT START WITH character ZERO OR left EQUALS character ZERO AND right DOES NOT END WITH character ZERO  
                    YIELD THE CONCATENATION OF left PLUS IF d NOT EQUALS n THEN character DOT OTHERWISE EMPTY STRING PLUS right  
                END IF  
            END FOR  
        END FUNCTION  
        SET s TO the SUBSTRING OF s FROM POSITION two TO POSITION LENGTH OF s MINUS one  
        RETURN A LIST OF THE STRING WITH characters OPEN PARENTHESIS PLUS a PLUS COMMA PLUS SPACE PLUS b PLUS CLOSE PARENTHESIS FOR i FROM one TO LENGTH OF s MINUS one FOR EACH a COMMA b IN THE CARTESIAN PRODUCT OF make APPLIED TO the SUBSTRING OF s FROM POSITION one TO POSITION i AND make APPLIED TO the SUBSTRING OF s FROM POSITION i PLUS one TO THE END  
    END FUNCTION  
END CLASS
