CLASS Solution  
    FUNCTION monotoneIncreasingDigits(n)  
        SET digits TO list of each character of string representation of n  
        SET length TO the quantity of elements in digits  
        SET marker TO length  
        FOR i FROM length MINUS one TO one IN DECREASING order  
            IF element at position i of digits LESS THAN element at position i MINUS one of digits  
                SET element at position i MINUS one of digits TO string representation of the integer value of element at position i MINUS one of digits MINUS one  
                SET marker TO i  
            END IF  
        END FOR  
        FOR i FROM marker TO length MINUS one  
            SET element at position i of digits TO string representation of nine  
        END FOR  
        SET result_string TO concatenation of all elements in digits in their current order  
        RETURN integer value of result_string  
    END FUNCTION  
END CLASS
