CLASS Solution  
    FUNCTION rearrangeSticks(n k)  
        SET MOD TO one billion PLUS seven  
        INITIALIZE two dimensional list dp with zeroes of dimensions n PLUS one BY k PLUS one  
        SET element at position zero BY zero of dp TO one  
        FOR i FROM one TO n  
            FOR j FROM one TO k  
                SET element at position i BY j of dp TO the sum of element at position i MINUS one BY j MINUS one of dp PLUS the product of i MINUS one AND element at position i MINUS one BY j of dp MODULO MOD  
            END FOR  
        END FOR  
        RETURN element at position n BY k of dp  
    END FUNCTION  
END CLASS
