CLASS Solution  
    FUNCTION reverseWords(s)  
        CALL the reversal of the entire sequence s  
        SET start TO zero  
        FOR each end FROM zero TO the length of the sequence s MINUS one  
            IF element at position end of sequence s EQUALS a space character  
                CALL the reversal of the subsequence of s from position start TO position end MINUS one  
                SET start TO end PLUS one  
            END IF  
        END FOR  
        CALL the reversal of the subsequence of s from position start TO the last position of s  
    END FUNCTION  
END CLASS
