CLASS Solution  
    FUNCTION findClosestElements(arr parameter of type List of integers k parameter of type integer x parameter of type integer) RETURNS List of integers  
        SET left TO the result of the operation bisect left in arr of x MINUS one  
        SET right TO left PLUS one  
        WHILE the result of right MINUS left MINUS one IS LESS THAN k  
            IF left EQUALS negative one OR right IS LESS THAN the LENGTH OF arr AND the absolute value of element at position right of arr MINUS x IS LESS THAN the absolute value of element at position left of arr MINUS x  
                INCREMENT right BY one  
            ELSE  
                DECREMENT left BY one  
            END IF  
        END WHILE  
        RETURN the slice of arr from position left PLUS one TO position right MINUS one inclusive  
    END FUNCTION  
END CLASS
