CLASS Solution  
    FUNCTION kInversePairs(n k)  
        SET MOD TO ten POWERED BY nine PLUS seven  
        INITIALIZE dp TO a two dimensional collection with n PLUS one collections each containing k PLUS one zeros  
        SET element at position zero of collection at position zero of dp TO one  

        FOR i FROM one TO n  
            SET prefix_sum TO zero  
            FOR j FROM zero TO k  
                SET prefix_sum TO prefix_sum PLUS element at position j of collection at position i MINUS one of dp  
                SET prefix_sum TO the remainder of prefix_sum DIVIDED BY MOD  
                IF j GREATER THAN OR EQUAL TO i  
                    SET prefix_sum TO prefix_sum MINUS element at position j MINUS i of collection at position i MINUS one of dp PLUS MOD  
                    SET prefix_sum TO the remainder of prefix_sum DIVIDED BY MOD  
                END IF  
                SET element at position j of collection at position i of dp TO prefix_sum  
            END FOR  
        END FOR  

        RETURN element at position k of collection at position n of dp  
    END FUNCTION  
END CLASS
