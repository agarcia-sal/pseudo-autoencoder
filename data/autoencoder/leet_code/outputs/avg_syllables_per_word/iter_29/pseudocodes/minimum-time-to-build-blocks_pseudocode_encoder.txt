CLASS Solution  
    FUNCTION minBuildTime(blocks, split)  
        TRANSFORM blocks INTO A MINIMUM HEAP STRUCTURE  
        WHILE THE LENGTH OF blocks IS GREATER THAN one  
            SET first TO THE SMALLEST ELEMENT REMOVED FROM blocks  
            SET second TO THE SMALLEST ELEMENT REMOVED FROM blocks  
            SET combined_time TO split PLUS second  
            ADD combined_time INTO blocks MAINTAINING MINIMUM HEAP PROPERTY  
        END WHILE  
        RETURN THE ELEMENT AT POSITION one OF blocks  
    END FUNCTION  
END CLASS
