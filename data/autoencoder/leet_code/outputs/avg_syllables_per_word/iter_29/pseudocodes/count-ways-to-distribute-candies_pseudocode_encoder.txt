CLASS Solution  
    FUNCTION waysToDistribute(n k)  
        SET MOD TO ten POWERED BY nine PLUS seven  
        INITIALIZE dp AS a two dimensional collection with n PLUS one rows and k PLUS one columns FILLED WITH zero  
        SET element at position zero of dp at position zero TO one  
        FOR i FROM one TO n  
            FOR j FROM one TO k  
                SET element at position i of dp at position j TO the REMAINDER AFTER DIVISION BY MOD of the SUM of j MULTIPLIED BY the element at position i MINUS one of dp at position j AND the element at position i MINUS one of dp at position j MINUS one  
            END FOR  
        END FOR  
        RETURN element at position n of dp at position k  
    END FUNCTION  
END CLASS
