CLASS Solution  
    FUNCTION nextGreaterElement(n)  
        SET digits TO the list of characters representing the number n  
        SET length TO the LENGTH OF digits  
        SET i TO length MINUS two  
        WHILE i GREATER THAN OR EQUAL TO zero AND character at position i of digits GREATER THAN OR EQUAL TO character at position i PLUS one of digits  
            DECREMENT i BY one  
        END WHILE  
        IF i EQUALS negative one  
            RETURN negative one  
        END IF  
        SET j TO length MINUS one  
        WHILE character at position j of digits LESS THAN OR EQUAL TO character at position i of digits  
            DECREMENT j BY one  
        END WHILE  
        SET temporary variable TO character at position i of digits  
        SET character at position i of digits TO character at position j of digits  
        SET character at position j of digits TO temporary variable  
        SET suffix TO the substring of digits from position i PLUS one TO the last position of digits  
        REVERSE the order of elements in suffix  
        SET digits TO the concatenation of the substring of digits from the first position TO position i PLUS one AND suffix  
        SET result TO the integer formed by concatenating all characters in digits  
        IF result GREATER THAN the integer value of two POWERED BY thirty ONE MINUS one  
            RETURN negative one  
        END IF  
        RETURN result  
    END FUNCTION  
END CLASS
