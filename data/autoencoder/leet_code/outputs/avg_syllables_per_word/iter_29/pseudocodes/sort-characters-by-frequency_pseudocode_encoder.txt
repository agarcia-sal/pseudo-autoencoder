CLASS Solution  
    FUNCTION frequencySort(s)  
        CALL frequency TO the mapping of character frequencies computed by counting the occurrences of each character in s  
        CALL sorted_characters TO the list of characters sorted by their frequency in frequency in descending order  
        SET result TO the concatenation of each character repeated by the number of times indicated by its frequency in sorted_characters  
        RETURN result  
    END FUNCTION  
END CLASS
