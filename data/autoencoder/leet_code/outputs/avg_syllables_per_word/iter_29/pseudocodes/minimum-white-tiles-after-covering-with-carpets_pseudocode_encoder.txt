CLASS Solution  
    FUNCTION minimumWhiteTiles(floor string_parameter numCarpets integer_parameter carpetLen integer_parameter) RETURNS integer  
        SET n TO the LENGTH OF floor  
        INITIALIZE dp TO a two dimensional collection with the number of rows EQUAL TO n PLUS one and the number of columns EQUAL TO numCarpets PLUS one FILLED WITH zero  
        FOR i FROM one TO n  
            SET dp at position i at position zero TO dp at position i MINUS one at position zero PLUS the evaluation of whether the character at position i MINUS one of floor EQUALS the character one  
        END FOR  
        FOR j FROM one TO numCarpets  
            FOR i FROM one TO n  
                SET dp at position i at position j TO dp at position i MINUS one at position j PLUS the evaluation of whether the character at position i MINUS one of floor EQUALS the character one  
                SET dp at position i at position j TO the MINIMUM VALUE BETWEEN dp at position i at position j AND dp at position the MAXIMUM VALUE BETWEEN zero AND i MINUS carpetLen at position j MINUS one  
            END FOR  
        END FOR  
        RETURN dp at position n at position numCarpets  
    END FUNCTION  
END CLASS
