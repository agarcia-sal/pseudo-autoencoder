CLASS Solution  
    FUNCTION numTilings(n)  
        SET MODULO TO ten POWERED BY nine PLUS seven  
        CREATE collection dp OF SIZE n PLUS one FILLED WITH zero  
        CREATE collection dp1 OF SIZE n PLUS one FILLED WITH zero  
        SET element at position zero of dp TO one  
        SET element at position one of dp TO one  
        IF n GREATER THAN OR EQUAL TO two  
            SET element at position two of dp TO two  
            SET element at position two of dp1 TO one  
        END IF  
        FOR index FROM three TO n  
            SET element at position index of dp TO the REMAINDER AFTER DIVIDING the SUM of the element at position index MINUS one of dp PLUS the element at position index MINUS two of dp PLUS two MULTIPLIED BY the element at position index MINUS one of dp1 BY MODULO  
            SET element at position index of dp1 TO the REMAINDER AFTER DIVIDING the SUM of the element at position index MINUS two of dp PLUS the element at position index MINUS one of dp1 BY MODULO  
        END FOR  
        RETURN element at position n of dp  
    END FUNCTION  
END CLASS
