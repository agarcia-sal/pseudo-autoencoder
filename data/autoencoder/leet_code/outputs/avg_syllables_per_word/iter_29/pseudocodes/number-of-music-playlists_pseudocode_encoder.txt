CLASS Solution  
    FUNCTION numMusicPlaylists(n number_of_songs goal desired_playlist_length k minimum_distance_between_repeats) RETURNS integer  
        SET MOD to the number one billion PLUS seven  
        
        INITIALIZE dp AS a two dimensional collection where for each index i from zero TO goal SET each element TO a collection of zero values with length n PLUS one  
        SET element at position zero of dp TO a collection where element at position zero IS one AND ALL OTHER elements are zero  
        
        FOR i FROM one TO goal  
            FOR j FROM one TO n  
                SET element at position j of element at position i of dp TO element at position j of element at position i of dp PLUS element at position j MINUS one of element at position i MINUS one of dp MULTIPLIED BY number_of_songs MINUS (j MINUS one)  
                SET element at position j of element at position i of dp TO element at position j of element at position i of dp MODULO MOD  
                
                IF j GREATER THAN k THEN  
                    SET element at position j of element at position i of dp TO element at position j of element at position i of dp PLUS element at position j of element at position i MINUS one of dp MULTIPLIED BY j MINUS k  
                    SET element at position j of element at position i of dp TO element at position j of element at position i of dp MODULO MOD  
                END IF  
            END FOR  
        END FOR  
        
        RETURN element at position n of element at position goal of dp  
    END FUNCTION  
END CLASS
