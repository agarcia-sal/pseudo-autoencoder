CLASS Solution  
    FUNCTION originalDigits(s)  
        SET count TO a mapping from characters to their frequencies in s  
        SET digits TO a list of length ten filled with zeroes  
        SET the element at position zero of digits TO the value associated with character z in count  
        SET the element at position two of digits TO the value associated with character w in count  
        SET the element at position four of digits TO the value associated with character u in count  
        SET the element at position six of digits TO the value associated with character x in count  
        SET the element at position eight of digits TO the value associated with character g in count  
        SET the element at position three of digits TO the value associated with character h in count MINUS the element at position eight of digits  
        SET the element at position five of digits TO the value associated with character f in count MINUS the element at position four of digits  
        SET the element at position seven of digits TO the value associated with character s in count MINUS the element at position six of digits  
        SET the element at position nine of digits TO the value associated with character i in count MINUS the element at position five of digits MINUS the element at position six of digits MINUS the element at position eight of digits  
        SET the element at position one of digits TO the value associated with character o in count MINUS the element at position zero of digits MINUS the element at position two of digits MINUS the element at position four of digits  
        SET result TO an empty list  
        FOR i FROM zero TO nine  
            APPEND the sequence of the character corresponding to the string representation of i repeated the element at position i of digits times TO result  
        END FOR  
        RETURN the concatenation of all elements in result  
    END FUNCTION  
END CLASS
