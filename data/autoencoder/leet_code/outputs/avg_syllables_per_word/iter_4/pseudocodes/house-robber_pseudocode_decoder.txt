CLASS Solution
    FUNCTION rob(nums)
        IF nums IS EMPTY
            RETURN 0
        IF LENGTH OF nums IS 1
            RETURN element AT index 0 IN nums
        SET dp TO list OF zeros WITH LENGTH EQUAL TO LENGTH OF nums
        SET element AT index 0 IN dp TO element AT index 0 IN nums
        SET element AT index 1 IN dp TO MAXIMUM OF element AT index 0 IN nums AND element AT index 1 IN nums
        FOR i FROM 2 TO LENGTH OF nums MINUS 1
            SET element AT index i IN dp TO MAXIMUM OF element AT index i MINUS 1 IN dp AND element AT index i MINUS 2 IN dp PLUS element AT index i IN nums
        RETURN element AT last index IN dp
    END FUNCTION
END CLASS
