CLASS Solution  
    FUNCTION distinctSubseqII(s)  
        SET MOD TO one billion PLUS seven  
        SET n TO the LENGTH OF s  
        SET dp TO a list of zeros of length n PLUS one  
        SET the element at position zero of dp TO one  
        SET last TO an empty key-value collection  
        FOR i FROM one TO n  
            SET char TO the element at position i MINUS one of s  
            SET the element at position i of dp TO the element at position i MINUS one of dp MULTIPLIED BY two  
            SET the element at position i of dp TO the remainder when the element at position i of dp is DIVIDED BY MOD  
            IF char IS IN last THEN  
                SET the element at position i of dp TO the element at position i of dp MINUS the element at position one LESS THAN the value mapped by char in last of dp PLUS MOD  
                SET the element at position i of dp TO the remainder when the element at position i of dp is DIVIDED BY MOD  
            END IF  
            SET the value mapped by char in last TO i  
        END FOR  
        RETURN the remainder when the element at position n of dp MINUS one is DIVIDED BY MOD  
    END FUNCTION  
END CLASS
