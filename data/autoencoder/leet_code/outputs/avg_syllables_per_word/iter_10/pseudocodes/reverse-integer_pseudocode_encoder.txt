CLASS Solution  
    FUNCTION reverse(x)  
        SET sign TO minus one IF x LESS THAN zero ELSE one  
        SET reversed_x TO integer conversion OF string conversion OF absolute value OF x reversed  
        SET result TO sign MULTIPLIED BY reversed_x  
        IF result LESS THAN negative two TO THE POWER OF thirty-one OR result GREATER THAN two TO THE POWER OF thirty-one MINUS one  
            RETURN zero  
        END IF  
        RETURN result  
    END FUNCTION  
END CLASS
