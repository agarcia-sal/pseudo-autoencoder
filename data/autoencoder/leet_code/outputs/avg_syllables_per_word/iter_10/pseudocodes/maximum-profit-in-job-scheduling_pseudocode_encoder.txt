CLASS Solution  
    FUNCTION jobScheduling(startTime, endTime, profit)  
        SET jobs TO the LIST CREATED BY SORTING THE ZIP OF startTime, endTime, profit BY the SECOND ELEMENT OF EACH TUPLE  
        SET end_times TO the LIST OF THE SECOND ELEMENT OF EACH job IN jobs  
        SET dp TO the LIST OF ZEROS WITH THE LENGTH OF THE LENGTH OF jobs PLUS ONE  
        FOR i FROM one TO THE LENGTH OF jobs  
            SET j TO THE POSITION FOUND BY BINARY SEARCHING end_times FOR THE VALUE OF THE FIRST ELEMENT OF THE job AT POSITION i MINUS ONE IN jobs  
            SET dp AT POSITION i TO THE GREATER OF dp AT POSITION i MINUS ONE AND dp AT POSITION j PLUS THE THIRD ELEMENT OF THE job AT POSITION i MINUS ONE IN jobs  
        END FOR  
        RETURN THE LAST ELEMENT OF dp  
    END FUNCTION  
END CLASS
