CLASS Solution  
    FUNCTION minBuildTime(blocks, split)  
        CALL heapify FUNCTION WITH PARAMETERS blocks TO CONVERT blocks INTO A MINIMUM HEAP  
        WHILE THE LENGTH OF blocks IS GREATER THAN one  
            SET first TO THE RESULT OF CALLING heappop FUNCTION WITH PARAMETERS blocks TO REMOVE THE SMALLEST ELEMENT  
            SET second TO THE RESULT OF CALLING heappop FUNCTION WITH PARAMETERS blocks TO REMOVE THE NEXT SMALLEST ELEMENT  
            SET combined_time TO split PLUS second  
            CALL heappush FUNCTION WITH PARAMETERS blocks AND combined_time TO ADD THE COMBINED TIME BACK INTO THE HEAP  
        END WHILE  
        RETURN element AT POSITION zero OF blocks  
    END FUNCTION  
END CLASS
