CLASS Solution  
    FUNCTION isNumber(s)  
        IMPORT regular expression module AS re  
        SET number_regular_expression TO a string representing the pattern for a valid number starting optionally with plus or minus sign followed by either one or more digits optionally followed by a decimal point and any number of digits OR a decimal point followed by one or more digits and optionally followed by an exponent part starting with the letter e optionally followed by plus or minus sign and one or more digits  
        RETURN whether the entire string s FULLY MATCHES the number_regular_expression pattern USING the fullmatch method FROM the regular expression module AND the result IS NOT EQUAL TO none  
    END FUNCTION  
END CLASS
