CLASS Solution  
    FUNCTION numberOfWays(self, numPeople)  
        SET MOD TO ten raised to the power of nine PLUS one  
        INITIALIZE list dp WITH zero repeated numPeople PLUS one times  
        SET element zero of dp TO one  
        SET element two of dp TO one  
        
        FOR i FROM four TO numPeople IN STEPS OF two  
            FOR j FROM zero TO i MINUS one IN STEPS OF two  
                SET element i of dp TO element i of dp PLUS element j of dp MULTIPLIED BY element (i MINUS two MINUS j) of dp  
                SET element i of dp TO element i of dp MODULO MOD  
            END FOR  
        END FOR  
        
        RETURN element numPeople of dp  
    END FUNCTION  
END CLASS
