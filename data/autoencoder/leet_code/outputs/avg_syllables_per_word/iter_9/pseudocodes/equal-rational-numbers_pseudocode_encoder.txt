CLASS Solution  
    FUNCTION isRationalEqual(self, s, t)  
        SET ratios TO list of values: one, one DIVIDED BY nine, one DIVIDED BY ninety-nine, one DIVIDED BY nine hundred ninety-nine, one DIVIDED BY nine thousand nine hundred ninety-nine  

        FUNCTION valueOf(string_s)  
            IF string_s DOES NOT CONTAIN character left parenthesis THEN  
                RETURN string_s CONVERTED TO floating point number  
            END IF  

            SET leftParenIndex TO position of character left parenthesis in string_s  
            SET rightParenIndex TO position of character right parenthesis in string_s  
            SET dotIndex TO position of character dot in string_s  

            SET integerAndNonRepeating TO substring of string_s from start up to leftParenIndex CONVERTED TO floating point number  
            SET nonRepeatingLength TO leftParenIndex MINUS dotIndex MINUS one  

            SET repeating TO integer represented by substring of string_s from position one after leftParenIndex up to rightParenIndex  
            SET repeatingLength TO rightParenIndex MINUS leftParenIndex MINUS one  

            RETURN integerAndNonRepeating PLUS repeating MULTIPLIED BY power of point one RAISED TO nonRepeatingLength MULTIPLIED BY ratios at index repeatingLength  
        END FUNCTION  

        RETURN absolute value of valueOf(s) MINUS valueOf(t) LESS THAN one times ten RAISED TO negative nine  
    END FUNCTION  
END CLASS
