CLASS Solution  
    FUNCTION monotoneIncreasingDigits(n)  
        SET digits TO list of characters of string representation of n  
        SET length TO the LENGTH OF digits  
        SET marker TO length  
        FOR i FROM length MINUS one DOWN TO one  
            IF character at position i of digits LESS THAN character at position i MINUS one of digits  
                SET character at position i MINUS one of digits TO string representation of integer value of character at position i MINUS one of digits MINUS one  
                SET marker TO i  
            END IF  
        END FOR  
        FOR i FROM marker TO length MINUS one  
            SET character at position i of digits TO character nine  
        END FOR  
        RETURN integer value of concatenation of characters in digits  
    END FUNCTION  
END CLASS
