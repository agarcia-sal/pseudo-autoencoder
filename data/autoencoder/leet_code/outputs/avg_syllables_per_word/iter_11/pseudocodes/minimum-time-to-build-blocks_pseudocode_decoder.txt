CLASS Solution  
    FUNCTION minBuildTime(blocks, split)  
        CALL heapify ON blocks TO TRANSFORM blocks INTO A MINIMUM HEAP  
        WHILE the LENGTH OF blocks IS GREATER THAN one  
            SET first TO the result OF heappop ON blocks  
            SET second TO the result OF heappop ON blocks  
            SET combined_time TO split PLUS second  
            CALL heappush ON blocks WITH combined_time  
        END WHILE  
        RETURN the element at position zero OF blocks  
    END FUNCTION  
END CLASS
