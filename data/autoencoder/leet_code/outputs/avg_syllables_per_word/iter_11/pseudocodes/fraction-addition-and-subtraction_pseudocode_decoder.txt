CLASS Solution  
    FUNCTION fractionAddition(expression)  
        SET fractions TO the result of replacing every occurrence of minus sign IN expression WITH plus sign FOLLOWED BY minus sign  
        SPLIT fractions BY plus sign INTO list of fraction strings  
        SET result TO the sum of converting each non-empty fraction string IN fractions TO a Fraction  
        RETURN the string formed by concatenating result numerator, a forward slash, and result denominator  
    END FUNCTION  
END CLASS
