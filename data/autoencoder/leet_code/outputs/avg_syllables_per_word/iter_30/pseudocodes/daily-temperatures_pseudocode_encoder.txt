CLASS Solution  
    FUNCTION dailyTemperatures WITH PARAMETER temperatures  
        SET n TO the LENGTH OF temperatures  
        SET answer TO a LIST OF zero REPEATED n TIMES  
        SET stack TO an EMPTY LIST  
        FOR i FROM zero TO n MINUS one  
            WHILE stack IS NOT EMPTY AND element at position i OF temperatures IS GREATER THAN element at position element at last position OF stack OF temperatures  
                SET prev_index TO the REMOVAL of last element FROM stack  
                SET element at position prev_index OF answer TO i MINUS prev_index  
            END WHILE  
            APPEND i TO stack  
        END FOR  
        RETURN answer  
    END FUNCTION  
END CLASS
