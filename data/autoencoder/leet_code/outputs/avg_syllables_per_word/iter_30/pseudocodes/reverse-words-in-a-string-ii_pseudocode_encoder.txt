CLASS Solution  
    FUNCTION reverseWords(s)  
        CALL reverse the order of elements IN s  
        SET start TO zero  
        FOR end FROM zero TO the LENGTH OF s MINUS one  
            IF element at position end OF s EQUALS a single space  
                CALL reverse the order of elements FROM position start TO position end MINUS one OF s  
                SET start TO end PLUS one  
            END IF  
        END FOR  
        CALL reverse the order of elements FROM position start TO the end OF s  
    END FUNCTION  
END CLASS
