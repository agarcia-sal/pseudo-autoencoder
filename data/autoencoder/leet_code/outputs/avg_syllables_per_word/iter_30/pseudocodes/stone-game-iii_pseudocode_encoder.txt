CLASS Solution  
    FUNCTION stoneGameIII(stoneValue)  
        SET n TO the LENGTH OF stoneValue  
        SET dp TO a new list of zero repeated n PLUS one times  
        FOR i FROM n MINUS one DOWN TO zero  
            SET dp at position i TO element at position i OF stoneValue MINUS dp at position i PLUS one  
            IF i PLUS two LESS THAN OR EQUAL TO n  
                SET dp at position i TO the MAXIMUM of dp at position i AND element at position i OF stoneValue PLUS element at position i PLUS one OF stoneValue MINUS dp at position i PLUS two  
            END IF  
            IF i PLUS three LESS THAN OR EQUAL TO n  
                SET dp at position i TO the MAXIMUM of dp at position i AND element at position i OF stoneValue PLUS element at position i PLUS one OF stoneValue PLUS element at position i PLUS two OF stoneValue MINUS dp at position i PLUS three  
            END IF  
        END FOR  
        IF dp at position zero GREATER THAN zero  
            RETURN Alice  
        ELSE IF dp at position zero LESS THAN zero  
            RETURN Bob  
        ELSE  
            RETURN Tie  
        END IF  
    END FUNCTION  
END CLASS
