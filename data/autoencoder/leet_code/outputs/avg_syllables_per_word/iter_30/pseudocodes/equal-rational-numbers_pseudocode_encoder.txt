CLASS Solution  
    FUNCTION isRationalEqual(s, t)  
        DEFINE ratios AS list containing one followed by one DIVIDED BY nine followed by one DIVIDED BY ninety nine followed by one DIVIDED BY nine hundred ninety nine followed by one DIVIDED BY nine thousand nine hundred ninety nine  

        FUNCTION valueOf(s)  
            IF the position of the character open parenthesis IN s EQUALS minus one  
                RETURN the floating point number represented by s  
            END IF  

            SET leftParenIndex TO the position of the character open parenthesis IN s  
            SET rightParenIndex TO the position of the character close parenthesis IN s  
            SET dotIndex TO the position of the character dot IN s  

            SET integerAndNonRepeating TO the floating point number represented by the substring from position zero TO position leftParenIndex OF s  
            SET nonRepeatingLength TO leftParenIndex MINUS dotIndex MINUS one  

            SET repeating TO the integer represented by the substring from position leftParenIndex PLUS one TO position rightParenIndex OF s  
            SET repeatingLength TO rightParenIndex MINUS leftParenIndex MINUS one  
            RETURN integerAndNonRepeating PLUS repeating MULTIPLIED BY zero point one RAISED TO THE POWER OF nonRepeatingLength MULTIPLIED BY the element at position repeatingLength OF ratios  
        END FUNCTION  

        RETURN the absolute value of the difference between valueOf applied to s AND valueOf applied to t LESS THAN one MULTIPLIED BY ten RAISED TO THE POWER OF minus nine  
    END FUNCTION  
END CLASS
