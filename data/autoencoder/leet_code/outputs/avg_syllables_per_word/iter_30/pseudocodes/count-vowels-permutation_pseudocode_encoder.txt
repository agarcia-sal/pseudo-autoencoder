CLASS Solution  
    FUNCTION countVowelPermutation WITH PARAMETER n  
        SET MOD TO ten raised to the power of nine PLUS one  
        SET a TO one  
        SET e TO one  
        SET i TO one  
        SET o TO one  
        SET u TO one  
        FOR each number FROM two TO n  
            SET a_next TO the sum of e PLUS i PLUS u  
            SET e_next TO the sum of a PLUS i  
            SET i_next TO the sum of e PLUS o  
            SET o_next TO i  
            SET u_next TO the sum of i PLUS o  
            SET a TO remainder of a_next DIVIDED BY MOD  
            SET e TO remainder of e_next DIVIDED BY MOD  
            SET i TO remainder of i_next DIVIDED BY MOD  
            SET o TO remainder of o_next DIVIDED BY MOD  
            SET u TO remainder of u_next DIVIDED BY MOD  
        END FOR  
        SET result TO remainder of the sum of a PLUS e PLUS i PLUS o PLUS u DIVIDED BY MOD  
        RETURN result  
    END FUNCTION  
END CLASS
