CLASS Solution  
    FUNCTION findClosestElements(arr, k, x)  
        SET left TO the position bisect_left would insert x in arr MINUS one  
        SET right TO left PLUS one  
        WHILE right MINUS left MINUS one LESS THAN k  
            IF left EQUALS minus one OR right LESS THAN the LENGTH OF arr AND the absolute value of element at position right of arr MINUS x LESS THAN the absolute value of element at position left of arr MINUS x  
                INCREMENT right BY one  
            ELSE  
                DECREMENT left BY one  
            END IF  
        END WHILE  
        RETURN the elements from position left PLUS one TO position right EXCLUSIVE OF arr  
    END FUNCTION  
END CLASS
