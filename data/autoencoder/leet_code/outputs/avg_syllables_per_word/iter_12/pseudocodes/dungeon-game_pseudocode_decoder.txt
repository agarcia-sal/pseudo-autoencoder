CLASS Solution  
    FUNCTION calculateMinimumHP(dungeon)  
        SET m TO the LENGTH OF dungeon  
        SET n TO the LENGTH OF element at position zero of dungeon  
        SET dp TO a new two-dimensional list with dimensions m PLUS one BY n PLUS one, each element INITIALIZED TO infinity  
        SET element at position m of dp and position n MINUS one TO one  
        SET element at position m MINUS one of dp and position n TO one  
        FOR i FROM m MINUS one DOWN TO zero  
            FOR j FROM n MINUS one DOWN TO zero  
                SET min_health TO the MINIMUM OF element at position i PLUS one of dp and position j AND element at position i of dp and position j PLUS one MINUS element at position i of dungeon and position j  
                SET element at position i of dp and position j TO the MAXIMUM OF min_health AND one  
            END FOR  
        END FOR  
        RETURN element at position zero of dp and position zero  
    END FUNCTION  
END CLASS
