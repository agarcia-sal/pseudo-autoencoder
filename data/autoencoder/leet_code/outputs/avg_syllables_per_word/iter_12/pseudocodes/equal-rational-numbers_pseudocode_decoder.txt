CLASS Solution  
    FUNCTION isRationalEqual(s, t)  
        SET ratios TO list containing one, one DIVIDED BY nine, one DIVIDED BY ninety-nine, one DIVIDED BY nine hundred ninety-nine, one DIVIDED BY nine thousand nine hundred ninety-nine  

        FUNCTION valueOf(s)  
            IF position of character left parenthesis IN s EQUALS negative one  
                RETURN floating point value of s  
            END IF  

            SET leftParenIndex TO position of character left parenthesis IN s  
            SET rightParenIndex TO position of character right parenthesis IN s  
            SET dotIndex TO position of character dot IN s  

            SET integerAndNonRepeating TO floating point value of substring of s from start TO (leftParenIndex exclusive)  
            SET nonRepeatingLength TO leftParenIndex MINUS dotIndex MINUS one  

            SET repeating TO integer value of substring of s from position leftParenIndex PLUS one TO rightParenIndex exclusive  
            SET repeatingLength TO rightParenIndex MINUS leftParenIndex MINUS one  

            RETURN integerAndNonRepeating PLUS repeating MULTIPLIED BY zero point one RAISED TO THE POWER OF nonRepeatingLength MULTIPLIED BY element at position repeatingLength of ratios  
        END FUNCTION  

        RETURN absolute value of valueOf(s) MINUS valueOf(t) LESS THAN one times ten RAISED TO THE POWER OF negative nine  
    END FUNCTION  
END CLASS
