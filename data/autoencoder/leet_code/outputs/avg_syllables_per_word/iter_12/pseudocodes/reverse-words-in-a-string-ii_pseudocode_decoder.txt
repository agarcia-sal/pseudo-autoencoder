CLASS Solution  
    FUNCTION reverseWords(s)  
        CALL reverse ENTIRE list s  
        SET start TO zero  
        FOR end FROM zero TO LENGTH OF s MINUS one  
            IF element at position end of s EQUALS single space  
                CALL reverse SUBLIST of s from position start TO position end MINUS one  
                SET start TO end PLUS one  
            END IF  
        END FOR  
        CALL reverse SUBLIST of s from position start TO end OF s  
    END FUNCTION  
END CLASS
