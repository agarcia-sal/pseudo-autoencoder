CLASS Solution  
    FUNCTION isInterleave WITH PARAMETERS s1 s2 s3  
        IF the LENGTH OF s1 PLUS the LENGTH OF s2 NOT EQUAL TO the LENGTH OF s3  
            RETURN false  
        END IF  

        SET dp TO a list of boolean false values with the LENGTH OF s2 PLUS one positions  
        SET element at position zero of dp TO true  

        FOR index i FROM zero TO the LENGTH OF s1 INCLUSIVE  
            FOR index j FROM zero TO the LENGTH OF s2 INCLUSIVE  
                IF index i GREATER THAN zero  
                    SET element at position j of dp TO element at position j of dp AND the CHARACTER at position i MINUS one of s1 EQUALS the CHARACTER at position i PLUS j MINUS one of s3  
                END IF  
                IF index j GREATER THAN zero  
                    SET element at position j of dp TO element at position j of dp OR (element at position j MINUS one of dp AND the CHARACTER at position j MINUS one of s2 EQUALS the CHARACTER at position i PLUS j MINUS one of s3)  
                END IF  
            END FOR  
        END FOR  

        RETURN element at position the LENGTH OF s2 of dp  
    END FUNCTION  
END CLASS
