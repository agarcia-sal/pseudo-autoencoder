CLASS Solution  
    FUNCTION computeArea with parameters ax1 ay1 ax2 ay2 bx1 by1 bx2 by2  
        SET area1 TO the difference between ax2 AND ax1 MULTIPLIED BY the difference between ay2 AND ay1  
        SET area2 TO the difference between bx2 AND bx1 MULTIPLIED BY the difference between by2 AND by1  
        SET overlap_width TO the maximum BETWEEN zero AND the difference BETWEEN the minimum BETWEEN ax2 AND bx2 AND the maximum BETWEEN ax1 AND bx1  
        SET overlap_height TO the maximum BETWEEN zero AND the difference BETWEEN the minimum BETWEEN ay2 AND by2 AND the maximum BETWEEN ay1 AND by1  
        SET overlap_area TO overlap_width MULTIPLIED BY overlap_height  
        SET total_area TO area1 PLUS area2 MINUS overlap_area  
        RETURN total_area  
    END FUNCTION  
END CLASS
