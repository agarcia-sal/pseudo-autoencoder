CLASS Solution  
    FUNCTION findNthDigit WITH PARAMETER n  
        SET length TO one  
        SET count TO nine  
        SET start TO one  
        WHILE n IS GREATER THAN length MULTIPLIED BY count HOLDS  
            DECREMENT n BY length MULTIPLIED BY count  
            INCREMENT length BY one  
            MULTIPLY count BY ten  
            MULTIPLY start BY ten  
        END WHILE  
        INCREMENT start BY the RESULT OF integer DIVISION of (n MINUS one) BY length  
        RETURN the DIGIT at the POSITION of the REMAINDER of (n MINUS one) DIVIDED BY length in the STRING REPRESENTATION of start converted TO integer  
    END FUNCTION  
END CLASS
