CLASS Solution  
    FUNCTION profitableSchemes(n, minProfit, group, profit)  
        SET MODULO TO ten RAISED TO THE POWER OF nine PLUS one  
        INITIALIZE dp AS a two dimensional list with the number of rows EQUAL TO n PLUS one and the number of columns EQUAL TO minProfit PLUS one FILLED WITH zeroes  
        FOR i FROM zero TO n INCLUSIVE  
            SET element at position zero of row i of dp TO one  
        END FOR  
        FOR each pair of elements p and g TAKEN SIMULTANEOUSLY from profit and group  
            FOR i FROM n DOWN TO g  
                FOR j FROM minProfit DOWN TO zero  
                    SET element at position j of row i of dp TO the sum of element at position j of row i of dp AND element at position maximum of zero AND j MINUS p of row i MINUS g of dp TAKEN MODULO MODULO  
                END FOR  
            END FOR  
        END FOR  
        RETURN element at position minProfit of row n of dp  
    END FUNCTION  
END CLASS
