CLASS Solution  
    FUNCTION findDuplicate(nums)  
        FUNCTION f(x)  
            RETURN the SUM of ONE for each element v IN nums WHERE element v is LESS THAN OR EQUAL TO x is GREATER THAN x  
        END FUNCTION  
        RETURN the result of applying BINARY SEARCH to the range from zero TO the LENGTH OF nums MINUS ONE with target TRUE using function f as the comparison key  
    END FUNCTION  
END CLASS
