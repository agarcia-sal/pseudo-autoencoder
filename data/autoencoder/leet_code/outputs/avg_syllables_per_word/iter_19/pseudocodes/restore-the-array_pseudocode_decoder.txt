CLASS Solution  
    FUNCTION numberOfArrays WITH PARAMETERS s AND k  
        SET MOD TO ten RAISED TO THE POWER OF nine PLUS seven  
        SET n TO the LENGTH OF s  
        INITIALIZE dp AS a list of LENGTH n PLUS one WITH all elements SET TO zero  
        SET element at position zero of dp TO one  
        FOR i FROM one TO n  
            FOR j FROM zero TO i MINUS one  
                SET condition_one TO the character at position j of s NOT EQUALS zero  
                SET substring_value TO the NUMERIC VALUE of substring from position j TO i MINUS one of s  
                SET condition_two TO substring_value GREATER THAN OR EQUAL TO one AND substring_value LESS THAN OR EQUAL TO k  
                IF condition_one AND condition_two holds THEN  
                    SET element at position i of dp TO element at position i of dp PLUS element at position j of dp MODULO MOD  
                END IF  
            END FOR  
        END FOR  
        RETURN element at position n of dp  
    END FUNCTION  
END CLASS
