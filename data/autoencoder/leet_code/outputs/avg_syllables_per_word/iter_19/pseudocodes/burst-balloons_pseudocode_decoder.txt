CLASS Solution  
    FUNCTION maxCoins(nums)  
        SET nums TO the LIST formed by APPENDING one TO the FRONT of nums AND one TO the END of nums  
        SET n TO the LENGTH OF nums  
        SET dp TO a TWO DIMENSIONAL LIST of zeroes with dimensions n BY n  
        FOR each length FROM two TO n MINUS one  
            FOR each i FROM zero TO n MINUS length MINUS one  
                SET j TO i PLUS length  
                FOR each k FROM i PLUS one TO j MINUS one  
                    SET coins TO the ELEMENT at position i of nums MULTIPLIED BY the ELEMENT at position k of nums MULTIPLIED BY the ELEMENT at position j of nums PLUS the ELEMENT at position i AND k of dp PLUS the ELEMENT at position k AND j of dp  
                    SET dp at position i AND j TO the GREATER VALUE BETWEEN the ELEMENT at position i AND j of dp AND coins  
                END FOR  
            END FOR  
        END FOR  
        RETURN the ELEMENT at position zero AND n MINUS one of dp  
    END FUNCTION  
END CLASS
