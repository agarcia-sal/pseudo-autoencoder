CLASS Solution  
    FUNCTION checkRecord WITH PARAMETER n  
        SET MOD TO one billion PLUS seven  
        SET dp TO a three dimensional collection of size n PLUS one BY two BY three filled with zero  
        SET element at position zero at position zero at position zero of dp TO one  
        FOR i FROM one TO n  
            FOR j FROM zero TO one  
                FOR k FROM zero TO two  
                    SET element at position i at position j at position zero of dp TO the sum of element at position i at position j at position zero of dp AND element at position i MINUS one at position j at position k of dp MODULO MOD  
                    IF j EQUALS zero  
                        SET element at position i at position one at position zero of dp TO the sum of element at position i at position one at position zero of dp AND element at position i MINUS one at position zero at position k of dp MODULO MOD  
                    END IF  
                    IF k LESS THAN two  
                        SET element at position i at position j at position k PLUS one of dp TO the sum of element at position i at position j at position k PLUS one of dp AND element at position i MINUS one at position j at position k of dp MODULO MOD  
                    END IF  
                END FOR  
            END FOR  
        END FOR  
        SET result TO zero  
        FOR j FROM zero TO one  
            FOR k FROM zero TO two  
                SET result TO the sum of result AND element at position n at position j at position k of dp MODULO MOD  
            END FOR  
        END FOR  
        RETURN result  
    END FUNCTION  
END CLASS
