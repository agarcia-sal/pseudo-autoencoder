CLASS Solution  
    FUNCTION reverse(x)  
        SET sign TO negative one IF x IS LESS THAN zero OTHERWISE positive one  
        SET reversed_x TO the integer CONVERSION OF the string FORM of the absolute value of x REVERSED in order  
        SET result TO sign MULTIPLIED BY reversed_x  
        IF result IS LESS THAN negative two RAISED TO the power of thirty ONE OR result IS GREATER THAN positive two RAISED TO the power of thirty ONE MINUS one  
            RETURN zero  
        END IF  
        RETURN result  
    END FUNCTION  
END CLASS
