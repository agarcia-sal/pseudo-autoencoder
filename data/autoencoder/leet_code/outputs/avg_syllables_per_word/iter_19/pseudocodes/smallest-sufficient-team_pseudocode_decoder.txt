CLASS Solution  
    FUNCTION smallestSufficientTeam(self, req_skills, people)  
        SET d TO EMPTY MAPPING FROM STRING TO INTEGER  
        FOR each index AND skill IN enumeration of req_skills  
            SET mapping of skill TO index IN d  
        END FOR  
        SET m TO the LENGTH OF req_skills  
        SET n TO the LENGTH OF people  
        SET p TO a list with n elements each SET TO zero  
        FOR each index i AND skill_set ss IN enumeration of people  
            FOR each skill s IN skill_set ss  
                SET element at position i of p TO element at position i of p BITWISE OR WITH one LEFT SHIFTED BY mapping of s in d  
            END FOR  
        END FOR  
        SET f TO a list with length two TO THE POWER OF m each element SET TO infinite  
        SET g TO a list with length two TO THE POWER OF m each element SET TO zero  
        SET h TO a list with length two TO THE POWER OF m each element SET TO zero  
        SET element at position zero of f TO zero  
        FOR each integer i FROM zero TO two TO THE POWER OF m MINUS one  
            IF element at position i of f EQUALS infinite  
                CONTINUE TO the next iteration  
            END IF  
            FOR each integer j FROM zero TO n MINUS one  
                IF element at position i of f PLUS one IS LESS THAN element at position the BITWISE OR of i and element at position j of p of f  
                    SET element at position the BITWISE OR of i and element at position j of p of f TO element at position i of f PLUS one  
                    SET element at position the BITWISE OR of i and element at position j of p of g TO j  
                    SET element at position the BITWISE OR of i and element at position j of p of h TO i  
                END IF  
            END FOR  
        END FOR  
        SET i TO two TO THE POWER OF m MINUS one  
        SET ans TO EMPTY LIST  
        WHILE i IS NOT zero  
            APPEND element at position i of g TO ans  
            SET i TO element at position i of h  
        END WHILE  
        RETURN ans  
    END FUNCTION  
END CLASS
