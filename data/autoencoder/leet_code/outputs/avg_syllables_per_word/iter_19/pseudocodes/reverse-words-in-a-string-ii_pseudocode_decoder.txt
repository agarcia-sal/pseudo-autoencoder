CLASS Solution  
    FUNCTION reverseWords WITH PARAMETERS s  
        CALL reverse ENTIRE collection s  
        SET start TO zero  
        FOR end FROM zero TO the LENGTH OF collection s MINUS one  
            IF element at position end of collection s EQUALS a space character  
                CALL reverse ON the elements FROM position start TO one LESS THAN position end of collection s  
                SET start TO position end PLUS one  
            END IF  
        END FOR  
        CALL reverse ON the elements FROM position start TO the END OF collection s  
    END FUNCTION  
END CLASS
