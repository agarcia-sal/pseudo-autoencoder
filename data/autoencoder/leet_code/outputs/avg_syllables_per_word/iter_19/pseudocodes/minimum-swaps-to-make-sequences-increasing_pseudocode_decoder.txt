CLASS Solution  
    FUNCTION minSwap(nums1, nums2)  
        SET n TO the LENGTH OF nums1  
        SET dp TO a LIST OF LISTS each containing two zeros REPEATED n TIMES  
        
        SET element at position zero of dp at position zero TO zero  
        SET element at position zero of dp at position one TO one  
        
        FOR i FROM one TO n MINUS one  
            IF the element at position i of nums1 IS GREATER THAN the element at position i MINUS one of nums1 AND the element at position i of nums2 IS GREATER THAN the element at position i MINUS one of nums2 THEN  
                IF the element at position i of nums1 IS GREATER THAN the element at position i MINUS one of nums2 AND the element at position i of nums2 IS GREATER THAN the element at position i MINUS one of nums1 THEN  
                    SET element at position i of dp at position zero TO THE MINIMUM OF element at position i MINUS one of dp at position zero AND element at position i MINUS one of dp at position one  
                    SET element at position i of dp at position one TO element at position i of dp at position zero PLUS one  
                ELSE  
                    SET element at position i of dp at position zero TO element at position i MINUS one of dp at position zero  
                    SET element at position i of dp at position one TO element at position i MINUS one of dp at position one PLUS one  
                END IF  
            ELSE  
                IF the element at position i of nums1 IS GREATER THAN the element at position i MINUS one of nums2 AND the element at position i of nums2 IS GREATER THAN the element at position i MINUS one of nums1 THEN  
                    SET element at position i of dp at position zero TO element at position i MINUS one of dp at position one  
                    SET element at position i of dp at position one TO element at position i MINUS one of dp at position zero PLUS one  
                ELSE  
                    SET element at position i of dp at position zero TO element at position i MINUS one of dp at position one  
                    SET element at position i of dp at position one TO element at position i MINUS one of dp at position zero PLUS one  
                END IF  
            END IF  
        END FOR  
        
        RETURN THE MINIMUM OF element at position n MINUS one of dp at position zero AND element at position n MINUS one of dp at position one  
    END FUNCTION  
END CLASS
