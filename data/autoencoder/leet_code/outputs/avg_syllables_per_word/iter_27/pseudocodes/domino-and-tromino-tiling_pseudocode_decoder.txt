CLASS Solution  
    FUNCTION numTilings WITH PARAMETER n  
        SET MOD TO ten raised to the power of nine PLUS one  
        
        INITIALIZE dp AS a list of zero values WITH length n PLUS one  
        INITIALIZE dp1 AS a list of zero values WITH length n PLUS one  
        
        SET element at position zero of dp TO one  
        SET element at position one of dp TO one  
        
        IF n GREATER THAN OR EQUAL TO two  
            SET element at position two of dp TO two  
            SET element at position two of dp1 TO one  
        END IF  
        
        FOR i FROM three TO n  
            SET element at position i of dp TO the sum of element at position i MINUS one of dp PLUS element at position i MINUS two of dp PLUS two MULTIPLIED BY element at position i MINUS one of dp1 TAKEN modulo MOD  
            SET element at position i of dp1 TO the sum of element at position i MINUS two of dp PLUS element at position i MINUS one of dp1 TAKEN modulo MOD  
        END FOR  
        
        RETURN element at position n of dp  
    END FUNCTION  
END CLASS
