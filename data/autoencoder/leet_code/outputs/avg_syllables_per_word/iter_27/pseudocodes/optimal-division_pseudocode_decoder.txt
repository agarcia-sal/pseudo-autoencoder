CLASS Solution  
    FUNCTION optimalDivision(nums)  
        IF the LENGTH OF nums EQUALS one  
            RETURN the ELEMENT at position one of nums REPRESENTED AS STRING  
        END IF  
        IF the LENGTH OF nums EQUALS two  
            RETURN the ELEMENT at position one of nums REPRESENTED AS STRING CONCATENATED WITH the STRING REPRESENTING DIVISION AND CONCATENATED WITH the ELEMENT at position two of nums REPRESENTED AS STRING  
        END IF  
        RETURN the ELEMENT at position one of nums REPRESENTED AS STRING CONCATENATED WITH the STRING REPRESENTING DIVISION AND CONCATENATED WITH an OPENING PARENTHESIS CONCATENATED WITH the ELEMENT at position two of nums REPRESENTED AS STRING CONCATENATED WITH the STRING REPRESENTING DIVISION CONCATENATED WITH the ELEMENTS FROM position three TO THE END OF nums REPRESENTED AS STRINGS AND SEPARATED BY DIVISION SYMBOLS CONCATENATED WITH a CLOSING PARENTHESIS  
    END FUNCTION  
END CLASS
