CLASS Solution  
    FUNCTION kthSmallestPrimeFraction(arr parameter representing a list of integers, k parameter representing an integer) RETURNS list of integers  
        SET n TO LENGTH OF arr  
        SET min_heap TO empty list representing a collection for minimum heap  
        FOR i FROM zero TO the value of n MINUS one MINUS one  
            CALL heap push on min_heap with parameters the value of element at position i of arr DIVIDED BY element at position n MINUS one of arr the value of i and the value of n MINUS one  
        END FOR  
        FOR each iteration FROM zero TO the value of k MINUS one MINUS one  
            CALL heap pop on min_heap ASSIGN TO three values discard first value assign to i assign to j  
            IF the difference between j and one is GREATER THAN i  
                CALL heap push on min_heap with parameters the value of element at position i of arr DIVIDED BY element at position j MINUS one of arr the value of i and the value of j MINUS one  
            END IF  
        END FOR  
        CALL heap pop on min_heap ASSIGN TO three values discard first value assign to i assign to j  
        RETURN list containing element at position i of arr and element at position j of arr  
    END FUNCTION  
END CLASS
