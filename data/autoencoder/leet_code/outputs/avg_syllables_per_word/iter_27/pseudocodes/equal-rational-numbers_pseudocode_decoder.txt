CLASS Solution  
    FUNCTION isRationalEqual WITH PARAMETERS s STRING AND t STRING RETURNS BOOLEAN  
        SET ratios TO list OF ONE PLUS ONE DIVIDED BY NINE PLUS ONE DIVIDED BY NINETY-NINE PLUS ONE DIVIDED BY NINE HUNDRED NINETY-NINE PLUS ONE DIVIDED BY NINE THOUSAND NINE HUNDRED NINETY-NINE

        FUNCTION valueOf WITH PARAMETER s STRING RETURNS FLOAT  
            IF the position of character open parenthesis IN s EQUALS negative one  
                RETURN the floating point value OF s  
            END IF  

            SET leftParenIndex TO the position of character open parenthesis IN s  
            SET rightParenIndex TO the position of character close parenthesis IN s  
            SET dotIndex TO the position of character dot IN s  

            SET integerAndNonRepeating TO the floating point value OF the substring of s FROM position zero TO position leftParenIndex MINUS one  
            SET nonRepeatingLength TO leftParenIndex MINUS dotIndex MINUS one  

            SET repeating TO the integer value OF the substring of s FROM position leftParenIndex PLUS one TO position rightParenIndex MINUS one  
            SET repeatingLength TO rightParenIndex MINUS leftParenIndex MINUS one  

            RETURN integerAndNonRepeating PLUS repeating MULTIPLIED BY the base ten raised to the negative nonRepeatingLength MULTIPLIED BY the element at position repeatingLength of ratios  
        END FUNCTION  

        RETURN the absolute value OF valueOf applied to s MINUS valueOf applied to t LESS THAN one MULTIPLIED BY ten raised to the negative nine  
    END FUNCTION  
END CLASS
