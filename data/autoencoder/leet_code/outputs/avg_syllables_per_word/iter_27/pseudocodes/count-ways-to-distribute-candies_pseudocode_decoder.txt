CLASS Solution  
    FUNCTION waysToDistribute(n PARAMETER, k PARAMETER)  
        SET MOD TO one billion PLUS seven  
        INITIALIZE dp AS a list of lists of zeroes with outer length n PLUS one and inner length k PLUS one  
        SET element at position zero of dp to the element at position zero to one  
        FOR i FROM one TO n  
            FOR j FROM one TO k  
                SET element at position i of dp to the element at position j TO the remainder of the sum of the product of j AND the element at position i MINUS one of dp to the element at position j PLUS the element at position i MINUS one of dp to the element at position j MINUS one DIVIDED BY MOD  
            END FOR  
        END FOR  
        RETURN element at position n of dp to the element at position k  
    END FUNCTION  
END CLASS
