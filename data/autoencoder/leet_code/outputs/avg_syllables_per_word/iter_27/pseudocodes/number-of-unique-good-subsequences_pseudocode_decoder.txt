CLASS Solution  
    FUNCTION numberOfUniqueGoodSubsequences(binary)  
        SET MOD TO ten MULTIPLIED BY ten MULTIPLIED BY ten MULTIPLIED BY ten MULTIPLIED BY ten MULTIPLIED BY ten MULTIPLIED BY ten MULTIPLIED BY ten MULTIPLIED BY ten PLUS seven  
        SET end_with_0 TO zero  
        SET end_with_1 TO zero  
        SET has_zero TO the condition that binary CONTAINS the character zero  
        FOR each bit IN binary  
            IF bit EQUALS the character one  
                SET end_with_1 TO the remainder after division of the sum of end_with_0 PLUS end_with_1 PLUS one BY MOD  
            ELSE  
                SET end_with_0 TO the remainder after division of the sum of end_with_0 PLUS end_with_1 BY MOD  
            END IF  
        END FOR  
        RETURN the remainder after division of the sum of end_with_0 PLUS end_with_1 PLUS has_zero BY MOD  
    END FUNCTION  
END CLASS
