CLASS Solution  
    FUNCTION findClosestElements(arr k x)  
        SET left TO the position where x would be inserted into arr to maintain sorted order MINUS one  
        SET right TO left PLUS one  
        WHILE the difference between right and left MINUS one is LESS THAN k  
            IF left EQUALS negative one OR right is LESS THAN the LENGTH OF arr AND the absolute value of element at position right of arr MINUS x is LESS THAN the absolute value of element at position left of arr MINUS x  
                INCREMENT right BY one  
            ELSE  
                DECREMENT left BY one  
            END IF  
        END WHILE  
        RETURN the sublist of arr starting from position left PLUS one TO position right MINUS one  
    END FUNCTION  
END CLASS
