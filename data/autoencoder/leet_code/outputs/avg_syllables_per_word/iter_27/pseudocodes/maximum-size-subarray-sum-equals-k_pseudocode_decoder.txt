CLASS Solution  
    FUNCTION maxSubArrayLen(nums k)  
        SET cumulative sum index TO a dictionary mapping zero TO negative one  
        SET cumulative sum TO zero  
        SET max length TO zero  
        FOR each index and number IN enumerated nums  
            INCREMENT cumulative sum BY number  
            IF cumulative sum MINUS k EXISTS AS a key IN cumulative sum index  
                SET max length TO the greater VALUE BETWEEN max length AND index MINUS the value AT key cumulative sum MINUS k IN cumulative sum index  
            END IF  
            IF cumulative sum DOES NOT EXIST AS a key IN cumulative sum index  
                ASSIGN reference AT key cumulative sum IN cumulative sum index TO index  
            END IF  
        END FOR  
        RETURN max length  
    END FUNCTION  
END CLASS
