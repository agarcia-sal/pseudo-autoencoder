CLASS Solution  
    FUNCTION change(amount, coins)  
        INITIALIZE dp AS list of zeros with length amount PLUS 1  
        SET dp at index 0 TO 1  

        FOR each coin IN coins  
            FOR x FROM coin TO amount  
                SET dp at index x TO dp at index x PLUS dp at index x MINUS coin  
        RETURN dp at index amount  
    END FUNCTION  
END CLASS
