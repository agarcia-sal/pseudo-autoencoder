CLASS Solution  
    FUNCTION fractionAddition(expression)  
        SET fractions TO result of replacing all occurrences of minus sign in expression with plus sign followed by minus sign THEN splitting result by plus sign  
        SET result TO sum of calling Fraction constructor WITH each fraction IN fractions IF fraction IS NOT empty  
        RETURN string composed of result numerator concatenated with forward slash concatenated with result denominator  
    END FUNCTION  
END CLASS
