CLASS Solution  
    FUNCTION reverseWords(s)  
        CALL reverse the entire list s  
        SET start TO 0  
        FOR end FROM 0 TO length of s MINUS 1  
            IF element at s[end] EQUALS space character  
                CALL reverse the sublist of s from start TO end MINUS 1  
                SET start TO end PLUS 1  
        CALL reverse the sublist of s from start TO end of list  
    END FUNCTION  
END CLASS
