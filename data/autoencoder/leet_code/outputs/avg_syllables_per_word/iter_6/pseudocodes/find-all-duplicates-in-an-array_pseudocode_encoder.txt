CLASS Solution  
    FUNCTION findDuplicates(nums)  
        FOR i FROM 0 TO LENGTH OF nums MINUS 1  
            WHILE value at index i IS NOT EQUAL TO value at index (value at index i MINUS 1)  
                SWAP value at index (value at index i MINUS 1) WITH value at index i  
        RETURN list of values v WHERE for each index i and value v IN nums, value v IS NOT EQUAL TO i PLUS 1  
    END FUNCTION  
END CLASS
