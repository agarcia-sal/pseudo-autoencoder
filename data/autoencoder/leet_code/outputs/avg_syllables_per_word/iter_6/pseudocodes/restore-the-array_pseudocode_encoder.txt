CLASS Solution  
    FUNCTION numberOfArrays(s, k)  
        SET MOD TO ten raised to the power of nine PLUS seven  
        SET n TO LENGTH OF s  
        INITIALIZE dp AS list of zeros with length n PLUS one  
        SET dp at index zero TO one  
        FOR i FROM one TO n  
            FOR j FROM zero TO i MINUS one  
                IF character at index j of s NOT EQUALS zero AND integer value of substring from j TO i GREATER THAN OR EQUAL TO one AND LESS THAN OR EQUAL TO k  
                    SET dp at index i TO (dp at index i PLUS dp at index j) MODULO MOD  
        RETURN dp at index n  
    END FUNCTION  
END CLASS
