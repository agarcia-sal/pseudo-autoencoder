CLASS Solution  
    FUNCTION optimalDivision(nums)  
        IF LENGTH OF nums EQUALS 1  
            RETURN STRING REPRESENTATION OF the first element IN nums  
        ELSE IF LENGTH OF nums EQUALS 2  
            RETURN STRING REPRESENTATION OF the first element IN nums CONCATENATED WITH division character CONCATENATED WITH STRING REPRESENTATION OF the second element IN nums  
        ELSE  
            RETURN STRING REPRESENTATION OF the first element IN nums CONCATENATED WITH division character CONCATENATED WITH open parenthesis CONCATENATED WITH concatenation of STRING REPRESENTATION OF each element IN nums EXCLUDING first element SEPARATED BY division character CONCATENATED WITH close parenthesis  
    END FUNCTION  
END CLASS
