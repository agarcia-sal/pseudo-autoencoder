    FUNCTION isRationalEqual WITH PARAMETERS s AND t  
        SET ratios TO a list containing one and one DIVIDED BY nine and one DIVIDED BY ninety nine and one DIVIDED BY nine hundred ninety nine and one DIVIDED BY nine thousand nine hundred ninety nine

        FUNCTION valueOf WITH PARAMETER s  
            IF the position of character left parenthesis in s EQUALS negative one THEN  
                RETURN the conversion of s TO floating point number  
            END IF

            SET leftParenIndex TO the position of character left parenthesis in s  
            SET rightParenIndex TO the position of character right parenthesis in s  
            SET dotIndex TO the position of character dot in s  

            SET integerAndNonRepeating TO the conversion TO floating point number OF the substring of s from position zero TO the position leftParenIndex MINUS one  
            SET nonRepeatingLength TO leftParenIndex MINUS dotIndex MINUS one  

            SET repeating TO the integer conversion OF the substring of s from position leftParenIndex PLUS one TO the position rightParenIndex MINUS one  
            SET repeatingLength TO rightParenIndex MINUS leftParenIndex MINUS one  

            RETURN integerAndNonRepeating PLUS repeating MULTIPLIED BY zero point one RAISED TO the POWER OF nonRepeatingLength MULTIPLIED BY the element at position repeatingLength OF ratios  
        END FUNCTION

        RETURN the absolute value OF valueOf of s MINUS valueOf of t LESS THAN one TIMES ten RAISED TO the POWER OF negative nine  
    END FUNCTION  
END CLASS
