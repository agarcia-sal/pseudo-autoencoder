CLASS Solution  
    FUNCTION waysToDistribute(n, k)  
        SET MOD TO the integer one followed by nine zeros PLUS seven  
        SET dp TO a list of length n PLUS one where each element is a list of length k PLUS one filled with zero  
        SET the element at position zero of dp at position zero TO one  
        FOR i FROM one TO n  
            FOR j FROM one TO k  
                SET the element at position i of dp at position j TO the remainder after dividing the sum of the product of j MULTIPLIED BY the element at position i MINUS one of dp at position j AND the element at position i MINUS one of dp at position j MINUS one BY MOD  
            END FOR  
        END FOR  
        RETURN the element at position n of dp at position k  
    END FUNCTION  
END CLASS
