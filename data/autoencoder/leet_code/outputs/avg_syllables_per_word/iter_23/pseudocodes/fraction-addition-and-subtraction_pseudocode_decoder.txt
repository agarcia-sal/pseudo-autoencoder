CLASS Solution  
    FUNCTION fractionAddition(expression)  
        SET fractions TO the result of replacing every minus sign in expression with a plus sign followed by a minus sign and then splitting the entire expression at every plus sign  
        SET result TO the summation of the conversion to fraction of each fraction in fractions where the fraction is not empty  
        RETURN the numerator of result as a string followed by a forward slash character followed by the denominator of result as a string  
    END FUNCTION  
END CLASS
