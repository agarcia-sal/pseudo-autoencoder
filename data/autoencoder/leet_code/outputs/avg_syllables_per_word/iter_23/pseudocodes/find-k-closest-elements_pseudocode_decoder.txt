CLASS Solution  
    FUNCTION findClosestElements(arr k x)  
        SET left TO the position result of bisect left operation of x in arr MINUS one  
        SET right TO left PLUS one  
        WHILE the difference of right MINUS left MINUS one is LESS THAN k holds  
            IF left EQUALS negative one OR right is LESS THAN the LENGTH OF arr AND the absolute value of element at position right of arr MINUS x is LESS THAN the absolute value of element at position left of arr MINUS x  
                INCREMENT right BY one  
            ELSE  
                DECREMENT left BY one  
            END IF  
        END WHILE  
        RETURN the sublist from element at position left PLUS one TO element at position right of arr  
    END FUNCTION  
END CLASS
