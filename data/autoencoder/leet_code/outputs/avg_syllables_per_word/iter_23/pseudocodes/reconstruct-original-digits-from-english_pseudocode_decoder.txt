CLASS Solution  
    FUNCTION originalDigits(s)  
        SET count TO a mapping from each character in s TO the number of times that character appears  
        SET digits TO a list of ten zeros representing counts of digits from zero TO nine  
        SET the element at position zero of digits TO the number of times the character z appears in count  
        SET the element at position two of digits TO the number of times the character w appears in count  
        SET the element at position four of digits TO the number of times the character u appears in count  
        SET the element at position six of digits TO the number of times the character x appears in count  
        SET the element at position eight of digits TO the number of times the character g appears in count  
        SET the element at position three of digits TO the number of times the character h appears in count MINUS the element at position eight of digits  
        SET the element at position five of digits TO the number of times the character f appears in count MINUS the element at position four of digits  
        SET the element at position seven of digits TO the number of times the character s appears in count MINUS the element at position six of digits  
        SET the element at position nine of digits TO the number of times the character i appears in count MINUS the element at position five of digits MINUS the element at position six of digits MINUS the element at position eight of digits  
        SET the element at position one of digits TO the number of times the character o appears in count MINUS the element at position zero of digits MINUS the element at position two of digits MINUS the element at position four of digits  
        SET result TO an empty list  
        FOR i FROM zero TO nine  
            APPEND the string representation of i REPEATED the element at position i of digits TIMES TO result  
        END FOR  
        RETURN the concatenation of all elements in result  
    END FUNCTION  
END CLASS
