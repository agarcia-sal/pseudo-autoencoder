CLASS Solution  
    FUNCTION kthSmallestProduct(nums1 parameter of list of integers nums2 parameter of list of integers k parameter of integer) RETURNS integer  
        FUNCTION count(p parameter of integer) RETURNS integer  
            SET cnt TO zero  
            SET n TO the LENGTH OF nums2  
            FOR each element x IN nums1  
                IF x GREATER THAN zero THEN  
                    INCREMENT cnt BY the position from the right where p DIVIDED BY x would be inserted in nums2 to maintain sorted order  
                ELSE IF x LESS THAN zero THEN  
                    INCREMENT cnt BY n MINUS the position from the left where p DIVIDED BY x would be inserted in nums2 to maintain sorted order  
                ELSE  
                    INCREMENT cnt BY n MULTIPLIED BY the truth value of p GREATER THAN OR EQUAL TO zero  
                END IF  
            END FOR  
            RETURN cnt  
        END FUNCTION

        SET mx TO the maximum of the absolute value of the first element of nums1 AND the absolute value of the last element of nums1 MULTIPLIED BY the maximum of the absolute value of the first element of nums2 AND the absolute value of the last element of nums2  
        RETURN the position from the left in the range from negative mx TO mx INCLUDING the value k using count as a key MINUS mx  
    END FUNCTION  
END CLASS
