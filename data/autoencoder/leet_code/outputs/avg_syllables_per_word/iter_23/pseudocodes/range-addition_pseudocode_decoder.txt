CLASS Solution  
    FUNCTION getModifiedArray(length parameter, updates parameter)  
        SET diff TO an empty list of zeros with the number of elements equal to the value of length parameter PLUS one  
        FOR each element IN updates parameter AS startIdx variable endIdx variable inc variable  
            INCREMENT element at position startIdx variable of diff BY inc variable  
            IF the value of endIdx variable PLUS one is LESS THAN the value of length parameter  
                DECREMENT element at position endIdx variable PLUS one of diff BY inc variable  
            END IF  
        END FOR  
        SET arr TO an empty list of zeros with the number of elements equal to the value of length parameter  
        SET element at position zero of arr TO element at position zero of diff  
        FOR index i FROM one TO the value of length parameter MINUS one  
            SET element at position i of arr TO element at position i MINUS one of arr PLUS element at position i of diff  
        END FOR  
        RETURN arr  
    END FUNCTION  
END CLASS
