CLASS Solution  
    FUNCTION reverseWords(s)  
        CALL reverse the order of elements in s  
        SET start TO zero  
        FOR end FROM zero TO the length of s MINUS one  
            IF element at position end of s EQUALS a space character  
                CALL reverse the order of elements in s from position start TO position end MINUS one  
                SET start TO end PLUS one  
            END IF  
        END FOR  
        CALL reverse the order of elements in s from position start TO the last position of s  
    END FUNCTION  
END CLASS
