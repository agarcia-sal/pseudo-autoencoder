CLASS Solution  
    FUNCTION sumScores WITH PARAMETER s  
        SET n TO the LENGTH OF s  
        SET lcp TO a LIST OF zeroes WITH THE LENGTH n  
        SET l TO zero  
        SET r TO zero  
        
        FOR i FROM one TO n MINUS one  
            IF i LESS THAN OR EQUAL TO r  
                SET lcp ELEMENT AT POSITION i TO THE MINIMUM OF r MINUS i PLUS one AND lcp ELEMENT AT POSITION i MINUS l  
            END IF  
            
            WHILE i PLUS lcp ELEMENT AT POSITION i LESS THAN n AND ELEMENT AT POSITION lcp ELEMENT AT POSITION i OF s EQUALS ELEMENT AT POSITION i PLUS lcp ELEMENT AT POSITION i OF s  
                INCREMENT lcp ELEMENT AT POSITION i BY one  
            END WHILE  
            
            IF i PLUS lcp ELEMENT AT POSITION i MINUS one GREATER THAN r  
                SET l TO i  
                SET r TO i PLUS lcp ELEMENT AT POSITION i MINUS one  
            END IF  
        END FOR  
        
        RETURN THE SUM OF ELEMENTS OF lcp PLUS n  
    END FUNCTION  
END CLASS
