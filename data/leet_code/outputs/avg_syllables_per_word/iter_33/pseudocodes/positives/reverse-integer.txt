CLASS Solution  
    FUNCTION reverse(x)  
        IF x LESS THAN zero  
            SET sign TO negative one  
        ELSE  
            SET sign TO one  
        END IF  
        SET absolute value of x TO the absolute value of x  
        SET string representation TO the string form of absolute value of x  
        SET reversed string TO the characters of string representation in reverse order  
        SET reversed number TO the integer form of reversed string  
        SET result TO sign MULTIPLIED BY reversed number  
        IF result LESS THAN negative one MULTIPLIED BY two RAISED TO THE power of thirty one OR result GREATER THAN two RAISED TO THE power of thirty one MINUS one  
            RETURN zero  
        END IF  
        RETURN result  
    END FUNCTION  
END CLASS
