CLASS Solution  
    FUNCTION isRationalEqual(s, t)  
        SET ratios TO list containing one FOLLOWED BY one DIVIDED BY nine FOLLOWED BY one DIVIDED BY ninety nine FOLLOWED BY one DIVIDED BY nine hundred ninety nine FOLLOWED BY one DIVIDED BY nine thousand nine hundred ninety nine  

        FUNCTION valueOf(s)  
            IF the position of the opening parenthesis character in s EQUALS negative one THEN  
                RETURN the floating point conversion of s  
            END IF  

            SET leftParenIndex TO the position of the opening parenthesis character in s  
            SET rightParenIndex TO the position of the closing parenthesis character in s  
            SET dotIndex TO the position of the decimal point character in s  

            SET integerAndNonRepeating TO the floating point conversion of the substring from the start of s TO the position before leftParenIndex of s  
            SET nonRepeatingLength TO leftParenIndex MINUS dotIndex MINUS one  

            SET repeating TO the integer conversion of the substring from the position after leftParenIndex TO the position before rightParenIndex of s  
            SET repeatingLength TO rightParenIndex MINUS leftParenIndex MINUS one  
            RETURN integerAndNonRepeating PLUS repeating MULTIPLIED BY zero POINT one RAISED TO THE POWER OF nonRepeatingLength MULTIPLIED BY the element at position repeatingLength of ratios  
        END FUNCTION  

        RETURN the absolute value of the result of valueOf applied to s MINUS the result of valueOf applied to t LESS THAN one TIMES ten RAISED TO THE POWER OF negative nine  
    END FUNCTION  
END CLASS
