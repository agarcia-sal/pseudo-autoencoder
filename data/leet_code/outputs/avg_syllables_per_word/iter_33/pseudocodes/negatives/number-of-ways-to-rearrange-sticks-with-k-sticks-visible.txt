CLASS Solution  
    FUNCTION rearrangeSticks(n integer, k integer) RETURNS integer  
        SET MODULO TO ten raised to the power of nine PLUS one  
        SET dp TO a two-dimensional list of zeros with dimensions n PLUS one by k PLUS one  
        SET element at position zero of element at position zero of dp TO one  
        FOR i FROM one TO n  
            FOR j FROM one TO k  
                SET element at position i of element at position j of dp TO  
                    the remainder of the sum of element at position i MINUS one of element at position j MINUS one of dp PLUS the product of i MINUS one and element at position i MINUS one of element at position j of dp DIVIDED BY MODULO  
            END FOR  
        END FOR  
        RETURN element at position n of element at position k of dp  
    END FUNCTION  
END CLASS
