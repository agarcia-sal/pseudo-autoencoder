CLASS Solution  
    FUNCTION numberOfUniqueGoodSubsequences(binary)  
        SET MOD TO ten TO THE POWER OF nine PLUS one  
        SET end_with_0 TO zero  
        SET end_with_1 TO zero  
        SET has_zero TO the condition that the character zero is IN binary  
        FOR each bit IN binary  
            IF bit EQUALS the character one  
                SET end_with_1 TO the remainder of the sum of end_with_0 PLUS end_with_1 PLUS one DIVIDED BY MOD  
            ELSE  
                SET end_with_0 TO the remainder of the sum of end_with_0 PLUS end_with_1 DIVIDED BY MOD  
            END IF  
        END FOR  
        RETURN the remainder of the sum of end_with_0 PLUS end_with_1 PLUS has_zero DIVIDED BY MOD  
    END FUNCTION  
END CLASS
