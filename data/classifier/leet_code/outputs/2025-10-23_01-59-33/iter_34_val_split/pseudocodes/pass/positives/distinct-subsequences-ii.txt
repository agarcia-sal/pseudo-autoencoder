CLASS Solution  
    FUNCTION distinctSubseqII WITH parameter s  
        SET MODULO TO ten RAISED TO THE power of nine PLUS seven  
        SET n TO the LENGTH OF s  
        INITIALIZE dp AS a list of zeros WITH LENGTH n PLUS one  
        SET element at position zero of dp TO one  
        INITIALIZE last AS an empty dictionary  
        FOR i FROM one TO n  
            SET character TO element at position i MINUS one of s  
            SET element at position i of dp TO the value of element at position i MINUS one of dp MULTIPLIED BY two MODULO MODULO  
            IF character IS a key in last  
                SET element at position i of dp TO element at position i of dp MINUS element at position last of character MINUS one of dp PLUS MODULO MODULO MODULO  
            END IF  
            ASSIGN i TO last at character  
        END FOR  
        RETURN element at position n of dp MINUS one MODULO MODULO  
    END FUNCTION  
END CLASS
