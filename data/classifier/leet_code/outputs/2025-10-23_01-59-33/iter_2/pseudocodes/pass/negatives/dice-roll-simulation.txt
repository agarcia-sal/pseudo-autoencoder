```
MOD = 10^9+7
dp = 3D array (n+1) x 6 x (max rollMax +1), all 0
for j in 0..5: dp[1][j][1] = 1

for i in 2..n:
  for j in 0..5:
    for k in 1..rollMax[j]:
      if k>1: dp[i][j][k] = dp[i-1][j][k-1]
      else dp[i][j][k] = sum(dp[i-1][x][y] for x in 0..5 if x!=j for y in 1..rollMax[x]) mod MOD

return sum(dp[n][j][k] for j in 0..5 for k in 1..rollMax[j]) mod MOD
```
